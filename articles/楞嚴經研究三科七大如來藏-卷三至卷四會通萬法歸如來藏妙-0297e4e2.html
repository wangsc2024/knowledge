楞嚴經研究：三科七大如來藏

卷三至卷四會通萬法歸如來藏妙真如性

主題概述

《大佛頂首楞嚴經》卷三至卷四是全經義理的核心樞紐。在前兩卷「七處徵心」破妄、「十番顯見」顯真之後，佛陀進一步向阿難開示：一切現象界的「三科」（五蘊、六入、十二處、十八界）與「七大」（地、水、火、風、空、見、識）皆是「如來藏妙真如性」。這段教法被稱為「會通萬法歸如來藏」，是楞嚴經從破妄顯真到修行實踐的關鍵轉折。

佛陀以一統貫的公式——「如來藏中，性X真空，性空真X，清淨本然，周遍法界，隨眾生心，應所知量，循業發現」——逐一開示每一科、每一大的如來藏性，展示了一個完整的佛教本體論框架。

經文重點段落

一、五蘊本如來藏（卷三）

佛告阿難：「復次阿難，云何五陰本如來藏妙真如性？」

佛陀依次說明五蘊各自的虛妄性與如來藏性：

1. 色陰：譬如有人以清淨目觀晴明空，唯一晴虛無所有。若目不瞬不動瞪發勞者，則於虛空中別見狂花。色陰當知亦復如是——本非因緣、非自然性，而是如來藏妙真如性的顯現。

2. 受陰：譬如有人手足安和，百骸調適，忽如忘生，性無違順。其人無故以二手掌於空相摩，於二手中妄生澀滑冷熱諸相。受陰當知亦復如是——本無實體，隨妄想而現。

3. 想陰：譬如口中說醋梅時，口中水出；思蹋懸崖，足心酸澀。想陰的特質是「想」能引發身心反應，但這些都是虛妄不實。

4. 行陰：譬如瀑流，望似恬靜，流急不見。行陰如暴流，微細遷流不息，看似安靜實則念念生滅。

5. 識陰：譬如有人取頻伽瓶，塞其兩孔，滿中擎空，千里遠行，用餉他國。識陰如瓶中之空，看似獨立存在，實則與外空本無分別。

白話總結：五蘊看似真實的存在，實際上都如同「空花」「手摩生觸」「口說梅酸」「急流似靜」「瓶中之空」——皆是虛妄的顯現，其本性就是如來藏清淨妙明的真如自性。

二、六入本如來藏（卷三）

佛告阿難：「云何六入本如來藏妙真如性？」

六入即六根（眼、耳、鼻、舌、身、意）接觸六塵（色、聲、香、味、觸、法）的入口。佛陀逐一分析每一根入的虛妄性，指出六入「非因緣生，非自然有」，本是如來藏妙真如性。

三、十二處本如來藏（卷三）

十二處是六根與六塵的配對（眼與色、耳與聲等），佛陀說明根塵對待而生識，三者互依互存，本無自性，皆歸如來藏。

四、十八界本如來藏（卷三）

十八界在十二處基礎上加入六識（眼識、耳識、鼻識、舌識、身識、意識）。佛陀開示「此十八界皆如來藏妙真如性」——根、塵、識三者和合而生的一切認識活動，本質都是如來藏的清淨顯現。

五、七大如來藏（卷三末至卷四）

這是本段教法最具開創性的部分。佛陀超越傳統的四大（地水火風），加入空、見（覺知）、識三大，形成「七大」體系：

核心公式（以地大為例）： 「如來藏中，性色真空，性空真色，清淨本然，周遍法界，隨眾生心，應所知量，循業發現。」

各大的展開：

  • 地大：性色真空，性空真色 → 物質的堅性本是空性
  • 水大：性水真空，性空真水 → 濕性流動本是空性
  • 火大：性火真空，性空真火 → 暖性光明本是空性
  • 風大：性風真空，性空真風 → 動性流轉本是空性
  • 空大：性覺真空，性空真覺 → 虛空之性本是覺性
  • 見大（覺知）：能見之性即是如來藏性
  • 識大：了別之識即是如來藏性

白話總結：七大的公式統一展示了一個深刻的道理——物質現象（地水火風）、空間（空）、覺知能力（見）、分別認識（識）這七種構成宇宙與生命的基本元素，其本質都是如來藏清淨本然、周遍法界的妙真如性。它們因眾生的心念與業力而顯現為千差萬別的現象世界。

義理解析

1. 「性X真空，性空真X」的辯證結構

這是楞嚴經最精妙的哲學表述之一：

  • 「性色真空」：色（物質）的本性是真正的空——不是空無，而是超越有無對立的「真空」
  • 「性空真色」：這個空的本性就是真正的色——不是斷滅的虛無，而是能生萬物的「妙有」

此結構與《心經》的「色不異空，空不異色，色即是空，空即是色」互為呼應，但楞嚴經更進一步：不僅僅是色與空的互攝，而是一切萬法（三科七大）都統歸如來藏妙真如性。

2. 如來藏思想的特色

如來藏（Tathāgatagarbha，如來之胎藏）在楞嚴經中的特色：

  • 清淨本然：如來藏本來清淨，不受染汙
  • 周遍法界：無處不在，遍及一切
  • 隨眾生心，應所知量：隨順眾生的認知能力而顯現不同層次
  • 循業發現：依眾生的業力因緣而展現為不同的現象

這四句構成了楞嚴經如來藏思想的核心架構——如來藏是萬法的本源，但它的顯現是隨眾生的心量與業力而有差異的。

3. 三科七大的系統性

卷三至卷四的教法具有嚴密的系統性：

從五蘊到七大，是從「個體主觀」到「宇宙客觀」的完整覆蓋，確保一切萬法無一遺漏地歸入如來藏。

修行應用

1. 日常生活中的觀照

當我們看見花朵（色）、聽到鳥鳴（聲）、品嘗食物（味）時，可以觀照：這些感官經驗的本質不是「外在的客觀存在」，也不是「內在的主觀幻想」，而是如來藏隨眾生心、循業發現的妙用。

2. 破除二元對立

「性色真空，性空真色」教導我們超越「有/無」「物質/精神」「主觀/客觀」的二元對立。在面對困境時，不執著於「問題實有」，也不逃避到「一切皆空」，而是在空有不二中如實應對。

3. 「隨眾生心，應所知量」的啟示

同一個如來藏，在不同眾生的心中顯現為不同的世界。這提醒我們：世界的面貌取決於我們的心量與認知。擴大心量、淨化認知，世界自然轉變。

4. 「循業發現」的因果觀

如來藏雖然清淨本然，但眾生的業力會決定如來藏以什麼樣的面貌顯現。善業循善發現，惡業循惡發現。這強化了因果法則在修行中的重要性。
