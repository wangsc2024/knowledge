ChatGPT 不允許輸入直到 Cloudflare 讀取 React State

摘要

Hacker News 熱門文章（367☆/277評），揭露 ChatGPT 網頁版在使用者輸入前，Cloudflare 的安全腳本會先讀取 React 元件的內部狀態。研究者解密了執行此操作的程式，引發隱私與安全討論。

重點

  • Cloudflare Turnstile 在背景讀取 React state 進行 bot 偵測
  • 觸發時機：使用者首次互動前
  • 引發討論：前端狀態是否應被第三方腳本存取
  • 反映趨勢：AI 服務的前端安全防護日益複雜
