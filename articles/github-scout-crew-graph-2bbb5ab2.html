專案觀測摘要

langchain-ai/langgraph (~50k★)

  • runtime_model: 圖式編排（Directed Stateful Graph）：節點=函數/代理，邊=條件轉換；支援 fork/join 並行
  • state_strategy: 內建 checkpointing（thread_id 區隔 session）；time-travel debug；生產環境推薦 PostgreSQL/Redis durable checkpointer
  • human_review_model: interrupt-point 模式 — interrupt() API 暫停圖執行，持久化狀態後等待人工 Command(resume=...) 注入；支援 interrupt_before/interrupt_after 靜態 breakpoints；適合同步短延遲審核

crewAIInc/crewAI (45.9k★)

  • runtime_model: 角色扮演多代理（Agent+Task+Crew）；CrewAI Flows 以 @start/@listen/@router 裝飾器建構 DAG 工作流，支援 parallel kickoff
  • state_strategy: 任務間 context 傳遞；Flows 支援 structured State 持久化；無 time-travel checkpoint
  • human_review_model: human_input=True 任務標記（CLI 輸入等待）；非 interrupt-point 模式；適合非同步 batch 場景

microsoft/agent-framework (GA 2026-04-03, 前身 SK+AutoGen 合計 75k+★)

  • runtime_model: 多模式：sequential/concurrent/handoff/group-chat/Magentic-One；graph-based workflows（SK 傳承）；MCP+A2A 原生支援
  • state_strategy: Agent memory + context providers；checkpoint 機制（AutoGen AgentChat 傳承）；跨 .NET/Python 統一
  • human_review_model: AutoGen HumanProxyAgent 傳承；middleware hooks 實現人工審核注入；middleware-intercept 模式，非 interrupt-point；適合異步高延遲審核
