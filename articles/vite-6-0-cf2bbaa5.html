Vite 6.0 重點摘要（官方部落格彙整）

來源：Vite 6.0 is out!（2024-11-26）。本筆記僅整理官方敘述，供知識庫與靜態站同步管線使用。

發布與採用

  • Vite 6 於 2024-11-26 發布；官方稱為自 Vite 2 以來變動最大的 major。
  • npm 週下載自 Vite 5 發布一年內由約 750 萬增至約 1700 萬（官方數字）。

生態與品牌

  • 官方網域遷移至 vite.dev（舊連結應更新）。
  • Vitest、Storybook 測試、TanStack Start、Ember 等持續擴張生態。
  • Rolldown、Oxc 與 VoidZero 團隊推進 Rust 工具鏈整合（長期路線，非單一版本即完全取代既有 bundler 的聲明）。

Node.js

  • Vite 6 支援 Node.js 18、20、22+；已放棄 Node 21。
  • Node 18 EOL 後官方可能再以新 major 調升最低 Node 版本（官方遷移指南有說明）。

實驗性 Environment API

  • 讓 framework 作者在 dev 更接近 production、並共享新 building blocks；單一 client 環境的 SPA 行為與先前相容。
  • API 仍為實驗性，下一大版前會與生態共同收斂。

升級

  • 多數專案可直接升級，仍應閱讀官方 Migration Guide 與 Breaking Changes 頁。

快速開始

  • pnpm create vite、vite.new 線上試用；另有 pnpm create vite-extra 取得額外模板。
