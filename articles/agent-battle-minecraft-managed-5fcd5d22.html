Agent Battle: Minecraft 挖鑽石競賽 — Managed Agent 建置與 Eval 爬坡（Code with Claude London）

youtuberesearchautoanthropicagentmanaged-agentsevalminecraft

Anthropic Applied AI 團隊（Ben & Jeff）在 Code with Claude London 舉辦的互動競賽工作坊，以 Minecraft 挖鑽石為場景，教授三項核心 Agent 技能。

三大學習目標：

  1. 建置 Claude Managed Agent：用官方 Harness 快速部署受管 Agent，由平台處理基礎設施
  2. 理解 Agent 配置影響：系統提示、模型選擇、Skills 與 MCP 插件如何共同決定 Agent 行為
  3. Eval 爬坡實戰：衡量現有行為 → 迭代改進 → 觀察效果，形成閉環

競賽規則：35 分鐘優化期，每人只計最佳一輪（5 分鐘挖礦），以最多鑽石數定勝負；同分者以 Token 效率 分高下（強調最佳系統提示而非盲目使用最重模型）。

技術細節：使用 MindFlare bot 連接 Minecraft 複製環境，MCP 工具集涵蓋移動、採礦等基礎動作。參與者在 my_agent.py 調整模型、系統提示、Skill 及 MCP 組合。評分腳本每次執行約 1 分鐘可快速迭代。

關鍵觀念：Token 效率是核心指標。盲目選最大模型不如精心設計系統提示；同種 Managed Agent 架構，配置品質決定最終表現。
