摘要

Anthropic 宣布 Claude Opus 4.6 和 Sonnet 4.6 的 1M token context window 從有限預覽進入正式全面開放（GA）。這使得 Claude 能夠處理超長文件、大型程式碼庫分析、長對話歷史等應用場景，是 LLM 實用性的重大突破。

重點

  • 1M token ≈ 約 75 萬字或數百頁文件
  • 從預覽版進入 GA，所有用戶可使用
  • 適用場景：大型程式碼審查、長文件摘要、複雜對話
  • HN 社群高度關注（835 分 / 325 評論）
  • 對 daily-digest-prompt 專案的意義：可處理更大的 context window
