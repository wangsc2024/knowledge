AI 深度研究：世界模型（World Models）— 從語言預測到物理世界模擬的典範轉移

研究日期：2026-02-18

研究類型：AI 深度研究計畫（4 階段完成）

---

摘要

2026 年，AI 世界模型（World Models）從學術概念正式進入產業競賽。以 Google DeepMind 的 Genie 3、Yann LeCun 創辦的 AMI Labs（JEPA 架構）、Fei-Fei Li 的 World Labs（Marble 平台）為代表的三大技術路線同時推進，720p/24fps 即時互動成為技術基準線。世界模型致力於建構物理世界的內部表示，解決大型語言模型（LLM）無法觸及的因果推理與物理一致性問題，被視為通往通用人工智能（AGI）的必要架構。本報告交叉比對 7 篇核心來源，提煉 5 項共識、3 項分歧與 7 項關鍵洞見。

---

一、背景與動機

1.1 LLM 的根本限制

大型語言模型（GPT-4、Claude、Gemini）在文字理解與生成上取得巨大成功，但有一個被數學證明的本質缺陷：統計模式預測無法實現物理真實（Introl Blog 引用 2024 年數學證明）。LLM 部署後不從經驗學習、無法即時更新對世界的理解（Scientific American），這意味著無論規模多大，純語言模型都需要互補架構才能通往 AGI。

1.2 世界模型的定義

世界模型的本質是：維持持續更新的物理環境內部地圖，追蹤物體在時空中的狀態。它不是「看見」世界的快照，而是理解世界的運作規則——物體如何互動、因果如何傳遞、時間如何影響狀態變化。

1.3 從 2D 到 4D 的質變

技術演進路線：NeRF（3D 靜態重建）→ NeoVerse/TeleWorld（4D 動態模擬）。第四維度（時間）的加入意味著模型需要理解因果關係，而非僅重現空間結構。Meta Orion AR 眼鏡已依賴 4D 世界模型實現穩定虛擬物件放置。

二、核心技術架構解析

2.1 三大技術路線

| 路線 | 代表 | 核心架構 | 特點 |

|------|------|---------|------|

| 自迴歸生成式 | Genie 3 (DeepMind) | 自迴歸 Transformer, 110B 參數 | 720p/24fps, 逐幀生成, 物理自學習 |

| 抽象表示式 | AMI Labs (LeCun) | I-JEPA (Joint Embedding Predictive Architecture) | 無標籤學習, 模仿直覺物理認知, 非生成式 |

| 混合實用式 | 混元 1.5 (騰訊) | MoE + 因果注意力 + 少步蒸餾 | 16fps, 延遲 <1 秒, 效能/品質平衡 |

| 持久環境式 | GWM-1 (Runway) | 持久內部環境表示 | 物體持久性, 3 個專用變體 |

| 可編輯 3D 式 | Marble (World Labs) | 可編輯 3D 環境 | $5B 估值, 空間智能 |

2.2 Genie 3 技術細節

  • 解析度：720p, 24fps 即時互動
  • 記憶：視覺記憶回溯達一分鐘
  • 物理學習：水、光照、環境互動為自學習（非硬編碼）
  • 可程式化：支援文字指令即時修改環境（promptable world events）
  • 延伸應用：Waymo 基於 Genie 3 架構專門化自動駕駛模擬，支援多模態輸出（視覺 + LiDAR 點雲）

2.3 AMI Labs / JEPA 路線

  • 創辦者：Yann LeCun（離開 Meta 12 年後創辦，巴黎）
  • 核心主張：JEPA 非生成式 AI，而是學習世界的抽象表示
  • 方法：I-JEPA 無標籤抽象視覺學習，模仿人類直覺物理認知
  • 估值：尋求 €5 億融資，€3B 估值
  • 哲學：「擴展 LLM 不會產生通用智能，世界模型才是正確路線」

2.4 Runway GWM-1

  • 三個專用變體：GWM Worlds（可探索環境）、GWM Avatars（音訊驅動角色）、GWM Robotics（機器人訓練資料）
  • 核心突破：物體持久性（轉身後物件位置不變、物理一致、光照隨相機移動）
  • 成績：Video Arena 排行榜第一（超越 Veo 3 和 Sora 2）

三、關鍵洞見

洞見 1：LLM 幻覺的數學不可解性為世界模型提供理論基礎（高重要性）

2024 年的數學證明指出統計模式預測在本質上無法實現物理真實，這不是工程問題而是原理性限制。意味著無論 LLM 規模多大，都需要世界模型作為補充架構。

洞見 2：計算成本鴻溝將決定商業化路線（高重要性）

世界模型推理需 8-32 GPU（LLM 僅 1-8），訓練需數萬 GPU。短期內世界模型只會以 API 服務形式存在（如 Waymo 模擬平台），而非本地部署。

洞見 3：4D 時空模型是質變而非量變（高重要性）

從 3D 靜態重建到 4D 動態模擬代表根本性突破。時間維度的加入意味著模型需要理解因果關係，而非僅重現空間結構。

洞見 4：NVIDIA Cosmos 正成為世界模型的 CUDA（中重要性）

9000 兆 tokens 訓練、200 萬次下載。正如 CUDA 在 LLM 時代扮演的角色，Cosmos 可能成為世界模型時代的基礎設施壟斷者。

洞見 5：「可程式化物理世界」開啟全新計算範式（高重要性）

Genie 3 的 promptable world events 和 GWM-1 的環境持久性，意味著我們正在從「程式化資訊世界」（軟體工程）走向「程式化物理世界」（世界工程）。這是計算範式的典範轉移。

洞見 6：混合架構代表實用主義路線的崛起（中重要性）

混元 1.5 不走純路線而是混合多種技術（MoE + 因果注意力 + 蒸餾），代表中國團隊走「夠用就好」的實用化路線。

洞見 7：開源世界模型可能重演 LLM 的 Llama 效應（中重要性）

螞蟻靈波 LingBot-World 開源（10 分鐘級生成 + 場景點雲轉化），可能在工業模擬、遊戲開發等垂直領域率先突破。

四、分歧與爭論

4.1 生成式 vs 表示式：智能的哲學之爭

  • 生成式（DeepMind, Runway）：直接生成像素/幀，產出可見的視覺內容
  • 表示式（LeCun / AMI Labs）：學習世界的抽象表示而非生成內容，模仿人類直覺物理認知
  • 啟示：這是對「智能本質」的哲學分歧——智能是否需要「看見」世界？

4.2 開源 vs 閉源

  • 開源陣營：中國廠商（螞蟻靈波）、NVIDIA Cosmos 公共基礎設施
  • 閉源陣營：AMI Labs、DeepMind、World Labs（估值 $3-5B）
  • 趨勢：開源世界模型可能像 Llama 一樣重塑競爭格局

4.3 語言介面的角色

  • 語言控制派（Genie 3, Waymo）：文字指令即時修改環境
  • 超越語言派（JEPA）：語言是理解世界的瓶頸而非助力

五、實務應用建議

5.1 對 Daily-Digest-Prompt 專案的啟示

  1. Agent 記憶架構：世界模型的「持續更新內部表示」概念可借鑑到 digest-memory.json 的設計——不只記錄事實，而是維持一個持續更新的「世界狀態」
  2. 物理一致性 → 邏輯一致性：世界模型追求物理一致性的方法（因果推理、時空追蹤），可類比到自動任務的邏輯一致性檢查（研究去重、主題輪替）
  3. 多模態趨勢：未來 Agent 可能需要處理視覺輸入（截圖分析、圖表理解），關注 Genie 3 的多模態架構演進

5.2 技術社群應關注的方向

  1. NVIDIA Cosmos API：作為世界模型的「CUDA」，值得追蹤其 API 開放進度
  2. Waymo 世界模型模擬平台：最成熟的商業應用，可能最先提供公開 API
  3. 開源世界模型：LingBot-World 等開源專案，適合小團隊研究與實驗
  4. JEPA 架構論文：LeCun 的路線雖然非主流，但理論基礎最強，值得持續關注

七、未來展望

  1. 2026 下半年：Genie 3 可能開放公開 API，Waymo 模擬平台商業化
  2. 2027 預測：AMI Labs 的 JEPA 路線若能展示超越生成式的理解能力，可能引發技術路線大轉向
  3. 長期趨勢：世界模型 + LLM + Agent 三位一體架構可能成為 AGI 的標準組合
  4. 風險：計算成本過高可能導致只有少數科技巨頭能參與，加劇 AI 發展的集中化
