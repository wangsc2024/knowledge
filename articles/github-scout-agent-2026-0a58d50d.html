專案觀測摘要

microsoft/agent-framework（11,367★）

觀測日期：2026-06-16 | 來源：GitHub REST API（api.github.com/repos/microsoft/agent-framework）

最近推送：2026-06-16T00:48:46Z | 相關性：High

2026-04-03 GA，整合 AutoGen + Semantic Kernel。支援 sequential/concurrent/handoff/group-chat 四種多代理模式。

  • runtime_model：多代理編排，圖式 + 事件驅動混合
  • state_strategy：Checkpointing + durable execution，支援 pause/resume 長任務
  • human_review_model：handoff-based HITL — agents 透過 explicit handoff 轉移控制權，HITL 為高層抽象 approval callback。適合多代理委派審查場景，對本系統 Phase 1a dispatch 語義有直接借鑑價值。

agentscope-ai/agentscope（26,873★）

觀測日期：2026-06-16 | 來源：GitHub REST API（api.github.com/repos/agentscope-ai/agentscope）

最近推送：2026-06-15T04:30:16Z | 相關性：High

設計哲學「agents you can see, understand and trust」，以可觀測性為核心。

  • runtime_model：事件驅動，unified event bus 架構，分散式多代理
  • state_strategy：事件溯源，狀態透過 event bus 流動，前端可即時監控
  • human_review_model：interrupt-point 模式 — human agent 節點插入事件流，等待 approve/edit 後繼續，適合同步短延遲審核

pydantic/pydantic-ai（17,775★）

觀測日期：2026-06-16 | 來源：GitHub REST API（api.github.com/repos/pydantic/pydantic-ai）

最近推送：2026-06-16T02:52:43Z | 相關性：Medium-High

Pydantic 型別安全 agent 框架，整合 A2A + MCP。

  • runtime_model：單代理型，無內建多代理編排
  • state_strategy：Pydantic 型別系統確保狀態合規，無 checkpoint
  • human_review_model：signal-based HITL — tool_approval 機制，工具執行前傳 signal 等待 operator approve，適合非同步高延遲審核

P1-1：借鑑 microsoft/agent-framework handoff-based HITL 強化 Phase 1a dispatch

來源：microsoft/agent-framework | 風險：中

execution_contract：

  • operation_type: config_update
  • idempotency_required: true
  • human_review_required: true
  • run_metadata: {trace_id: github_scout-20260616-r11, task_key: github_scout, operation_name: daily_digest.github_scout.review}
