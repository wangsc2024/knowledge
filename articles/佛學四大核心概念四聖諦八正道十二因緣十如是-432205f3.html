佛學四大核心概念：四聖諦、八正道、十二因緣、十如是

2026-02-03
佛學四聖諦八正道十二因緣十如是緣起佛教基礎

佛學四大核心概念及其關連

四聖諦（Four Noble Truths）

佛陀初轉法輪時所說的核心教義：

  1. 苦諦 - 人生的本質是苦（生、老、病、死、愛別離、怨憎會、求不得、五蘊熾盛）
  2. 集諦 - 苦的原因是貪愛與執著
  3. 滅諦 - 苦可以被滅除（涅槃）
  4. 道諦 - 滅苦的方法即「八正道」

八正道（Eightfold Path）

四聖諦中「道諦」的具體內容，分為三學：

戒學

  • 正語：不妄語、不惡口、不兩舌、不綺語
  • 正業：不殺生、不偷盜、不邪淫
  • 正命：以正當方式謀生

定學

  • 正精進：努力斷惡修善
  • 正念：保持覺知，念住四念處
  • 正定：修習禪定，達到心一境性

慧學

  • 正見：正確理解四聖諦
  • 正思惟：離欲、無瞋、不害之思惟

十二因緣（Twelve Links of Dependent Origination）

說明生死輪迴的因果鏈：

無明 → 行 → 識 → 名色 → 六入 → 觸 → 受 → 愛 → 取 → 有 → 生 → 老死

三世因果

  • 過去因：無明、行
  • 現在果：識、名色、六入、觸、受
  • 現在因：愛、取、有
  • 未來果：生、老死

十如是（Ten Suchnesses）

出自《法華經》，描述諸法實相的十個面向：

相、性、體、力、作、因、緣、果、報、本末究竟等

這是從「空」的角度觀察一切法的完整面貌。

四者之間的關連

核心原理：緣起性空

  1. 十二因緣解釋「苦」如何產生（集諦的詳細展開），無明是苦的根源
  2. 四聖諦是診斷與治療的框架：發現病症（苦）→找出病因（集）→確認可治癒（滅）→開出藥方（道）
  3. 八正道是「道諦」的具體實踐方法
  4. 十如是從法華經的角度，說明一切法的真實面貌，是更深層的實相觀

簡要總結

  • 十二因緣說明「為何輪迴」
  • 四聖諦說明「如何解脫」
  • 八正道說明「怎樣修行」
  • 十如是說明「諸法實相」

四者皆以「緣起」為核心：因為一切法因緣而生、因緣而滅，所以苦可以被滅除，八正道才能有效。
