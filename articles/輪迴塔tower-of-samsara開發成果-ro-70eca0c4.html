輪迴塔（Tower of Samsara）開發成果

遊戲概述

  • 名稱：輪迴塔 — Tower of Samsara
  • 類型：Roguelike 無限攀塔 + 禪意美學
  • 路徑：d:/source/game/samsara-tower/
  • 部署：Cloudflare Pages 自動部署

核心創意

結合佛教六道輪迴概念與 Doodle Jump 式平台跳躍。玩家操控修行者攀登無限高塔，場景隨分數從「地獄道」漸變至「天道」。收集禪字獲得覺悟加持（道具），掉落即墮入輪迴（遊戲結束）。

技術亮點

  1. Fixed Timestep + rAF：穩定 60 FPS，跨裝置一致
  2. 物件池系統：200 粒子池 + 60 平台池，零 GC 壓力
  3. Web Audio API 8-bit 合成：7 種音效（跳躍、落地、收集、覺悟、死亡、選單、道變換）
  4. 完整狀態機：MENU → PLAYING → PAUSED → GAMEOVER
  5. 三輸入支援：鍵盤（WASD/方向鍵/Space）+ 觸控（滑動+點擊）+ 滑鼠
  6. 水平環繞：左出右進，增加策略深度
  7. 漸進難度：隨時間自動增加障礙物出現率

平台類型

  • 普通平台：穩定跳板
  • 崩塌平台：踩上後短暫計時崩塌
  • 移動平台：左右往復移動
  • 尖刺平台：接觸即死（護盾可免疫）

道具系統

  • 二段跳（++）：空中可再跳一次
  • 護盾（()）：10 秒內免疫尖刺
  • 加速（>>）：5 秒移動速度提升

上癮設計

  1. Roguelike 隨機性 — 每次平台配置不同
  2. 六道場景漸變 — 視覺里程碑驅動
  3. 禪字收集 — 滿足收集欲
  4. 即時重玩 — 死亡後零等待
  5. 最高分/最高層雙排行
