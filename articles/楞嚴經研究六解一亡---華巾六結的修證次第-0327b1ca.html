楞嚴經研究：六解一亡 - 華巾六結的修證次第

主題概述

「六解一亡」是《楞嚴經》第五卷中極為精要的修行法門，佛陀以華巾打結的譬喻，說明眾生六根的結縛及其解除方法。此法門承接「二決定義」，為具體的修證次第指導，最終導向二十五圓通的證入。核心義理在於：六根本為一體（同一清淨覺性），因虛妄分別而成六結，依次第解除後，六結盡亡，本元真性即顯——這就是「六解一亦亡」的深義。

經文重點段落

華巾六結譬喻（第五卷）

如來從師子座上起立整衣斂容，從七寶几上取來劫波羅天人奉獻的華巾，就在大眾面前將華巾挽成一個結。如來問阿難：「這叫做什麼？」阿難和大眾回答：「這叫做結。」如來又在華巾上疊挽一結，如此一層層挽成六個結。

白話翻譯：佛陀用一條華巾打了六個結，說明本來只是一條華巾（喻一心真如），但因為六次綰結（喻六根攀緣六塵），就產生了六個不同的結（喻六根各成障礙）。

六解一亡偈頌

「結解同所因，聖凡無二路。審觀根塵間，既不是空亦不是有。迷惑晦昧即是無明，顯發明悟便得解脫。解纏脫縛依次循進，六根解脫一根亦無。」

「解結因次第，六解一亦亡，根選擇圓通，入流成正覺。」

白話翻譯：結的形成與解除同出一因（皆由心起），聖道與凡途本無二路。細審根與塵的關係，既非全空也非實有。迷惑昏昧就是無明，覺醒明悟就是解脫。解開纏縛要依次第循進，六根都解脫了，連「一」（能解之主體）也歸於無。選擇適合的根門修圓通，入流（返聞自性）即可成就正覺。

義理解析

1. 根塵同源，縛脫無二

這是六解一亡的根本義理。「根」（六根）與「塵」（六塵）看似相對，實則同源於一心真如。捆縛與解脫也是一件事——同樣的六根，向外攀緣即是結縛，返觀自性即是解脫。這與《心經》「色即是空，空即是色」相呼應，強調煩惱與菩提不二、生死與涅槃同源。

佛學術語解釋：

  • 六根：眼、耳、鼻、舌、身、意——六種感知器官及其功能
  • 六塵：色、聲、香、味、觸、法——六根所對應的外境
  • 根塵相對：眼見色、耳聞聲……六根各攀六塵，形成十二處

2. 六結的形成與解除次第

佛陀打結的順序是「由內而外」（從中心向兩端），解結則須「從外而內」（從最後打的結開始解）。這象徵：

  • 結的形成：最初一念無明（根本結）→ 妄想分別 → 攀緣六塵 → 習氣深重
  • 解的次第：先解粗重習氣（外層結）→ 漸入微細無明 → 最終破根本無明

3. 六結對應六根（依觀音耳根圓通）

4. 入流亡所——觀音法門的關鍵

「入流」指心識不向外流散（出流），而是返歸覺性之流（入流）。「亡所」指超越對外境（所聽之聲）的執著。這是觀音法門的核心原則：

  • 出流：順十二因緣，流浪生死
  • 入流：逆十二因緣，返妄歸真

具體修法：以聽聲音為修行對象，但不是聽外在的聲音，而是反聞聞自性（向內聽能聞的覺性）。動靜二相來去，聞性不動——這個不動的聞性就是本覺。

與已有研究的關聯

承上

  • 七處徵心、十番顯見（破妄）→ 知道妄心不可得、見性本自具足
  • 八還辨見（辨真）→ 分辨見性與塵境，見性不可還
  • 如來藏、三科七大（顯真）→ 一切法皆如來藏妙真如性
  • 二決定義（審因）→ 以不生滅心為本因，審煩惱根本

啟下

  • 四種清淨明誨：戒律是解結的基礎，持戒清淨才能有效修定
  • 二十五圓通：六解一亡說明解結原理，二十五圓通是具體的選根修法
  • 五十陰魔：解六結過程中可能遭遇的魔境，對應破五陰的各階段

本篇定位

六解一亡是「二決定義」與「二十五圓通」之間的橋樑：二決定義指出「以不生滅心為因」的原則，六解一亡說明「如何依次第解開六根的結縛」，二十五圓通則是「選擇哪一根門來具體修證」的實例。

修行應用

1. 日常生活中的「入流亡所」

不必等到打坐時才修，隨時可以練習：

  • 聽聲音時：注意力從「聲音的內容」轉向「聽的能力本身」
  • 覺察習慣：發現自己追逐聲塵時，輕輕返回能聞的覺性
  • 不壓不隨：不壓制聲音（修死定），也不跟隨聲音（散亂心）

2. 解結的漸進原則

  • 粗先細後：先處理明顯的習氣（如瞋恨、貪欲），再對治微細的執著
  • 不急不躁：結是「次第而結」，解也要「次第而解」，不可強行斷除
  • 覺知即解：每一個覺知的當下，結就在鬆動——重點是持續覺知

3. 與「原子習慣」的對照

習慣的形成與結的形成有相似之處：

  • 習慣：提示 → 渴望 → 回應 → 獎賞（循環強化）
  • 六結：無明 → 攀緣 → 執著 → 習氣（循環強化）

解除之道也類似：

  • 習慣：讓提示不可見、讓行為不吸引、讓執行困難、讓獎賞不滿足
  • 六結：覺知攀緣、不隨妄動、返聞自性、漸破執著
