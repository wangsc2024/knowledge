Claude Code Ultra：多代理並行工作流實踐

youtuberesearchClaude CodeAI工具工作流自動化多代理系統

核心發現

Claude Code 新增「Ultra Code」功能，結合深度思考（Extra High level）與動態工作流（Dynamic Workflows），能同時執行多達 100 個智能代理並行處理任務，具有自動事實檢查機制。

工作原理

  • Extra High 思考層級：深度思考強度介於 Max 與普通層級之間
  • 動態工作流：Claude 先制定計畫，再生成多個代理並行執行任務的不同部分
  • 質量控制：代理相互驗證結果後方才回傳

適用場景

  • 複雜多部分研究任務（如創業準備清單、商業計畫）
  • 需要並行資訊蒐集的決策
  • 需要事實檢查的內容生成

啟用方式

從 /effort 選單選擇 Ultra Code，設定對該會話所有後續任務生效。

成本考量

Ultra Code 是 Claude Code 最昂貴的執行模式，因同時運行多個代理且使用 Extra High 思考層級。可用 /usage 指令查看成本。

實測效果

實驗：研究食品卡車創業準備，Ultra Code 並行搜尋許可證、設備成本、供應商等，生成完整且事實檢查過的創業指南。
