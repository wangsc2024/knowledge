楞嚴經與圓覺經的修行體系會通 — 兩經在如來藏思想、觀行次第與證果境界的異同與互補共學指南

楞嚴經與圓覺經的修行體系會通 — 兩經在如來藏思想、觀行次第與證果境界的異同與互補共學指南

一、引言

楞嚴經與圓覺經同為大乘如來藏系經典，在中國佛教思想史上佔有極其重要的地位。楞嚴經以「破妄顯真」為綱領，透過七處徵心、十番顯見等層層剖析，揭示眾生本具的如來藏妙真如性；圓覺經則以「悟圓覺性」為核心，透過十二位菩薩的問答，闡明眾生本具的圓覺妙心及其修證次第。

本研究承接知識庫中既有的楞嚴經系列研究（包括與天台宗、唯識學、華嚴經的會通），進入更深層的跨經典對話。本文聚焦於兩經在如來藏思想、觀行次第與證果境界三個面向的異同，並提供互補共學的實修指南。

二、如來藏思想的同源異名

2.1 圓覺經的如來藏論

圓覺經開宗明義即指出：「一切眾生本來成佛」，此本具的佛性在因地稱為「如來藏」，在果地稱為「圓覺」。經中文殊菩薩章云：「一切眾生種種幻化，皆生如來圓覺妙心」，點明眾生與佛無二無別，差異僅在迷悟。

圓覺經將覺性分為五個層次：

  1. 本覺（原始具足的覺性）
  2. 始覺（開始覺悟）
  3. 正覺（正確的覺悟）
  4. 分證覺（部分證得）
  5. 等覺（接近圓滿）
  6. 妙覺（圓滿覺悟，即圓覺）

這一覺性開展的次第，顯示如來藏從潛在到顯現的完整歷程。

2.2 楞嚴經的如來藏論

楞嚴經卷三「會通四科圓彰七大」段落，將五陰、六入、十二處、十八界以及地水火風空見識七大，一一會歸「如來藏妙真如性」。經云：「如來藏中，性色真空，性空真色，清淨本然，周遍法界」，揭示如來藏具足一切法，而一切法不離如來藏。

楞嚴經的如來藏思想特色在於：

  • 破妄顯真：透過七處徵心破除妄心，十番顯見顯示真見
  • 性相不二：如來藏非空非有，非因緣非自然，超越二元對立
  • 即妄即真：五陰六入等世間法，當體即是如來藏
  • 修證一如：因地心與果地覺相應，審因心果覺

2.3 同源異名的會通

印順法師指出，楞嚴經與圓覺經同屬晚期真常唯心論經典。兩經的核心義理可會通如下：

| 圓覺經 | 楞嚴經 | 華嚴經 | 天台宗 |

|--------|--------|--------|--------|

| 圓覺 | 如來藏 | 一真法界 | 實相 |

| 本覺 | 妙覺明心 | 法性 | 中道實相 |

| 幻化 | 客塵 | 無明 | 無明惑 |

兩經都強調「這念心本自具足，一切法不離自心」。圓覺一法與如來藏性，名殊體一，無非一心清淨本覺。差異在於表述方式：圓覺經以「覺」為名，強調覺性的圓滿具足；楞嚴經以「藏」為名，強調真性的含藏周遍。

三、觀行次第的頓漸會通

3.1 圓覺經的三觀二十五輪

圓覺經的修行體系以「三觀」為基礎，衍生「二十五輪」：

三觀（三種禪修法門）：

  1. 奢摩他（止/空觀）：「停」義，以靜為行相，對應空觀。修此取靜為行的方便。
  2. 三摩缽提（觀/假觀）：「等持」義，以幻化為行相，對應假觀。修此如幻觀行的方便。
  3. 禪那（中道觀）：「靜慮」義，寂幻雙照，對應中道觀。修此定慧平均觀行的方便。

二十五輪（辯音菩薩章）：

依三觀衍生 25 種清淨定輪，包括：

  • 單修：獨修一觀（3 種）
  • 雙修：合修二觀（3×2 = 6 種，再考慮先後順序共 12 種）
  • 圓修：三觀齊修（10 種，包含不同次第組合）

經云：「以能修之機不一，致所修之門亦多」，強調「適合自己即是好法門」，體現圓覺經的圓融無礙特色。

3.2 楞嚴經的耳根圓通與二十五圓通

楞嚴經卷五、卷六記載二十五位聖者各陳圓通法門，涵蓋六塵、六根、六識、七大，最後由文殊菩薩揀選「耳根圓通」為最適合此方眾生之法門。

觀世音菩薩耳根圓通（六結解脫）：

  1. 動靜二相，了然不生（解動結）
  2. 聞所聞盡（解靜結）
  3. 盡聞不住（解根結）
  4. 覺所覺空（解覺結）
  5. 空所空滅（解空結）
  6. 生滅既滅，寂滅現前（解滅結）

耳根圓通的核心是「反聞聞自性，性成無上道」，從聞性入手，層層深入，直至能所雙亡。

楞嚴修證體系（卷八五十五位真菩提路）：

  • 乾慧地（伏惑未斷）
  • 十信、十住、十行、十迴向、四加行
  • 十地、等覺、妙覺

3.3 頓漸會通的要點

| 面向 | 圓覺經 | 楞嚴經 |

|------|--------|--------|

| 修行取向 | 頓悟為主，漸修為輔 | 理頓事漸，頓悟漸修 |

| 入手方法 | 觀「幻化」本質 | 從「聞性」契入 |

| 法門結構 | 三觀二十五輪（橫向擴展） | 二十五圓通（縱向深入） |

| 證果次第 | 本覺→始覺→妙覺 | 乾慧→十信→等覺→妙覺 |

普賢菩薩章的頓悟精神：

圓覺經普賢章云：「知幻即離，不作方便；離幻即覺，亦無漸次。」這是圓覺經頓教特色的典型表述。只要當下認知諸法如幻，立即離執，覺性自然顯現，無需漸次修行。

普眼菩薩章的漸修方便：

但圓覺經也並非純粹頓教。普眼菩薩章為初學者提供「修行漸次、安居修證」的具體方便，包括三期安居、懺悔方法、道場儀軌等，顯示圓覺經頓漸並重的圓融立場。

與楞嚴經的會通：

  • 楞嚴經卷一「審因心果覺」即是圓覺經「以果地覺為因地心」的同義表述
  • 楞嚴經「二決定義」中的「審煩惱根本」，呼應圓覺經「離幻」的方法論
  • 楞嚴經的「三種漸次」（除助因、刳正性、違現業）對應圓覺經「堅持禁戒」的修行基礎
  • 楞嚴經耳根圓通的「反聞聞自性」與圓覺經「圓照清淨覺相」異曲同工，都是以覺照為修行核心

四、四相與五十陰魔的辨證關聯

4.1 圓覺經的四相

圓覺經文殊菩薩章揭示兩種四相：

（一）迷識四相（依第六識、枝末無明）：

  1. 我相：執有實我
  2. 人相：執有他人
  3. 眾生相：執有輪迴主體
  4. 壽者相：執有時間延續

此四相障礙小乘涅槃，使眾生無法出離生死輪迴。

（二）迷智四相（依第八識、根本無明）：

  1. 我相（更深層）：證道者仍有能證之我執
  2. 人相：有所度化的對象
  3. 眾生相：有修行的主體
  4. 壽者相：執著於證悟的階位

此四相障礙大乘菩提，使修行者無法究竟成佛。即使已證阿羅漢果，若未破此四相，仍無法成就無上菩提。

4.2 楞嚴經的五十陰魔

楞嚴經卷九、卷十詳述五陰（色受想行識）各有十種陰魔，共五十種。這是禪修過程中可能遭遇的各種幻境、錯覺與偏差：

色陰十魔：如見身外有身、內徹骨髓等身心變異

受陰十魔：如悲魔、狂魔、憶魔等情緒極端

想陰十魔：如貪求神通、執著古聖等妄想增長

行陰十魔：如執著不生不滅、執著因緣自然等哲學偏差

識陰十魔：如執著空無、執著永恆等見地錯謬

經云：「不作聖心，名善境界；若作聖解，即受群邪。」

4.3 兩經辨魔體系的會通

圓覺經的四相與楞嚴經的五十陰魔，都是修行過程中的障道因緣，但角度不同：

| 面向 | 圓覺經四相 | 楞嚴經五十陰魔 |

|------|-----------|---------------|

| 層次 | 根本執著（我法二執） | 修證過程的偏差 |

| 來源 | 無明煩惱（深層） | 禪定境界的執著（淺層） |

| 對治 | 般若智慧照見 | 覺照不執、不作聖解 |

| 影響 | 障涅槃與菩提 | 入魔障道 |

互補關係：

  • 圓覺經四相是本（根本煩惱），五十陰魔是末（枝葉偏差）
  • 破除迷識四相對應破除色受想三陰魔，破除迷智四相對應破除行識二陰魔
  • 圓覺經提供原理性的覺悟框架，楞嚴經提供操作性的辨魔指南

共學建議：

  1. 先學圓覺經四相，建立正見（知道執著的根本在哪裡）
  2. 再學楞嚴經五十陰魔，學會辨識（知道修行過程會遇到什麼陷阱）
  3. 兩經合參，達到「知幻即離」（圓覺）與「不作聖解」（楞嚴）的修證智慧

五、兩經修行體系的異同總結

5.1 共同基礎

| 項目 | 內容 |

|------|------|

| 理論基礎 | 如來藏思想，一心本具 |

| 修行目標 | 破妄顯真，證得本覺 |

| 核心方法 | 覺照觀心，離念契真 |

| 障道因緣 | 無明執著，妄想顛倒 |

5.2 主要差異

| 面向 | 圓覺經 | 楞嚴經 |

|------|--------|--------|

| 經文風格 | 簡潔直接，頓悟色彩濃 | 層層剖析，理路綿密 |

| 入手方法 | 觀諸法如幻，知幻即離 | 從聞性入手，反聞自性 |

| 修證次第 | 本覺→始覺→妙覺（較簡） | 乾慧→五十五位（極詳） |

| 法門結構 | 三觀二十五輪（橫向廣） | 二十五圓通（縱向深） |

| 辨魔體系 | 四相（根本執著） | 五十陰魔（過程偏差） |

5.3 互補關係

楞嚴經補圓覺經之處：

  1. 修證階位更詳細：五十五位提供完整的菩薩道路線圖
  2. 辨魔更具體：五十陰魔提供實修過程的詳細預警
  3. 破妄更徹底：七處徵心、十番顯見的層層剖析
  4. 戒律更嚴格：四種清淨明誨提供明確的行為規範

圓覺經補楞嚴經之處：

  1. 修行更靈活：二十五輪提供多種法門選擇
  2. 頓悟更直截：知幻即離，不作方便
  3. 覺性論更明：本覺始覺妙覺的覺性開展
  4. 圓融更顯著：單修雙修圓修，隨根器選擇

六、互補共學的實修指南

6.1 初學入門路徑

建議順序：

  1. 先學圓覺經文殊章與普賢章，建立「一切眾生本來成佛」的正見，了解「知幻即離」的頓悟精神
  2. 再學楞嚴經七處徵心與十番顯見，透過邏輯辨析，深化對「妄心」與「真心」的理解
  3. 回到圓覺經普眼章，學習具體的修行方便與安居方法
  4. 進入楞嚴經二十五圓通，選擇適合自己的法門深入

原理：先建立頓悟的高度（圓覺經），再透過漸修的深度（楞嚴經）落實，最後回到圓融無礙（圓覺經）。

6.2 中級修證路徑

從圓覺經入楞嚴經：

  1. 先修圓覺經三觀（奢摩他、三摩缽提、禪那）
  2. 對應楞嚴經的三種法門：空觀（破妄）、假觀（顯真）、中觀（不二）
  3. 選擇楞嚴經二十五圓通中與三觀相應的法門深入（如耳根圓通對應奢摩他）

從楞嚴經入圓覺經：

  1. 先修楞嚴經耳根圓通「反聞聞自性」
  2. 證得「聞所聞盡」後，進入圓覺經「知幻即離」的境界
  3. 用圓覺經二十五輪擴展修法，不局限於耳根一門

6.3 高級整合路徑

融合兩經的日常觀照：

早課（重破妄）：

  1. 誦楞嚴經「七處徵心」段落，提醒自己不認妄為真
  2. 用楞嚴經「審因心果覺」檢查今日發心是否清淨

日中（重覺照）：

  1. 遇境界時，用圓覺經「知幻即離」：認知眼前境界如夢如幻
  2. 遇煩惱時，用楞嚴經「客塵」觀：煩惱是客，覺性是主

晚課（重反省）：

  1. 對照楞嚴經五十陰魔，檢查今日有無執著禪定境界
  2. 對照圓覺經四相，檢查今日有無我執法執
  3. 以圓覺經「圓照清淨覺相」安住，準備入睡

6.4 兩經會讀的四項原則

原則一：以覺性為本，不執名相

  • 圓覺、如來藏、佛性、真心、覺性，名雖不同，體是一個
  • 不因經典用詞差異而生分別心

原則二：頓悟漸修，理事並重

  • 先從圓覺經建立頓悟見地「本來成佛」
  • 再從楞嚴經學習漸修次第「五十五位」
  • 不偏空（只談理不修行），不偏有（只重事相不悟理）

原則三：廣學法門，一門深入

  • 兩經合計有 28 種圓通法門（楞嚴 25 + 圓覺 3 觀）
  • 廣學是為了了解，深入是為了證悟
  • 選一法門（如耳根圓通或奢摩他）終身不易

原則四：辨魔護念，不作聖解

  • 圓覺經教「離幻」，楞嚴經教「辨魔」
  • 修行有所體驗時，先用兩經標準檢驗
  • 凡有所執，即是偏差；不作聖解，魔不能擾

七、兩經在天台宗的判教定位

7.1 智者大師的判教

天台宗將佛法判為「藏、通、別、圓」四教：

  • 圓覺經：判為圓教
  • 楞嚴經：蕅益智旭判為圓教，但也含別教成分

7.2 兩經與天台止觀的關係

天台止觀有三種：

  1. 漸次止觀（25 方便 + 10 境 10 乘）
  2. 不定止觀（四種三昧）
  3. 圓頓止觀（一心三觀）

對應關係：

  • 圓覺經三觀 ≈ 天台一心三觀（空假中）
  • 楞嚴經耳根圓通 ≈ 天台常行三昧（般舟三昧）
  • 圓覺經二十五輪 ≈ 天台不定止觀

詳細的天台會通，可參考知識庫已有的「楞嚴經與天台宗的義理會通」與「教觀綱宗研究」系列。

八、結論：從楞嚴入圓覺的修學路徑建議

8.1 為何先楞嚴後圓覺？

  1. 理路層次：楞嚴經破妄更徹底，適合建立正見基礎
  2. 辨魔體系：先學五十陰魔，避免修圓覺經時執著境界
  3. 修證階位：先了解五十五位，知道自己在哪個階段
  4. 戒律基礎：四種清淨明誨提供明確規範，為修圓覺經打基礎

8.2 完整修學路線圖

第一階段（6 個月 - 1 年）：楞嚴基礎

  1. 熟讀卷一至卷四（七處徵心、十番顯見、會通四科）
  2. 建立「破妄顯真」的正見
  3. 受持四種清淨明誨（淫殺盜妄）

第二階段（1 - 2 年）：圓覺融通

  1. 熟讀圓覺經十二章
  2. 學習三觀理論，選一觀深入
  3. 結合楞嚴與圓覺，選定一門圓通法門

第三階段（3 年以上）：一門深入

  1. 專修選定法門（如耳根圓通 + 奢摩他）
  2. 用楞嚴經五十陰魔辨魔
  3. 用圓覺經四相檢查執著
  4. 依楞嚴五十五位檢驗證量

第四階段（終身）：圓融無礙

  1. 兩經義理融會貫通
  2. 日常起心動念皆是觀照
  3. 行住坐臥不離覺性
  4. 隨緣度眾，不離本心

8.3 最後提醒

印光大師云：「欲得佛法實益，須向恭敬中求。有一分恭敬，則消一分罪業，增一分福慧。」

學習楞嚴與圓覺兩經，不在於通曉文字、會講道理，而在於：

  1. 深信：深信自己本來成佛（圓覺經）、本具如來藏（楞嚴經）
  2. 真幹：選定一門，終身不易，老實修去
  3. 覺照：時時覺照這念心，不隨妄境流轉
  4. 離執：知幻即離（圓覺）、不作聖解（楞嚴）

願諸同修依此兩經，破妄顯真，證得圓覺，同登彼岸。
