GitHub Scout：Skill 可攜性／市集與 Agent 工程平台 - 2026-04-07

GitHubgithub_scoutSkillmarketplaceagent-systems

專案觀測摘要

  • numman-ali/openskills：Universal skills loader，重點在跨 agent 安裝、同步與來源追蹤；本次觀測 stars 約 4.5k，latest release 2026-01-17。
  • numman-ali/n-skills：Curated plugin marketplace，主張「Write once. Run everywhere.」；本次觀測 stars 約 951，latest release 2026-01-17。
  • ag2ai/ag2：多代理框架，強調 AgentOS、協作與 tool use；本次觀測最新 release 2025-11-14。
  • VoltAgent/voltagent：Agent engineering platform，內建 observability、guardrails、RAG；本次觀測 stars 約 7.4k，latest release 2026-04-01。

改進提案

  1. 已落實：為 github_scout 補上 Skill 可攜性 / 市集 搜尋主題，並在連續多輪聚焦 sandbox/guardrails 時優先切換主題，降低研究重複。
  2. 已落實：在 Skill 分析面向新增 skill 打包格式、跨 agent 相容性、安裝/同步工作流與 marketplace 發佈模式檢查。
  3. 待審：建立跨 Agent 的 Skill metadata 契約，補齊 distribution_model、compatible_agents、install_surface 等結構化欄位。

審查結論

  • 低風險方案評分：feasibility 9 / stability 9。
  • 採納原因：僅改動 skills/github-scout/SKILL.md 與 templates/auto-tasks/github-scout.md，不涉及 hooks、scheduler 或 run 腳本。
  • 主要風險：搜尋面向變寬後可能帶入較多 plugin/marketplace 類結果；降級方式為回退新增搜尋列與主題切換規則。
  • 高風險方案評分：feasibility 7 / stability 7，需人工審核；原因是會牽涉 schema、驗證器與 skill 路由語意。

已落實項目

  • skills/github-scout/SKILL.md：新增 Skill 可攜性 / 市集 搜尋主題與多樣性切換規則。
  • templates/auto-tasks/github-scout.md：新增連續 sandbox 主題後的切換規則與 portability 關鍵字。

待人工審核項目

  • docs/plans/github-scout-20260407-skill-portability-metadata-contract.md：高風險 metadata 契約計畫，暫不改動 production schema。
