Dig Dug (挖金礦) v1.0 創建心得

遊戲概述

經典 Dig Dug 復古遊戲 HTML5 重製版，玩家在地底挖掘隧道、用充氣幫浦消滅怪物（Pooka 和 Fygar）。

技術架構

  • 單一 HTML 檔案（約 750 行 JS），Canvas 2D 渲染
  • Fixed Timestep 遊戲迴圈：requestAnimationFrame + accumulator，FIXED_DT = 1000/60
  • Delta time 上限 50ms 防止大跳幀
  • 完整狀態機：LOADING → MENU → PLAYING → PAUSED → DYING → LEVEL_CLEAR → GAME_OVER

核心機制實作

#### 1. Grid-based 地形系統

  • 15x15 Grid，三種地形：天空(2)、泥土(0)、隧道(1)
  • 玩家移動時自動挖掘泥土為隧道
  • 挖掘時產生粒子效果回饋

#### 2. 敵人 AI（雙類型）

  • Pooka（紅色圓球怪）：在隧道中巡邏，隨機方向但偏向玩家
  • Fygar（綠色龍怪）：額外噴火能力，3 格射程
  • Ghost Mode：5-10 秒後敵人可穿牆追蹤玩家

#### 3. 充氣攻擊系統（Pump）

  • 空白鍵發射充氣管，最大射程 4 格
  • 敵人充氣等級 0-4，達 4 級爆破消滅
  • 自動洩氣計時器（800ms 降一級）
  • 深度加分：位置越深得分越高

#### 4. 岩石物理

  • 2-3 顆岩石隨機分佈
  • 下方隧道挖空後 500ms 延遲開始掉落
  • 加速度物理，碰到泥土停止並碎裂
  • 可壓死敵人（500 分）和玩家

創意方向

  • CRT 濾鏡：掃描線 + 暈影 + 色差
  • 8-bit 音效：Web Audio API 方波/三角波/鋸齒波合成
  • 走路 BGM：8 音符循環旋律
  • 螢幕震動：岩石掉落和怪物爆破時觸發
  • 粒子系統：挖掘、爆破、岩石碎裂三種
