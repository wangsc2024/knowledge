研發筆記：Vite 6 重點（Environment API 與相容性）

AI技術Vite前端建置研發筆記

來源

  • Vite 6.0 release blog
  • Vite Features

要點（2024-11-26 穩定版）

  • Environment API：抽象 client/server 執行環境，可為不同環境設定獨立解析、轉譯與目標，利於 edge／自訂 runtime。
  • Node.js：Vite 6 需 Node 20+（放棄已 EOL 的 Node 18）。
  • CSS：內建更符合現行標準的巢狀等能力（減少額外 PostCSS 插件依賴）。
  • 方向：生態系持續往 Rolldown / Oxc 等 Rust 工具鏈整合以換取建置效能（長期由 VoidZero 等推進）。

與本專案

  • knowledge 與 game_web 皆採 Vite 靜態站／建置；升級大版前應先確認 Node 版本與 plugin 相容矩陣。
