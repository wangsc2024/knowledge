專案概述

禪境遊廊（Zen Game Portal）是一個包含五款禪意主題 HTML5 遊戲的合集網站，部署於 Cloudflare Pages（https://zen-game-portal.pages.dev/）。

五款遊戲

  1. 六根淨化（Six Roots Zen）：佛學打字修煉遊戲，六根階段制
  2. 念珠冥想（Mala Meditation）：108 顆念珠規律點擊冥想
  3. 正念記憶（Mindful Memory）：八正道符號配對記憶遊戲
  4. 貪吃蛇（Snake）：經典街機遊戲，禪意風格
  5. 禪意水墨（Zen Ink Flow）：全新創作 — 水墨畫風格的墨滴引導收集遊戲

品質優化重點

1. Portal 入口頁完整化

  • 原本入口頁只展示 2 款遊戲，修正為完整 5 款
  • 每款遊戲卡片包含圖標、標題、英文副標、描述、標籤、遊玩按鈕

2. 導航一致性

  • 為 six-roots-zen 補上「返回遊廊」back-link
  • 統一所有遊戲的返回導航（fixed position, z-index 300）

3. 暫停功能補全

  • 為 mindful-memory 新增暫停功能
  • Esc 鍵觸發暫停/恢復，暫停時停止計時器
  • 暫停覆蓋層含「繼續」與「重新開始」按鈕

4. 新遊戲「禪意水墨」設計要點

核心機制：引導墨滴穿越水墨禪境，收集 20 個禪字（禪/空/靜/悟/定/慧等），避開枯枝障礙

技術實現：

  • IIFE + 'use strict' 作用域隔離
  • requestAnimationFrame 遊戲迴圈 + deltaTime 上限
  • 完整狀態機：MENU → PLAYING → PAUSED → GAMEOVER
  • 三重輸入支援：鍵盤（WASD/方向鍵）、滑鼠追蹤、觸控
  • 圓形碰撞偵測
  • 粒子系統（MAX_PARTICLES=60 上限，物件池概念）
  • 程序化背景：遠山雲霧視差捲動
  • 墨跡拖尾（ink trail）漸層淡出
  • Web Audio API 程序化音效（收集/碰撞/結束）

讓人一玩再玩的設計（原子習慣四法則）：

  • 讓它顯而易見：HUD 即時顯示分數/禪字/連收，進度一目了然
  • 讓它有吸引力：水墨美學 + 禪字含義 + 連收倍率系統
  • 讓它簡單容易：直覺操控，漸進式難度
  • 讓它令人滿足：成就系統（10 個）、最高分追蹤、連續登入獎勵、遊戲結束禪語

LocalStorage 持久化：高分、最佳連收、成就、連續天數、總遊戲次數

部署流程

  1. git init + git commit → GitHub repo（wangsc2024/zen-game-portal）
  2. wrangler pages project create zen-game-portal
  3. wrangler pages deploy . --commit-message "..." → Cloudflare Pages
  4. 注意：wrangler commit message 必須為 ASCII（中文會觸發 UTF-8 驗證錯誤）

經驗教訓

  • 每款遊戲都應有統一的返回導航和暫停功能（SKILL.md 強制要求）
  • 粒子系統必須設上限（MAX_PARTICLES），否則低端裝置卡頓
  • Canvas 整數座標（| 0 位元運算取整）可減少子像素渲染
  • 批次繪製同類物件減少 context 切換
  • 程序化生成背景比靜態圖片更靈活且無載入延遲
