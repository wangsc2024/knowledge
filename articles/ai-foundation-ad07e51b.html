核心內容摘錄：

  • Cursor CLI 讓開發者在終端機直接驅動與編輯器同源能力的 AI agent，支援互動式 agent 與非互動 agent -p（--print）兩類場景；後者典型用途是 PowerShell 排程、run-todoist-agent-team.ps1 Phase 2、或 CI 內嵌的自動化任務檔 [來源: Cursor CLI overview, 2026]。與「純聊天 API」不同，CLI agent 會實際呼叫讀寫檔案、Shell、以及（若已設定）MCP 工具；因此產品方以沙箱界定「技術上允許的副程序邊界」，並以互動核准或旗標（如 --force、--approve-mcps）處理政策上是否自動放行 [來源: Implementing a secure sandbox for local agents · Cursor, 2026]。相較之下，OpenAI Codex CLI 以另一套命名（workspace-write、danger-full-access）與 approval_policy（never／on-request 等）描述類似雙層控制，且官方文件明確區分「沙箱邊界」與「核准策略」不得混為一談 [來源: OpenAI Codex — Sandbox concepts, 2026]。本專案同時面對 Windows 本機排程、多後端路由與 AGENTS.md 對 CLI／MCP 的邊界說明，理解 Cursor 這條鏈路的預設與可調參數，是降低 Phase 2 假失敗與過度放寬權限的関鍵。
  • 1. 非互動 -p 並不等於「自動寫入」 官方中文〈參數〉說明：--print「可存取所有工具，包括 write 和 shell」，但無頭文件另段指出：在腳本中要實際改檔需搭配 --force（--yolo 同義），否則「僅提議變更」[來源: 參數｜Cursor Docs（zh-Hant）, 2026；使用 Headless CLI（cn）, 2026]。這解釋了常見誤解：以為 -p 在 CI 內必然落盤，結果只得到文字建議而無檔案 diff。原理上，無頭模式缺少互動式「套用變更」確認，故產品以 --force 顯式承擔風險，與互動模式的人類按鈕對齊。
  • 2. sandbox.json：路徑、網路與合併優先序 sandbox.json 可置於 ~/.cursor/sandbox.json（使用者全域）與 repo 根目錄 /.cursor/sandbox.json（工作區）；兩者並存時工作區優先合併。欄位層級上，type 預設 workspace_readwrite，可降到 workspace_readonly 或 insecure_none（完全関閉沙箱）；networkPolicy 以 default、allow[]、deny[] 控制出站，且 deny 永遠贏過 allow；私有網段與 metadata IP 預設封鎖以降低 SSRF 面 [來源: sandbox.json Reference, 2026]。此外，.cursor/*/.json、.git/config、.cursorignore 等路徑恒為寫入保護；.cursor/rules/、skills/ 等子目錄可寫，直接對應本 repo 的 Skill／規則治理目錄結構 [來源: sandbox.json Reference, 2026]。
  • 3. 跨平台實作與 Windows 路徑 工程部落格說明：macOS 以動態生成的 Seatbelt profile 限制整棵子程序樹；Linux 以 Landlock + seccomp，並對 .cursorignore 做「不可讀寫的映射」；Windows 則在 WSL2 內執行 Linux 沙箱，原因是通用開發工具鏈的可支援原生 Windows 沙箱仍不足，官方與 Microsoft 合作推進中 [來源: Implementing a secure sandbox for local agents · Cursor, 2026]。Microsoft Learn（zh-tw）定義 WSL2 為輕量 VM 內的完整 Linux 核心，與 Windows 共用部分資源但隔離 PID／掛載等命名空間 [來源: 什麼是 WSL／WSL2, Microsoft Learn, 2025]。因此在本專案「純 pwsh + 本機 repo」場景，若 Cursor 終端沙箱依賴 WSL2，未安裝或未啟用 WSL2 可能導致與文件敘述不一致的邊界行為，需以實機 agent about／實際寫檔測試佐證而非臆測。

證據 6：大智度論深度研究 — 歷史背景、核心哲學、漢傳佛教影響與學術爭議（2026）

標籤：佛學、大智度論、龍樹、中觀、般若波羅蜜多、空性 ｜ KB id：bc256b69-cdf ｜ 原文 5927 字

原文章節導覽：

  • 執行摘要
  • 一、歷史背景與文本形成
  • 1.1 成書脈絡
  • 1.2 作者歸屬爭議
  • 二、篇幅結構與體例特色
  • 三、核心哲學體系
  • 3.1 空性（Śūnyatā）與緣起性空
  • 3.2 二諦圓融

核心內容摘錄：

  • 《大智度論》（梵：Mahāprajñāpāramitāśāstra，大正藏 T25n1509）是大乘佛教最龐大的論疏著作之一，共 100 卷、逾 100 萬字，傳為中觀學派開祖龍樹菩薩（Nāgārjuna，約西元 150-250 年）所造，由後秦譯師鳩摩羅什（Kumārajīva，344-413）於西元 402-405 年在長安逍遙園譯成漢文 [維基百科中文，2026；CBETA T25n1509；星雲大師全集，p082]。本論以《摩訶般若波羅蜜多經》（二萬五千頌本）為依據，逐品疏解，同時廣引印度各部派文獻，形成宏觀的佛教百科全書式格局。在漢傳佛教史上，本論直接催生三論宗、啟迪天台宗一心三智體系，並為華嚴宗所援引，是各宗派共同繼承的思想根源。現代學術界則在作者歸屬問題上存在持續爭論，核心焦點是本論究竟源自龍樹本人，或為鳩摩羅什譯場的集體編輯產物。
  • 《大智度論》的漢譯是後秦長安譯場最重要的成就之一。鳩摩羅什奉苻秦（實為後秦姚興）之邀，於長安逍遙園西明寺主持翻譯，弘始四年（402 CE）開譯，弘始七年（405 CE）譯竣，耗時約三年 [維基百科中文，2026；星雲大師全集，p082]。弟子僧叡等人參與筆受，精通梵漢兩文的鳩摩羅什更在翻譯過程中進行剪裁：前 34 卷完整翻譯底本《大品般若經》第一品〈序品〉（放光品），後 66 卷則以節略方式疏解其餘各品，形成「前詳後略」的特殊體例 [星雲大師全集，p082；WisdomLib Maha Prajnaparamita Sastra]。原梵文本據傳規模達 10 萬頌，然今已全佚，鳩摩羅什漢譯版是現存最完整的版本 [Internet Archive，MahaPrajnaparamitaSastra；WisdomLib，2026]。
  • 傳統佛教文獻一致將本論歸於龍樹菩薩，藏傳、漢傳與南傳均無異議。然而，20 世紀以降的現代佛學研究對此提出系統性質疑：
  • - Étienne Lamotte 質疑論（1944）：比利時學者 Lamotte 主持法文節譯（Le Traité de la Grande Vertu de Sagesse），認為本論文本存在「核心層」與「增補層」，古老核心可能出自龍樹，但大量後代增補形成複合文本，難以歸於單一作者 [英文 Wikipedia，Da Zhidu Lun；PhilPapers，Ramanan]。
  • - Po-kan Chou 編輯論：近年學者 Po-kan Chou 更進一步主張，本論實為鳩摩羅什弟子僧叡（352?-436?）主導編纂的產物，而非印度原典的直接漢譯 [英文 Wikipedia，Da Zhidu Lun，2026]。
  • - Ramanan 辯護論：另一方面，K. Venkata Ramanan 在其專著 Nagarjuna's Philosophy as Presented in the Maha-Prajnaparamita-Sastra（1966）中，藉由比對本論哲學立場與龍樹《中論》（Mūlamadhyamakakārikā）及《廻諍論》（Vigrahavyāvartanī）的高度一致性，為傳統歸屬辯護 [PhilPapers Ramanan，1966；WisdomLib，2026]。
  • 《大智度論》共 100 卷，大正藏收錄於第 25 冊（T25n1509），約 100 萬字以上 [CBETA T25n1509；星雲大師全集，p082]。體例具有以下特點：
  • 1. 注疏與義理雙軌：每節先引《大品般若經》原文，再以「釋曰」引出龍樹（或譯場）的詳細疏論，疏論篇幅往往遠超原文。
  • 2. 百科全書性格：廣引說一切有部（Sarvāstivāda）阿毘達磨論典、本生故事（Jātaka）、譬喻（Avadāna）、早期佛教律典等，所引小乘經典逾百部，是研究古印度佛教全貌的一手資料 [英文 Wikipedia，Da Zhidu Lun；京都大學稀覯資料數位典藏]。
  • 3. 前詳後略體例：前 34 卷全譯第一品，後 66 卷節略，反映鳩摩羅什「取其精要、不尚繁冗」的譯經方針 [星雲大師全集，p082]。
  • 本論的哲學核心是緣起性空：一切法皆因緣和合而生，無獨立自存之自性，故「性空」。這不是「虛無主義」的斷滅空，而是肯定緣起的「妙有」基礎上說空 [日文 Wikipedia 大智度論；空（佛教）Wikipedia；北大葉少勇「萬法皆空」]。論中廣辨「十八空」，將空性分析為自性空、他性空、空空等多個層次，是系統化的中觀空性論述。
