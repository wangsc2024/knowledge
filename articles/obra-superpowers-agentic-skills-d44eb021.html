一、專案概述

superpowers（obra/superpowers）是由 Jesse Vincent（Prime Radiant 團隊）開發的開源 Agentic Skills Framework，定位為「An agentic skills framework & software development methodology that works」。截至 2026 年 3 月已累積 96,000+ Stars、7,600+ Forks，採用 MIT 授權，於 2026 年 1 月正式納入 Anthropic 官方插件市場。

二、核心哲學：四大原則

  1. 測試驅動開發（TDD）：測試必須先於實作撰寫，違反則自動刪除程式碼
  2. 系統化流程優先：結構化方法論取代隨意編碼
  3. 降低複雜度：簡潔為王，避免過度工程
  4. 證據導向驗證：每一步驟都有可驗證的產出物

這四大原則與本專案（daily-digest-prompt）的 CLAUDE.md 規格驅動 + TDD 規則高度一致。

三、七階段結構化開發流程

階段 1：蘇格拉底式頭腦風暴（Brainstorming）

  • Agent 不直接寫程式碼，而是透過提問引導使用者釐清需求
  • 探索技術替代方案，精煉設計規格
  • 產出設計文件，分段展示供使用者驗證
  • 設計意圖：「It steps back and asks you what you're really trying to do」

階段 2：Git Worktrees 隔離工作區

  • 設計核准後自動建立隔離分支
  • 執行專案設定與測試基線驗證
  • 保護主分支完整性

階段 3：微任務規劃（Micro-Task Planning）

  • 將工作分解為 2-5 分鐘 的最小可驗證單元
  • 每個任務含：精確檔案路徑、完整程式碼範例、驗證步驟
  • 明確標註任務間依賴關係

階段 4：Subagent 並行開發

  • 多個專業子 Agent 同時處理基礎設施、UI、邏輯、測試
  • 相較於串行方式，實現 3-4 倍加速
  • Agent 可自主運作數小時不偏離計畫

階段 5：TDD 強制執行

  • 嚴格執行 RED-GREEN-REFACTOR 循環
  • 若實作先於測試，自動刪除實作程式碼
  • 目標達到 85-95% 測試覆蓋率

階段 6：系統化程式碼審查

  • 雙階段審查：(1) 規格符合度 (2) 程式碼品質
  • 自動檢驗覆蓋率、安全性、規格一致性
  • 嚴重度分級的問題評估

階段 7：專業化完成

  • 文件自動生成
  • 工作區清理
  • 合併/PR 決策

七、對 Daily-Digest-Prompt 系統的借鑑

7.1 高度相似的設計理念

| 面向 | superpowers | daily-digest-prompt |

|------|-------------|--------------------|

| 核心驅動 | Skill-First | Skill-First（ADR-001）|

| Skill 結構 | SKILL.md per skill | SKILL.md per skill |

| TDD 強制 | RED-GREEN-REFACTOR | 全程 TDD（CLAUDE.md 規則）|

| 並行執行 | Subagent-driven | Agent Team 策略（ADR-004）|

| 計畫階段 | 2-5 分鐘微任務 | specs/ 目錄任務分解 |

7.2 可借鑑的差異化特性

  1. 蘇格拉底式頭腦風暴：superpowers 在編碼前主動以提問釐清需求，本系統可在 Todoist 研究任務前加入類似的「需求精煉」步驟
  2. Git Worktrees 隔離：superpowers 每個功能在隔離 worktree 中開發，降低主分支風險
  3. Skills 獨立 Repository 模式：將 Skills 從主專案分離，透過 Git 子模組管理，實現社區級可擴展
  4. 自動 Skill 觸發：Agent 根據上下文自動選擇相關 Skill，類似本系統的 routing.yaml 但更隱式
  5. Memory 整合路線圖：計劃讓 Agent 從歷史專案決策中學習，與本系統的 continuity/ 連續記憶相呼應

---

八、關鍵洞察

  1. 方法論 > 工具：superpowers 的成功證明，AI Agent 的價值不在於生成速度，而在於結構化流程的品質保證
  2. Skill 可組合性是關鍵：模組化 Skill 讓用戶可以按需組合，避免 monolithic prompt 的維護困難
  3. 跨平台 Agent 統一標準：支援 5+ 個 AI 平台，暗示未來 Agent Skills 將走向標準化（類似 MCP 對工具的標準化）
  4. 社區驅動的 Skills Marketplace：從個人工具走向平台生態，是 Agent Skills 的下一個進化方向
