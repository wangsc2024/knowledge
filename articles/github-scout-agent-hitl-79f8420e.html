GitHub Scout 分析報告：Agent 架構 HITL Patterns（2026-04-28）

專案觀測摘要

AgentScope（agentscope-ai/agentscope）⭐ 24.4k

  • runtime_model：事件驅動訊息 hub；ReActAgent 支援即時中斷與恢復；v1.1.0 加入 Distributed Interrupt Service（2026-02）
  • state_strategy：InMemoryMemory + 長期 DB 記憶壓縮；支援自訂持久化復原邏輯
  • human_review_model：interrupt-point 模式 — 執行中即時取消並保留 context，resume 無縫繼續

HiClaw（agentscope-ai/HiClaw）⭐ 4.3k

  • runtime_model：Manager-Workers via Matrix room；OpenClaw/QwenPaw/Hermes 三種 runtime 協作
  • state_strategy：Workers 無狀態 + MinIO 中央存儲；憑證集中在 Higress Gateway
  • human_review_model：visibility-first — 每個 room 預設包含人類；@agent 訊息直接中斷並重導向

Mastra（mastra-ai/mastra）⭐ 23.4k

  • runtime_model：TypeScript 圖式編排（.then/.branch/.parallel）；40+ provider 路由
  • state_strategy：Storage system 維護執行狀態；支援無限期 pause/resume
  • human_review_model：signal-based — workflow.suspend() + await external approval signal
