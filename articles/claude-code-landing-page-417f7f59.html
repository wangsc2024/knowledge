【Claude Code 設計 Skill 系統整理】

三大場景的 Skill 選擇策略：

🖥 Landing Page / 行銷頁

• Anthropic 前端設計 Skill：強制模型先「承諾設計方向」，避免 AI slop（千篇一律紫漸層）

• GSAP Skill：動畫效能正確路徑（用 transform 而非 position/size 觸發重排）；scroll reveal 只是最低階用法

• AI LABS 架構：Marketing UI（藝術感）與 Functional UI（ShadCN）雙軌分離

📊 Dashboard / 應用功能層

• Shad CN = 預建組件庫（規則 Skill + MCP 直連 registry）；啟動即專業級組件，免手刻

• Dashboard Skill：先處理「資訊如何排列在畫面上」再處理組件，解決 AI 核心弱點

• UIUX Pro Max：執行前先跑引擎，從 161 個行業分類資料庫選配色/字體/版型

📱 Mobile

• Material 3：Google 設計系統（一個主色自動生成完整主題，彩色粗圓強動效）

• Swift UI Skill：從 Xcode 文件餵規則，輸出真正 native iOS 感

• Expo Skill：跨平台 iOS+Android 統一開發

關鍵原則：不堆疊所有 Skill，依場景選對一個即可。
