楞嚴經研究：會通四科圓彰七大 — 卷三如來藏妙真如性的體性開示

楞嚴經佛學四科七大如來藏會通四科圓彰七大五陰六入十二處十八界七大妙真如性性相不二

會通四科圓彰七大 — 卷三如來藏妙真如性的體性開示

主題概述

《大佛頂首楞嚴經》卷三承接卷二「十番顯見」揭示見性後，佛陀進一步以「會通四科」（五陰、六入、十二處、十八界）與「圓彰七大」（地、水、火、風、空、見、識）兩大論證架構，系統開示一切法皆是「如來藏妙真如性」的深義。此段經文是楞嚴經由「破妄」轉入「顯真」的關鍵轉折，不僅將小乘所分析的蘊處界諸法收攝回如來藏，更將器世間的七大種性也一併會歸，完成「性相不二」的圓教宗旨。圓瑛大師稱此段為「全經樞紐」，宣化上人則以「會通四科、圓彰七大」作為第四冊講述的核心。

---

一、「四科」開示：蘊處界皆本如來藏

1.1 經文總綱

佛告阿難：「五陰六入，從十二處至十八界，因緣和合虛妄有生，因緣別離虛妄名滅。殊不能知生滅去來，本如來藏常住妙明，不動周圓妙真如性。」

白話翻譯：五陰、六入乃至十二處、十八界，都因因緣和合而虛妄地呈現出生相，因因緣別離而虛妄地呈現出滅相。眾生不能了知，這些生滅來去的現象，其本體就是如來藏——常住不變、妙明不昧、不動搖而周遍圓滿的妙真如性。

1.2 五陰本如來藏

佛陀依次開示色、受、想、行、識五陰（五蘊），皆非因緣生、非自然性，而是如來藏中的幻化作用：

| 陰 | 譬喻 | 義理 |

|------|------|------|

| 色陰 | 如清水中現氣泡 | 「性色真空，性空真色」——色的本性即是真空，真空的本性即是真色，清淨本然，周遍法界 |

| 受陰 | 如以手觸體，冷暖交感 | 離開所觸之境，受無自體，本非因緣非自然性 |

| 想陰 | 如口說酸梅而生津液 | 虛妄想像，無有實體，如來藏中「覺明真識」的幻用 |

| 行陰 | 如瀑布流注不斷 | 剎那遷流，無有暫住，然其本體即如來藏妙真如性 |

| 識陰 | 如器中的頑空 | 「湛然不動，名恆常者」的虛妄認取，真識與妄識的交互混融 |

核心公式：「性X真空，性空真X」（X = 色/聲/香/味等），此公式貫穿四科七大，是楞嚴經獨特的表述方式。

1.3 六入本如來藏

「復次阿難，云何六入，本如來藏妙真如性？」

佛陀逐一論證眼、耳、鼻、舌、身、意六根（六入），離開明暗、動靜、通塞、甜苦、離合、生滅等對境時，「畢竟無體」。

  • 眼入：離明離暗，見精了了，此見精元是如來藏妙真如性
  • 耳入：離動離靜，聞性不滅，非因緣非自然
  • 鼻入：離通離塞，嗅覺明了，本如來藏
  • 舌入：離甜離淡，嘗性不壞，本如來藏
  • 身入：離觸離離，覺性周遍，本如來藏
  • 意入：離生離滅，知覺了然，本如來藏

關鍵義理：六入的功能（見聞覺知）並非依根而生、依塵而有，而是如來藏中本具的妙明作用。根塵和合只是「虛妄建立」的名相。

1.4 十二處本如來藏

「云何十二處，本如來藏妙真如性？」

將六根與六塵配對（眼與色、耳與聲等），論證每一對「根」與「塵」的兩處皆是虛妄：

  • 色處與眼見「俱無處所」
  • 聲與聽聞「俱無處所」
  • 香與嗅覺、味與舐嘗、觸與身覺、法塵與意念，皆此結構反覆

佛示「二處虛妄，本非因緣，非自然性」——根塵二處看似有所在、有生滅，實則無有定處，虛妄建立。

1.5 十八界本如來藏

「云何十八界，本如來藏妙真如性？」

在十二處基礎上加入六識，論證根、塵、識三者皆「三處都無」：

  • 眼根、色塵、眼識三界各自無有體性
  • 耳、鼻、舌、身、意識界亦復如是
  • 「既不是因緣和合而生，也不是自在自為」

義理深意：小乘析法將世間歸結為十八界，認為可以在蘊處界中尋得實法。楞嚴經則「會通」小乘所分析之法，指出其當體即是如來藏——不必離相覓性，也不必滅相證真。

二、「七大」開示：器世間亦本如來藏

2.1 為何需要「七大」

四科（五陰、六入、十二處、十八界）皆屬「有情法」——依止於有情眾生的身心現象。但器世間（山河大地、四大種性）是否也是如來藏？佛陀以「七大」涵攝情器兩界：

  • 地大（堅性）、水大（濕性）、火大（暖性）、風大（動性）：屬物質界四大種性
  • 空大（無礙性）：屬虛空
  • 見大（照性，或稱「覺大」）：屬根性、知覺
  • 識大（了別性）：屬心識

前四大加空大涵攝器世間，後二大（見大、識大）涵攝有情界的根與識，合為七大，遍攝一切法。

2.2 七大各論

| 大 | 經文要旨 | 核心義理 |

|------|---------|----------|

| 地大 | 「性地真空，性空真地」 | 地之堅性非內非外，遍一切處而無定所，本如來藏 |

| 水大 | 「性水真空，性空真水」 | 月光能引潮，示水性不定，隨緣而顯，本如來藏 |

| 火大 | 「性火真空，性空真火」 | 陽燧取火、林木生煙，火非外來亦非內出，本如來藏 |

| 風大 | 「性風真空，性空真風」 | 阿難整衣，風拂人面，風從何來到何處去？本如來藏 |

| 空大 | 「性覺真空，性空真覺」 | 虛空無形，因色而顯，空色不二，本如來藏 |

| 見大 | 「性見覺明，覺精明見」 | 見性周遍，不因根生不因塵滅，本如來藏 |

| 識大 | 「性識明知，覺明真識」 | 了別之性本非因緣和合，如來藏中本具湛明識性 |

2.3 七大總結

佛陀總結：「地水火風空見識，性真圓融，皆如來藏，本無生滅。」

又云：「世間無知，惑為因緣及自然性。皆是識心分別計度，但有言說，都無實義。」

此段破除兩種偏見：

  1. 因緣論：認為諸法由因緣和合而生（小乘、唯識宗立場）
  2. 自然論：認為諸法自然而有（外道立場）

楞嚴經立如來藏「隨緣不變，不變隨緣」之中道正義——諸法非因緣非自然，乃是如來藏不思議熏變之用。

三、義理深析：四科七大的思想意涵

3.1 性相不二

四科七大的核心論證是「即性即相」（或「性相不二」）：

  • 性：如來藏妙真如性（不變本體）
  • 相：五陰六入十二處十八界七大（隨緣顯現）

性不礙相，相不礙性；離相無性，離性無相。這正是華嚴「理事無礙」的精神，也是天台「即空即假即中」的圓教義理。

3.2 非因緣非自然

「本非因緣，非自然性」這一判語貫穿四科七大的每一段論證。這打破了三種知見：

  1. 打破小乘的因緣觀：小乘認為諸法由因緣和合，可以分析為蘊處界實法。楞嚴經指出因緣和合只是「虛妄有生」
  2. 打破外道的自然觀：外道認為諸法自然而有、無因而生。楞嚴經指出自然亦是「識心計度」
  3. 顯示如來藏的離言中道：如來藏超越因緣與自然的二元對立，是「言語道斷，心行處滅」的不思議境界

3.3 與全經的結構關聯

| 經文位置 | 內容 | 功能 |

|----------|------|------|

| 卷一 | 七處徵心 | 破妄心——心不在身內外等七處 |

| 卷二 | 十番顯見 | 顯真心——見性有十種殊勝特質 |

| 卷三 | 四科七大 | 會通萬法——一切法皆如來藏 |

| 卷四 | 二決定義 | 確立修因——以不生滅心為本修因 |

| 卷五-六 | 二十五圓通 | 示修行門——以耳根圓通為最勝 |

| 卷七 | 楞嚴咒 | 護持正修——以咒力加持修行 |

| 卷八 | 五十五位真菩提路 | 明修行階位——從乾慧地到等覺 |

| 卷九-十 | 五十陰魔 | 辨別邪正——修行中的魔境辨識 |

卷三的「會通四科圓彰七大」在全經中起承上啟下的關鍵作用：承接前面的「破妄顯真」，為後面的「修行法門」奠定理論基礎。

四、歷代祖師的精要詮釋

4.1 圓瑛大師

圓瑛大師畢生鑽研楞嚴經，著有《大佛頂首楞嚴經講義》二十四卷，稱此段為全經的「法理樞紐」。他指出：四科是「即相即性」——就著幻相當體便是真性；七大是「即性即相」——就著真性隨緣便是幻相。兩者合觀，「心色一如」的圓融義理得以完整展現。

4.2 宣化上人

宣化上人在第四冊淺釋中特以「會通四科」「圓彰七大」命名，強調：「會通」是將小乘所分的蘊處界法「會」歸一如來藏、「通」達無礙；「圓彰」是圓滿地彰顯七大種性皆是如來藏的圓融妙用。上人常以「月映千江」為喻——千江之月皆是一月，七大萬法皆是一如來藏。

4.3 交光真鑑

明代交光大師《楞嚴經正脈疏》以「捨識用根」為宗旨，認為四科七大之開示正是要眾生認識到：識心分別是虛妄的，根性（見聞覺知之性）才是真如之用。六入之「見大」特別被交光大師提出為修行入門的關鍵。

五、修行應用

5.1 日常覺照

四科七大的教義並非純理論，而是可在日常生活中實踐的：

  1. 觀色即空：見到眼前的一切色相（桌椅、山水、人物），提醒自己「性色真空」——所見之色的本性是空，但空中具足萬法
  2. 觀空即色：面對虛空或寂靜時，覺知「性空真色」——真空不是斷滅的空，而是含攝一切妙有的空
  3. 觀受不受：感受苦樂冷暖時，回觀受的本體——離開所觸對境，受並無自體，這就是如來藏的幻用
  4. 觀風大：感受微風拂面時，風從何處來？到何處去？既無來處也無去處，正是如來藏「不動周圓」的體現

5.2 打破二元對立

  • 不再執著「內心」與「外境」的對立——四科已示心境不二
  • 不再執著「有情」與「無情」的對立——七大已示情器不二
  • 不再執著「因緣」與「自然」的對立——皆是識心計度

5.3 建立正見

修行者當以「如來藏妙真如性」作為正見的基礎：

  • 不落於斷滅空（否認萬法的存在）
  • 不落於常見（執取萬法為實有）
  • 安住於「性真圓融」的中道見地

七、重要佛學術語

| 術語 | 解釋 |

|------|------|

| 如來藏 | 一切眾生本具的覺性，如來所藏之性德，不增不減、不垢不淨 |

| 妙真如性 | 真如的妙用——非靜止不動的空，而是「常住妙明、不動周圓」的活潑覺性 |

| 四科 | 小乘分析萬法的四種分類：五陰（蘊）、六入（根）、十二處（根+塵）、十八界（根+塵+識） |

| 七大 | 構成情器世間的七種基本元素：地水火風空見識，楞嚴經獨有的分類 |

| 會通 | 將分門別類的分析法「會」歸一處，「通」達無礙 |

| 圓彰 | 圓滿地彰顯、完整地開示 |

| 性色真空 | 色的本性是真空——不是斷滅空，是離言說相的第一義空 |

| 性空真色 | 真空的本性即是妙色——空中具足一切妙有 |

| 非因緣非自然 | 如來藏超越因緣論與自然論的二元對立，離言中道 |
