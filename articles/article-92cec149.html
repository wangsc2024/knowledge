淨土宗「十念法」與「計數念佛」研究：歷代祖師念佛方法的系統整理與實修指要

淨土宗「十念法」與「計數念佛」研究：歷代祖師念佛方法的系統整理與實修指要

主題概述

念佛是淨土宗最核心的修行方法，而「如何念佛才能攝心」是歷代淨土行者面臨的首要課題。從善導大師的「上盡一形，下至十念」，到慈雲遵式的「晨朝十念法」，再到印光大師的「十念記數法」，歷代祖師針對不同根器眾生，發展出一套完整的念佛攝心體系。本研究系統整理四種念佛（持名、觀像、觀想、實相）與歷代十念法的演變脈絡，並聚焦於最具實修價值的「計數念佛」方法，為現代淨土行者提供具體可行的修持指南。

---

一、四種念佛法門：從事修到理觀

淨土宗傳統將念佛修行分為四種層次，由淺入深：

1. 持名念佛（最普遍）

定義：繫心一佛，專心稱念「南無阿彌陀佛」名號，淨念相續。

經典依據：《佛說阿彌陀經》「執持名號，若一日……若七日，一心不亂」。

特點：三根普被、利鈍全收，是淨土宗最主要的修行方式。善導大師判此為「正定之業」——阿彌陀佛第十八願所選定的往生行業。印光大師稱：「持名一法，最為末法適機。」

2. 觀像念佛

定義：端坐諦觀佛像相好，令其歷歷分明。

方法：面對阿彌陀佛像，凝神專注，觀其三十二相、八十種好，直至閉眼亦能清晰呈現。

局限：離開佛像便難以維持，須在特定環境中修持。

3. 觀想念佛

定義：以觀像為基礎，閉目憶念觀想阿彌陀佛及極樂世界莊嚴。

經典依據：《觀無量壽佛經》十六觀，從日想觀到上品上生觀，層層深入。

善導大師的判釋：善導大師指出，凡夫眾生業障深重，所觀之境精微，能觀之心粗糙——「心識掉舉，精神散亂，觀想難以成就」。因此釋迦世尊大悲特開稱名一途，「正由稱名易故，相續即生」。

4. 實相念佛（最深）

定義：觀自身及一切萬法的真實自性，體悟「心、佛、眾生三無差別」。

特點：又稱「法身觀佛」，是佛法最深層次的修行——念自性彌陀、生唯心淨土。天台宗稱此為「約理觀佛」。

蕅益大師的圓融：《阿彌陀經要解》指出持名念佛即含攝實相——「一聲阿彌陀佛，即釋迦本師於五濁惡世所得之阿耨多羅三藐三菩提法。今以此果覺全體授與濁惡眾生。」因此持名念佛，全事即理，不可思議。

二、善導大師：「上盡一形，下至十念」的本願念佛

2.1 核心教義

善導大師（613-681）是中國淨土宗的實際創宗者，其念佛思想以阿彌陀佛第十八願為核心：

「彌陀世尊，本發深重誓願，以光明名號，攝化十方，但使信心求念；上盡一形，下至十聲、一聲等，以佛願力，易得往生。」

「上盡一形」——盡此一生持續念佛，從皈依發願之日起直至命終。

「下至十念」——即使只能念十聲乃至一聲佛號（如臨終者），依佛願力亦得往生。

2.2 「十即十生，百即百生」

善導大師在《往生禮讚》中開示：

「若能如上念念相續，畢命為期者，十即十生，百即百生。」

其原因有四：

  1. 無外雜緣得正念故——心中只有佛號，排除一切雜念。
  2. 與佛本願得相應故——稱名正合第十八願「乃至十念」之意。
  3. 不違教故——順從釋尊在《阿彌陀經》中「執持名號」的教導。
  4. 隨順佛語故——隨順阿彌陀佛「欲生我國」的召喚。

2.3 為何以稱名為正定業

善導大師解釋為何不教觀想而教稱名：

「眾生障重，境細心粗，識颺神飛，觀難成就。是以大聖悲憐，直勸專稱名字，正由稱名易故，相續即生。」

這段話道出了稱名念佛的殊勝：觀想需要高度定力，凡夫難以企及；而稱名人人能行，只要相續不斷，依佛願力即得往生。善導大師另引《文殊般若經》：「獨處空閑，捨諸亂意，繫心一佛，不觀相貌，專稱名字。」

三、慈雲遵式：晨朝十念法（十口氣念佛）

3.1 方法來源

宋代慈雲遵式禪師（964-1032）依據第十八願「乃至十念」，創立了「晨朝十念法」，專為事務繁忙、無暇長時修持的在家居士設計。

3.2 具體方法

每日清晨，面向西方（或佛像前）：

  1. 合掌恭敬，深吸一口氣至腹部
  2. 盡此一口氣稱念「南無阿彌陀佛」，能念多少聲就念多少聲
  3. 氣盡為一念
  4. 如是連續十口氣（十念）
  5. 念畢迴向發願往生

3.3 特點與限制

優點：

  • 用時極短（約 2-3 分鐘），適合忙碌居士
  • 因須盡氣力念佛，自然全神貫注，攝心效果極佳
  • 每日不間斷，積功累德

限制：

  • 只適合晨朝使用（最多早晚各一次）
  • 若二十、三十口氣連續，則傷氣成病
  • 不適合全日修持

四、印光大師：十念記數法（最切實際的攝心法）

4.1 創立背景

印光大師（1862-1940）觀察末法眾生根器鈍劣，念佛時妄念紛飛，雖知「都攝六根，淨念相繼」的道理，卻無法做到。因此在慈雲十念法的基礎上，另創「十念記數法」——一種可以從朝至暮、全天通用的攝心念佛法。

4.2 具體方法（三重功夫）

印光大師教導攝心念佛有三重功夫，層層深入：

第一重：至誠懇切

念佛時心存至誠恭敬，如面對阿彌陀佛，字字句句發自內心。

第二重：攝耳諦聽

無論出聲念或默念，都要用耳朵諦聽自己的佛號——佛號從心起、從口出、從耳入。

第三重：十念記數（終極攝心法）

若前兩重仍伏不住妄念，則加上十念記數——

「所謂十念記數者，當念佛時，從一句至十句，須念得分明，仍須記得分明。至十句已，又須從一句至十句念，不可二十三十。隨念隨記，不可掐珠，唯憑心記。」

4.3 三種分段法（因應不同根器）

| 方式 | 分段 | 適用者 |

|------|------|--------|

| 直記法 | 1→2→3→...→10 | 心念較細、能直接記數者 |

| 二段法 | (1-5) + (6-10) | 十句直記稍難者 |

| 三段法（三三四） | (1-3) + (4-6) + (7-10) | 初學者或妄念多者 |

4.4 核心要訣：「三個清楚」

  1. 念得清楚：佛號從心中生起，字字分明（心清楚）
  2. 聽得清楚：耳朵聽自己念的佛號，每個字都聽到（耳清楚）
  3. 記得清楚：心中記數，第幾句了然於心（意清楚）

三重功夫合於一聲佛號——心、口、耳、意全部繫在佛號上，妄念自然無處安身。

4.5 注意事項

  • 不可掐念珠計數：十念記數是「心記」，若掐珠則心力分散，反失攝心之效
  • 不可二十三十：只記一至十，到十即歸一，如此循環
  • 快慢隨意：念快亦記、念慢亦記，了無滯礙
  • 隨時可用：從朝至暮皆宜，不像慈雲十念法只限晨朝
  • 養神不傷氣：不需深呼吸，平常念佛即可

4.6 印光大師原文（節選）

「此攝心念佛之究竟妙法，在昔宏淨土者，尚未談及，以人根尚利，不須如此。今人根鈍，捨此十念記數之法，欲都攝六根，淨念相繼，大難大難。」

五、兩種十念法的比較

| 比較項目 | 慈雲遵式（晨朝十念法） | 印光大師（十念記數法） |

|----------|----------------------|----------------------|

| 「十念」的定義 | 十口氣為十念 | 十句佛號為十念 |

| 每念的佛號數 | 不定（盡一口氣能念多少） | 固定一句為一念 |

| 適用時段 | 僅限晨朝（早晚各一次） | 從朝至暮，全天通用 |

| 每次用時 | 2-3 分鐘 | 無限制，可持續整日 |

| 攝心原理 | 盡氣力念佛，身心並用 | 心記數目，意念專注 |

| 身體影響 | 多則傷氣 | 養神不傷氣 |

| 適用根器 | 事務繁忙的在家居士 | 一切根器，尤利鈍根 |

| 共同點 | 皆為攝妄歸真的念佛法 | 同上 |

印光大師明確指出：「晨朝十念，盡一口氣為一念，不論佛數多少……此法只可晨朝十念則可，若二十三十則傷氣成病。此十念記數，則念一句佛心知一句，念十句佛心知十句……不但去妄，最能養神，隨快隨慢，了無滯礙，從朝至暮，無不相宜。」

六、其他念佛計數與攝心方法

6.1 念珠計數法

以一百零八顆念珠計數，每念一聲撥一珠。適合散步經行或定課計數。印光大師認為，念珠計數適合定課（如日念萬聲），但若要攝心，仍以十念記數法為勝。

6.2 追頂念佛

一句接一句、綿密不斷地念佛，「阿彌陀佛阿彌陀佛阿彌陀佛……」中間不留間隙。此法攝心力強，但初學者容易疲勞，宜在精進佛七中使用。

6.3 禮拜念佛

每拜一拜，念一聲佛號。身口意三業同時運作——身禮拜、口稱名、意緣佛。此法身心並修，消業障效果極佳，但對體力要求較高。

6.4 經行念佛

邊走邊念佛號，步伐與佛號配合。一步一句或兩步一句，如善導大師「般舟念佛」。適合久坐後活動身體，或作為日常散步修行。

七、與已有研究的關聯

7.1 與善導大師五正行的關係

十念法是「稱名正行」（正定之業）的具體攝心方法。善導大師判五正行，以稱名為正定業，其餘讀誦、觀察、禮拜、讚歎供養為助業。十念法正是在稱名正行的框架下，解決「如何攝心稱名」的實修問題。

7.2 與大勢至念佛圓通章的關聯

印光大師的十念記數法，其終極目標是達到大勢至菩薩教授的「都攝六根，淨念相繼」。十念記數是「都攝六根」的前方便——先用計數之法令心歸一，久而久之自然達到「不假方便，自得心開」的境界。

7.3 與臨終正念的關係

善導大師的「下至十念」，直接關聯臨終助念。《觀經》下品下生者，臨終十念即得往生——這正是十念法的終極意義：平時用十念記數攝心念佛，是為臨終時能「心不顛倒」做準備。

7.4 與天台教觀的會通

蕅益大師從天台圓教的角度指出，持名念佛「全事即理」——事持名號即是觀照實相，不必另求實相念佛。這為十念記數法賦予了理論高度：看似簡單的計數念佛，實含一切佛法之精髓。

八、現代行者的實修建議

8.1 日課安排（三階段）

| 階段 | 程度 | 日課建議 |

|------|------|----------|

| 初學 | 剛接觸念佛 | 晨朝十念法（慈雲遵式）+ 睡前十念 |

| 進階 | 已建立習慣 | 早晚功課各半小時（十念記數法） + 日間散念 |

| 精進 | 長期修持 | 日念萬聲以上（十念記數法全天用），定期參加佛七 |

8.2 十念記數法入門步驟

  1. 選擇安靜環境，端身正坐
  2. 先念三聲佛號調身心
  3. 開始計數：採用「三三四」分段法（1-3、4-6、7-10）
  4. 每十句一個循環，不斷反覆
  5. 有妄念打斷計數→不必煩惱，從一重新開始
  6. 日漸熟練後，可改為二段法或直記法
  7. 念畢迴向：「願以此功德，莊嚴佛淨土……」

8.3 常見問題處理

  • 妄念太多，記不住數：正常現象，用「三三四」分段法，一週內可改善
  • 念快好還是念慢好：印光大師說「隨快隨慢，了無滯礙」，但初學宜慢不宜快
  • 出聲念還是默念：在家出聲念（攝心力強），公共場所默念（不擾他人），都要攝耳諦聽
  • 是否可以掐念珠：定課計數可用念珠，但攝心用十念記數法更佳（心不分散）
