楞嚴經佛學狂性自歇歇即菩提

楞嚴經研究：「狂性自歇，歇即菩提」— 卷五歇狂歸元與實修下手處

一、主題概述

《大佛頂首楞嚴經》卷五在層層抉擇「真妄二本」與「知見」之後，以極簡句點出：狂性止息之處，當下即是菩提；並強調勝淨明心本來周徧法界、不從他得。本筆記聚焦此組經句，說明「狂」與「歇」的佛學意涵、與前後「知見立知」「湛入合湛」的銜接，以及日常禪修中如何不落壓念與放縱二邊。經文用字請以 CBETA 《大正藏》第19冊 No.945 所收本對校；以下引文依通行讀本，異文處於註中略點。

二、經文重點段落（附白話翻譯）

2.1 狂歇即覺、心周法界

原文（節录，卷五）：

狂性自歇，歇即菩提。勝淨明心，本周法界，不從人得。何藉劬勞，肯綮修證。

白話大意：

顛倒追逐的「狂心」若自然止息，這份止息本身並非外加一物，而是菩提的顯現；那清淨妙明之心，本來圓滿周徧於法界，不是從別人那里得來的。既然如此，又何須把修行弄成對枝節的無謂強求，好像菩提是「修證才從外鑿出」的東西？

註：部分刻本或作「狂性若歇」，「若／自」皆表狂心止息之義，讀者以所依版本為準。

2.2 與近鄰經句的呼應（知見／無見）

同卷著名的：

知見立知，即無明本；知見無見，斯即涅槃無上菩提。

白話串聯：「立知」是於覺體上再橫加一層能所與抓取，故成無明本；若知見到「無所可立、亦無能立之見」，則契涅槃／菩提。與「狂性自歇，歇即菩提」並觀：歇狂是止息橫生之「立知」與顛倒行；菩提非新得，是本具心體之顯發。

三、義理解析（術語簡釋）

| 術語 | 解釋 |

|------|------|

| 狂性 | 非僅情緒激動，而是迷真認妄、於無法中起法、於無所得中强生得失的顛倒心行；可涵蓋貪瞋癡慢疑與種種「認賊為子」。 |

| 歇 | 止息、休息；在禪觀中多指不隨妄轉、不續第二念、不於覺上再加造作，非壓抑不起念之「死歇」。 |

| 菩提 | 覺；此處偏就本具覺體言，非指一個可從外累積的果報名相。 |

| 勝淨明心 | 超越對待染淨執取之妙明；「勝」簡別凡情所計之淨相。 |

| 本周法界 | 體性無方所而無所不遍；非空間比喻可盡，但可用「無內無外」作入門理解。 |

| 不從人得 | 簡別依他情見、名言授受而「外鑿」佛性；師友仍屬增上緣，究竟覺體不由外賜。 |

| 肯綮修證 | 語出《莊子》庖丁解牛喻；經文借指在枝節上劬勞，而昧於本具，故點醒修者勿離本求末。 |

與天台／禪門常談之會通（扼要）：天台重「性具」，禪門重「直指」，楞嚴此段以狂歇顯性語言，與「即心即佛」「本自圓成」可互照明，仍須回到經教戒定慧次第與後文圓通、陰魔之護行，避免口頭圓滑。

四、與知識庫既有研究的關聯（承上啟下）

  • 承上：KB 已有〈「知見立知，即無明本」與「知見無見」〉，析卷五「立知」與涅槃真淨之轉折；本篇以「狂歇」句，收束為可操作的止息方向。
  • 啟下：KB 已有〈湛入合湛與結解邏輯〉、〈卷六二十五圓通與文殊揀選耳根〉；狂歇所顯之本淨，正是後文選根圓通、遠離陰魔的體性依據——無得為本，方便為道。

五、修行應用（日常生活與禪坐）

  1. 認取「狂相」：當下強烈要「證得什麼」「壓掉什麼」時，多屬「狂性」現行；先覺知而不隨，較急於定義成敗。
  2. 歇的實義：試著在一件事上完結後，留一隙不評判、不編故事，讓心回到平常；非逃避責任，而是不續第二把火。
  3. 避免兩邊：

- 壓念：把「歇」做成硬忍，易成枯定或受陰區宇（後文陰魔所誡）。

- 放縱：把「不從人得」誤解為不假持戒、禪定，則與經教相違。

  1. 與戒行並進：同經卷六、七四種清淨明誨、三種漸次，是「狂性」能歇的緣助；本筆記義理須放回全經護行脈絡。
