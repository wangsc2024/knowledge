Space Invaders v2.3 優化心得

概述

小蜜蜂 v2.3 新增兩大社交/玩法機制：連擊計分系統（Combo Chain）和 Top 10 本地排行榜。這兩個功能相輔相成——連擊系統讓高手能刷出更高分數，排行榜則提供長期追分動力。

---

一、連擊計分系統（Combo Chain）

設計理念

經典 Space Invaders 的計分是固定的（10/20/30 分），缺乏「技巧獎勵」。Combo Chain 為連續擊殺提供倍率加成，獎勵精準且快速的射擊。

核心機制

  • Combo 窗口：每次擊殺後有 2 秒倒數，在窗口內再次擊殺累積連擊
  • 倍率公式：multiplier = min(8, 1 + floor(comboCount / 3))

- 1-2 擊：1x（基礎）

- 3-5 擊：2x

- 6-8 擊：3x

- 9-11 擊：4x

- ...最高 8x

  • 倒數重置：2 秒內未擊殺 → 連擊中斷，播放提示音效
  • 被命中重置：玩家被擊中時也會重置連擊

視覺回饋

  1. HUD Combo Bar：左上角顯示 combo 狀態，含倒數計時條

- 顏色分級：白(1x) → 橙(2-3x) → 紅(4x+)

- 脈動動畫增加緊迫感

  1. 浮動分數文字：擊殺位置顯示 +30 (x3) 等文字，向上飄移後淡出

- 使用物件池（MAX_FLOAT_TEXTS=10）避免 GC 壓力

  1. 音效遞進：連擊 3/6/9 次時播放越來越高的提示音

實作要點

  • registerKill(basePoints, x, y) 統一擊殺入口，取代直接 score +=
  • 倍率僅影響擊殺得分，不影響道具或關卡獎勵
  • 浮動文字池預分配，避免每次 new 物件

---

二、Top 10 排行榜

設計理念

原本只有單一 highScore，無法記錄不同場次的成就。Top 10 排行榜提供：

  • 多次遊玩的歷史記錄
  • 名稱+分數+關卡+日期的完整記錄
  • 視覺化排名展示（金色前三名）

資料結構


// localStorage key: 'si_leaderboard'
[{ name: 'ACE', score: 12350, level: 5, date: '2026-03-02' }, ...]

流程

  1. Game Over → 檢查分數是否能進入 Top 10
  2. 能進榜 → 顯示名稱輸入介面（最長 8 字元）

- 鍵盤輸入 + 閃爍游標

- Enter 確認 → 寫入 localStorage → 顯示排行榜

- 輸入期間阻擋所有其他按鍵（防止意外重啟）

  1. 不能進榜 → 直接顯示排行榜
  2. Menu 畫面 → 底部顯示 Top 3 預覽

防呆設計

  • nameInputActive 狀態鎖：名稱輸入期間阻擋 touch/click/keyboard 的遊戲操作
  • 名稱長度限制 8 字元
  • 空名稱不允許確認
  • JSON parse 失敗時回傳空陣列（容錯）

---

三、品質基線（v2.3）

  • requestAnimationFrame 遊戲迴圈（非 setInterval）
  • AABB 碰撞偵測
  • 物件池：粒子(200) + 子彈(30) + 浮動文字(10)
  • Web Audio API 8-bit 音效合成
  • CRT 復古濾鏡（離屏預渲染）
  • 動態背景音樂
  • 道具系統（4 種道具）
  • localStorage 高分+CRT偏好+排行榜 持久化
  • 觸控+鍵盤+滑鼠 三種輸入支援
