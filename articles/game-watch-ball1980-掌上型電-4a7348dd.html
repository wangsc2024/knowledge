Game & Watch: Ball（1980）— 掌上型電子遊戲始祖的極簡設計哲學

研究日期：2026-02-23 分類：骨灰級掌上型電玩研究 設計者：橫井軍平（Gunpei Yokoi）

一、遊戲簡介與歷史

1.1 誕生背景

Game & Watch: Ball 是任天堂於 1980 年 4 月 28 日推出的第一款 Game & Watch 系列遊戲，也是史上第一款掌上型 LCD 電子遊戲。設計者為任天堂 R&D1 部門負責人橫井軍平。

靈感來源：橫井在搭乘新幹線時，注意到一位無聊的上班族反覆按壓計算機按鈕打發時間。這個觀察啟發了他：如果能把這種小巧的計算機改造成可以在通勤時「不引人注目地」遊玩的玩具，一定會有市場。

1.2 技術背景

1980 年代初期，Sharp 和 Casio 在數位計算機市場激烈競爭，導致 LCD 液晶螢幕和半導體晶片產能過剩、價格暴跌。橫井看到了機會：用這些便宜的「枯れた技術」（成熟技術）做出全新的遊戲體驗。

1.3 商業成就

  • 橫井原本的目標：賣出 10 萬台
  • 實際銷量：Game & Watch 系列全球共售出 4,340 萬台（日本 1,287 萬 + 海外 3,053 萬）
  • 系列共推出近 60 款遊戲
  • 奠定任天堂在掌上型遊戲市場的霸主地位

二、核心機制分析

2.1 玩法描述

玩家操控一個雜技師角色，透過左右兩個按鈕控制角色的雙手左右移動，接住從空中落下的球並將其拋向另一側，持續循環不讓任何球落地。

2.2 雙模式設計

模式球數每次接球得分難度
Game A2 顆1 分入門
Game B3 顆10 分進階

設計巧思：用「球的數量」和「分數倍率」同時控制難度與獎勵，讓進階模式有更高的風險回報比。

2.3 難度曲線

  • 初始：球速緩慢，玩家可輕鬆判斷落點
  • 漸進加速：隨分數累積，球速逐漸加快
  • 節奏重置：每達 100 分自動降速，給玩家喘息空間
  • 結束條件：任何一顆球落地即 Game Over

這種「漸進加速 + 週期性緩和」的難度設計極為先進，現代跑酷遊戲和節奏遊戲仍在使用同樣的模式。

2.4 操控約束

玩家按下左鍵或右鍵時，角色的雙手同時移動。這個看似簡單的約束是整個遊戲張力的核心來源——當兩側都有球落下時，玩家必須精確計算時機，在一側接球後立刻切換到另一側。

三、設計哲學：枯れた技術の水平思考

3.1 原文解析

「枯れた技術の水平思考」（Lateral Thinking with Withered Technology）是橫井軍平的核心設計哲學：

  • 枯れた技術（Withered/Weathered Technology）：不是「過時」而是「成熟」的技術——大量生產、充分理解、成本極低
  • 水平思考（Lateral Thinking）：跳脫技術的原有用途，找到創新的應用方式

3.2 在 Ball 中的體現

成熟技術原有用途創新應用
LCD 液晶螢幕計算機顯示數字顯示遊戲角色動畫
半導體晶片計算機運算遊戲邏輯處理
按鈕計算機數字輸入遊戲控制器
薄型機身口袋型計算機口袋型遊戲機

3.3 哲學影響力

此哲學深刻影響了任天堂後續產品線：

  • Game Boy（1989）：刻意不用彩色 LCD（當時的尖端技術），換來更長電池壽命，擊敗 Sega Game Gear 和 Atari Lynx
  • Wii（2006）：用成熟的加速度感測器和紅外線感應（而非昂貴的陀螺儀）實現體感操控
  • Nintendo Switch（2017）：用行動裝置級 Tegra 晶片而非高階桌面 GPU，實現掌機/家機混合形態

四、創意元素與亮點

4.1 LCD 預繪動畫技術

LCD 技術的核心限制是：螢幕上的每個角色姿勢都必須預先蝕刻在液晶面板上。不像現代點陣螢幕可以自由繪製任何圖形，LCD 只能「顯示」或「隱藏」預設的圖案。

這個限制迫使設計師將遊戲動畫分解為最少的關鍵幀，每個姿勢都必須精心設計，確保玩家能清楚辨識角色狀態。這與現代像素藝術的精簡美學不謀而合。

4.2 雙重功能設計

「Game & Watch」——遊戲機同時也是手錶。當不玩遊戲時，裝置會顯示時間，賦予產品日常實用價值。這是早期「always-on 裝置」的概念先驅。

4.3 工程美學

原版 Ball 螢幕兩側有兩道黑色條紋，實際上是用來遮蔽無法避免的線路走線。任天堂在 2011 年復刻版中刻意保留了這些條紋（儘管技術上已不需要），將工程瑕疵轉化為品牌識別元素。

4.4 Mr. Game & Watch 文化遺產

Ball 中的雜技師角色後來被命名為「Mr. Game & Watch」，成為任天堂的經典角色之一，出現在《任天堂明星大亂鬥》系列中。他的 2D 黑色剪影造型已成為遊戲設計史的視覺符號。

五、可借鑒的 HTML5 遊戲設計要素

5.1 極簡操控設計

借鑒：只用兩個按鈕（左/右）就創造出深度遊玩體驗。

HTML5 應用：

  • 手機觸控遊戲只需左右兩個區域即可操控
  • 減少 UI 元素，讓玩家專注在核心機制
  • 約束反而產生深度——雙手同時移動的限制創造了策略層
// 極簡操控範例
document.addEventListener('keydown', (e) => {
  if (e.key === 'ArrowLeft') juggler.moveLeft();
  if (e.key === 'ArrowRight') juggler.moveRight();
});
// 觸控：螢幕左半/右半
canvas.addEventListener('touchstart', (e) => {
  const x = e.touches[0].clientX;
  x < canvas.width / 2 ? juggler.moveLeft() : juggler.moveRight();
});

5.2 漸進式難度 + 週期性緩和

借鑒：每 100 分自動降速，給玩家心理上的「呼吸空間」。

HTML5 應用：

  • 實作 difficultyPulse 函數：難度呈正弦波而非線性上升
  • 每個「階段」開始時短暫降低難度，讓玩家感受進步
  • 這比單純的線性加速更能維持玩家留存
// 難度脈動：每 100 分重置基底速度，但上限逐漸提高
function getDifficulty(score) {
  const cycle = Math.floor(score / 100);
  const progress = (score % 100) / 100;
  const baseSpeed = 1 + cycle * 0.15; // 每個週期基底速度+15%
  return baseSpeed + progress * 0.5;  // 週期內漸進加速
}

5.3 「枯れた技術」在 Web 開發中的應用

借鑒：不追求最新技術，用成熟技術創造新體驗。

HTML5 應用：

  • Canvas 2D 是「枯れた技術」——所有瀏覽器都支援、效能穩定、文件豐富
  • 不必追求 WebGL/WebGPU，Canvas 2D 足以實現出色的 2D 遊戲
  • localStorage 是「枯れた技術」——簡單可靠，足以實現排行榜和存檔
  • CSS transform 是「枯れた技術」——用它實現響應式縮放比自己計算 viewport 更穩定

5.4 LCD 風格復古視覺

借鑒：LCD 預繪圖案的獨特美學——有限的動畫幀、半透明的「幽靈圖案」。

HTML5 應用：

  • 模擬 LCD 效果：預先繪製所有可能的角色姿勢，用 globalAlpha = 0.1 顯示「背景幽靈」
  • 這種視覺風格在復古遊戲愛好者中極受歡迎
  • 可用 CSS filter 疊加綠色色調模擬經典 LCD 螢幕
// LCD 幽靈效果
function drawLCDGhost(ctx, sprites) {
  ctx.globalAlpha = 0.08;
  sprites.forEach(sprite => ctx.drawImage(sprite.img, sprite.x, sprite.y));
  ctx.globalAlpha = 1.0;
}

function drawActiveSprite(ctx, activeSprite) {
  ctx.drawImage(activeSprite.img, activeSprite.x, activeSprite.y);
}

5.5 「一次失敗即結束」的張力設計

借鑒：沒有生命值、沒有容錯——一顆球落地就 Game Over。

HTML5 應用：

  • 高張力模式適合短時間遊戲（通勤、排隊時）
  • 搭配即時重啟（零延遲 restart），降低挫敗感
  • 排行榜系統讓「再試一次」的動機更強

5.6 雙重實用性

借鑒：Game & Watch = 遊戲 + 手錶，產品在非遊玩時仍有價值。

HTML5 應用：

  • 遊戲網頁可包含實用功能（計時器、番茄鐘、天氣小工具）
  • PWA 化後可作為 always-on 的桌面小工具
  • 閒置時顯示有用資訊，提升應用黏著度

六、D-pad 十字鍵的誕生（額外彩蛋）

雖然 Ball 使用的是兩個獨立按鈕，但 Game & Watch 系列後來在《大金剛》中遇到了需要四方向移動的挑戰。橫井為此發明了 D-pad（十字方向鍵）——這個扁平的四方向控制器設計後來成為所有遊戲手把的標準配備，並在 2008 年獲得美國電視藝術與科學學院的技術與工程艾美獎。
