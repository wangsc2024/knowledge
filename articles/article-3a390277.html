主題概述（200字）

《大佛頂首楞嚴經》卷五記載二十五位菩薩、聖者各述其證道法門，構成「二十五圓通」的修行體系。此二十五法門依「六塵、六根、六識、七大」分類，展現了佛法「歸元無二路，方便有多門」的圓融精神。文殊菩薩受佛敕命，從中揀選最契此土眾生根機者，最終選定觀世音菩薩耳根圓通為第一，並留下千古名句：「此方真教體，清淨在音聞。」本研究系統性比較二十五圓通的分類架構、各宗派選擇圓通的歷史實證，以及修行者如何依根性、環境、宗派傳承選擇適合的圓通法門，為當代修行者提供實用指引。

一、二十五圓通的完整架構與經文脈絡

1.1 經文背景：二十五聖各陳圓通

楞嚴會上，佛陀為阿難開示真心本性後，二十五位已證聖果的菩薩與阿羅漢，應佛之請，各自敘述其最初修行入道的法門。這二十五法門涵蓋了感官知覺、意識活動、宇宙元素等修行入手處，證明「法法平等，門門皆可入道」。

1.2 四大分類：六塵、六根、六識、七大

二十五圓通依佛教對世界認識的分類體系排列：

（一）六塵圓通（外境為本修因）

塵是外境，修行者以外在對境為觀照對象：

  1. 憍陳那等五比丘：觀「客塵」— 以客塵煩惱本空為入道方便，證悟無生法忍
  2. 優波尼沙陀：觀「色性」— 觀色如聚沫，色性本空，不落有無
  3. 香嚴童子：觀「香塵」— 觀香嚴童子時聞眾香，知香非木非空，悟香性圓通
  4. 藥王、藥上二法王子：觀「味塵」— 以味供養如來，了知味性非即非離，得明圓通
  5. 跋陀婆羅：觀「觸塵」— 於浴僧時悟水因緣，非水非身，觸性圓明
  6. 摩訶迦葉：觀「法塵」— 於糞掃衣觀法塵清淨，滅諸細惑，證大阿羅漢

（二）六根圓通（六根為本修因）

根是感知器官，修行者直接以六根性為修行入手處：

  1. 阿那律陀：「眼根圓通」— 失明後反觀見性，不因眼壞而滅，見性圓明不由於眼
  2. 周利槃陀伽：「鼻根圓通」— 觀息出入，住無所住，得大無礙
  3. 憍梵波提：「舌根圓通」— 觀味知性，非即身心，非離身心
  4. 畢陵伽婆蹉：「身根圓通」— 因觀身無覺無知，悟身心虛妄，獲無礙解脫
  5. 須菩提：「意根圓通」— 觀意如空，意根即空，得成無諍
  6. 觀世音菩薩：「耳根圓通」— 反聞聞自性，從聞思修入三摩地（此為文殊揀選第一）

（三）六識圓通（六識為本修因）

識是認知活動，修行者以識心轉變為入道方便：

  1. 舍利弗：「眼識圓通」— 見光明相，眼識明淨，見十方空
  2. 普賢菩薩：「耳識圓通」— 心聞洞十方，生於大因力（大乘心聞法界）
  3. 孫陀羅難陀：「鼻識圓通」— 觀息久住，心得無漏
  4. 富樓那彌多羅尼子：「舌識圓通」— 宣說苦空，銷我億劫顛倒想
  5. 優波離：「身識圓通」— 親承佛敕，執身分別身心通利，觀身心通達
  6. 目犍連：「意識圓通」— 於意識中旋心旋見，離諸妄想

（四）七大圓通（宇宙元素為本修因）

七大是構成宇宙萬法的七種元素性，修行者體認元素性即如來藏性：

  1. 持地菩薩：「地大圓通」— 平填道路，觀地性平等無差別
  2. 月光童子：「水大圓通」— 修習水觀，觀身中水性與香水海等無差別
  3. 烏芻瑟摩：「火大圓通」— 觀火光三昧，化多淫心成智慧火
  4. 琉璃光法王子：「風大圓通」— 觀風性無依，成無上覺道
  5. 虛空藏菩薩：「空大圓通」— 觀四大無依，妄想生滅，虛空無二
  6. 彌勒菩薩：「識大圓通」— 修習唯識觀，了達萬法唯識所現
  7. 大勢至菩薩：「根大圓通」— 以念佛心入無生忍，都攝六根淨念相繼（淨土宗核心法門）

1.3 圓通要義：門門皆可入道，歸元性無二

經中明言：「歸元性無二，方便有多門。」雖然入手法門有二十五種，但最終證悟的真如本性是同一的。就像從長安到洛陽，可以選擇走陸路、水路、騎馬、坐車，路徑不同，但目的地相同。

二、文殊揀選圓通：「此方真教體，清淨在音聞」

2.1 文殊菩薩的選擇標準

佛陀囑咐文殊菩薩從二十五圓通中選出最適合此土眾生修行的法門。文殊菩薩以偈頌答覆，核心要點：

  1. 適合此方眾生根機：娑婆世界眾生「耳根最利」，聞性圓通易於入手
  2. 三昧最易成就：耳根不受時空限制（眼見必對明，耳聞不論動靜）
  3. 初心到極果最圓滿：從初發心到證果，耳根圓通可貫徹始終

2.2 文殊偈頌（節選）

**此方真教體，清淨在音聞；
欲取三摩提，實以聞中入。**

娑婆世界的真正教化方式，是通過聲音和聽聞來達到清淨；若想證得正定（三摩地），實際上應當從「聞性」入手。

**反聞聞自性，性成無上道；
圓通實如是，此是微塵佛，一路涅槃門。**

觀世音菩薩的法門即是「反聞聞自性」— 不向外馳求聲塵，而是反觀能聞的本性。這條路是過去無量諸佛共同走過的涅槃正路。

2.3 耳根圓通的優勝性

文殊菩薩列舉耳根優於其他五根之處：

| 對比項 | 眼根 | 耳根 |

|-------|------|------|

| 對境條件 | 必須有光明，對色相 | 不論動靜，暗室亦聞 |

| 覺照範圍 | 僅見前方，不見背後 | 十方俱聞，無遠無近 |

| 穿透性 | 隔牆不見 | 隔牆能聞 |

| 持續性 | 閉眼則不見 | 塞耳聲仍入（聞性不滅） |

耳根「動靜二相，了然不生」，最易契入如來藏性，故為此土眾生第一選擇。

三、各宗派選擇圓通的歷史實證

3.1 禪宗：眼根、意根、耳根並重

經文依據：

  • 阿那律陀「眼根圓通」— 見性不滅
  • 須菩提「意根圓通」— 意根如空
  • 觀世音「耳根圓通」— 反聞自性

禪宗應用：

禪宗以「直指人心，見性成佛」為宗旨，不執著特定法門，但歷代祖師多有依楞嚴經印證的記載：

  • 憨山大師（1546-1623）：三十歲於五臺山獨修時，因風聲干擾請益妙峰大師，被告知修「耳根圓通」。大師修習「三十年聞水聲不轉意根」，終證觀音圓通。開悟後以《楞嚴經》印證，八個月通達全經，著《楞嚴懸鏡》。
  • 虛雲老和尚（1840-1959）：強調「不讀楞嚴，不知修心迷悟之關鍵」。在開示中說：「現正是末法時代，你到那裡參訪善知識呢？不如熟讀一部《楞嚴經》，修行就有把握。」虛雲老和尚撰《楞嚴經玄要》，將楞嚴與禪宗參究結合。
  • 禪宗公案與眼根：許多公案涉及「見性」，如六祖惠能「不是風動，不是幡動，仁者心動」，即是透過眼根所見回歸心性，與阿那律陀眼根圓通異曲同工。

禪宗選擇原則：「隨緣不變，不變隨緣」，不拘泥於特定圓通，但以耳根、眼根、意根三者為主流，因契合禪宗「直指本心」之風格。

3.2 淨土宗：大勢至念佛圓通章（根大圓通）

經文依據：

大勢至菩薩（第 25 位）「念佛心」入無生忍，都攝六根淨念相繼。

核心修法：

都攝六根，淨念相繼，得三摩地，斯為第一。
  • 都攝六根：眼不觀色、耳不聽聲、鼻不嗅香、舌不嚐味、身無觸覺、意不起念，六根全攝於一句佛號。
  • 淨念相繼：念佛時念念清淨無雜，一句接一句相續不斷。

與淨土宗關係：

印光大師將《大勢至菩薩念佛圓通章》與《無量壽經》《阿彌陀經》《觀無量壽經》《普賢行願品》合稱「淨土五經」，可見此章在淨土宗的核心地位。

實修特點：

  • 不依禪定次第：不必經歷四禪八定，只需淨念相繼
  • 即凡心是佛心：「如子憶母」，以念佛心契入佛心
  • 橫超三界：帶業往生，不斷煩惱得解脫

大勢至圓通雖列第 25 位（非文殊揀選第一），但其「簡便直捷」之特性，使之成為末法眾生最契機的法門之一。淨土行者多以此為正行，輔以耳根圓通（聞佛號聲反聞自性）。

3.3 密宗：火大圓通與楞嚴咒

經文依據：

烏芻瑟摩（火頭金剛）「火大圓通」— 觀火光三昧，化多淫心成智慧火。

密宗應用：

密宗重視「三密相應」（身密手印、口密真言、意密觀想），楞嚴咒作為「咒中之王」，在密宗修法中佔重要地位。

  • 楞嚴咒五會：對應五方佛、五智，密宗行者以之為早課必誦
  • 火大三昧：密宗護摩法（火供）即是觀火性空，以火燒煩惱成智慧
  • 身心轉化：烏芻瑟摩因多淫而墮，後遇佛教導觀淫欲即菩提，化欲火為智火，此「即身成佛」理念契合密宗核心

密宗選擇原則：注重「本尊相應」，若修觀音法則取耳根圓通，若修不動明王則取火大圓通，依本尊而選擇相應圓通法門。

3.4 天台宗：耳根圓通與止觀雙運

經文依據：

觀世音菩薩「耳根圓通」— 從聞思修入三摩地。

天台宗應用：

天台宗以《法華經》為宗，重視「一心三觀」（空假中三觀）與「止觀雙運」。觀世音菩薩耳根圓通的「反聞聞自性，性成無上道」，與天台止觀高度契合：

  • 止：「入流亡所」— 制心一處，捨外塵而反觀內性
  • 觀：「動靜二相了然不生」— 觀空假中，了達聞性非動非靜

天台宗祖師註疏：

智者大師雖未親註楞嚴（楞嚴經唐代才傳入，智者已往生），但後世天台學者多以天台教觀詮釋楞嚴。蕅益大師（天台九祖）著《楞嚴文句》，以天台五時八教判釋楞嚴，認為楞嚴圓通章屬「圓教」。

天台選擇原則：以耳根圓通為主，因其「初於聞中，入流亡所」契合天台圓頓止觀，且觀音法門在天台宗中地位崇高。

四、修行者如何依根性選擇圓通

4.1 根性利鈍的判別

利根行者（上根人）：

  • 特徵：聞法即悟，不必依次第修，能直下承擔本性
  • 適合圓通：意根圓通（須菩提「意根如空」）、眼根圓通（阿那律陀「見性不滅」）
  • 修法：直接從「根性」入手，一悟即至佛地

中根行者：

  • 特徵：需要方便引導，逐步深入
  • 適合圓通：耳根圓通（觀世音「反聞自性」）、鼻根圓通（周利槃陀伽「觀息」）
  • 修法：從「觀境」或「觀息」入手，循序漸進

鈍根行者（末法眾生多屬此類）：

  • 特徵：煩惱重，根機暗鈍，需簡易直捷法門
  • 適合圓通：根大圓通（大勢至「念佛圓通」）、法塵圓通（摩訶迦葉「糞掃衣觀」）
  • 修法：持名念佛、持咒、簡單觀法

4.2 環境條件的考量

| 環境條件 | 適合圓通 | 原因 |

|---------|---------|------|

| 都市喧囂 | 耳根圓通 | 「此方真教體，清淨在音聞」，聲塵最易接觸 |

| 山林靜修 | 意根、眼根圓通 | 環境清淨，利於直觀本性 |

| 在家居士 | 念佛圓通、耳根圓通 | 簡便易行，不礙世務 |

| 專職出家 | 任何圓通皆可 | 有充足時間深入修習 |

4.3 宗派傳承的依循

原則：「依法不依人，依義不依語」，但初學者應依宗派傳承，避免盲修瞎練。

  • 禪宗弟子：依師父指示，多以「話頭」「公案」為入手，可參考眼根、耳根、意根圓通
  • 淨土弟子：以大勢至念佛圓通為正行，輔以觀音耳根圓通（聞佛號反聞自性）
  • 天台弟子：以耳根圓通為主，配合《摩訶止觀》修習
  • 密宗弟子：依上師傳授本尊，選擇相應圓通（如修觀音則耳根，修不動明王則火大）

4.4 選擇圓通的實用步驟（五步驟）

第一步：自我評估

  • 我的根性：聞法易悟？需循序漸進？煩惱深重？
  • 我的環境：在家？出家？都市？山林？
  • 我的因緣：是否已歸依特定宗派？有無師父指導？

第二步：試修體驗

  • 依文殊揀選，先試「耳根圓通」七日（反聞聞自性）
  • 若覺不契機，再試「念佛圓通」七日（都攝六根念佛）
  • 記錄身心反應：是否易入定？妄念是否減少？

第三步：師承指導

  • 有師父：請師父依你根機指示
  • 無師父：依經典祖師註疏自學，切忌東拼西湊

第四步：專一深入

  • 選定一門後，至少專修三個月到一年，不可朝三暮四
  • 「制心一處，無事不辦」

第五步：融會貫通

  • 一門深入後，旁通其他圓通，了達「歸元性無二」
  • 最終境界：「一即一切，一切即一」，圓通無礙

五、二十五圓通的現代修行應用

5.1 日常生活中的圓通實踐

耳根圓通的現代應用：

  • 聽音樂時：不執著音聲美醜，反觀能聞之性
  • 噪音環境：觀「動靜二相了然不生」，不被噪音轉
  • 念佛持咒：聽自己念佛聲，耳根與根大圓通結合

眼根圓通的現代應用：

  • 看手機時：觀「見性」不隨螢幕內容生滅
  • 讀經書時：不著文字相，直觀能見之性

念佛圓通的現代應用（最普遍）：

  • 上下班通勤：默念佛號，都攝六根
  • 工作休息：十念記數法（印光大師法），攝心念佛
  • 臨睡前：念佛十分鐘，淨念而眠

5.2 圓通與心理健康

二十五圓通不僅是證道法門，亦有心理調適功能：

  • 焦慮症：修「空大圓通」（虛空藏菩薩），觀四大本空，減輕執著
  • 憂鬱症：修「念佛圓通」，以佛號提起正念，轉移負面思維
  • 強迫症：修「意根圓通」（須菩提「意根如空」），觀念頭來去自在

注意：嚴重精神疾病應就醫治療，圓通修法為輔助，非替代醫療。

5.3 圓通法門的跨宗派整合

現代佛教走向融合，修行者可整合多種圓通：

  • 定課：早課楞嚴咒（密宗）+ 大悲咒（耳根圓通）
  • 正行：持名念佛（念佛圓通）
  • 助行：讀誦經典（眼根圓通）+ 參話頭（意根圓通）

原則：主輔分明，以一門為主，其他為助，切忌雜修無章。

六、修行應用：從理論到實踐

6.1 初學者入門建議（三階段）

階段一：聞思（一至三個月）

  • 通讀《楞嚴經》卷五圓通章
  • 選讀宣化上人《楞嚴經淺釋·二十五聖圓通章》
  • 了解二十五圓通分類與文殊揀選理由

階段二：選擇（試修一個月）

  • 依前述「五步驟」自我評估
  • 優先試修：耳根圓通（文殊揀選）或念佛圓通（契末法機）
  • 記錄修行日誌，觀察身心變化

階段三：專修（一年以上）

  • 確定一門，專精深入
  • 有條件者參加禪七、佛七、閉關
  • 定期請益善知識，調整修法

6.2 進階修行的注意事項

避免五種偏差：

  1. 貪多嚼不爛：今天修耳根，明天修念佛，後天參禪，無一成就
  2. 執理廢事：只談理論，不實修，如畫餅充飢
  3. 著相求通：修圓通不是求神通，而是契入圓通本性
  4. 離師自修：末法時代魔強法弱，無師承易入歧途
  5. 急功近利：修行如煮飯，急火反成夾生飯

把握三要訣：

  1. 深信因果：一切修行從深信因果開始
  2. 發菩提心：非為自求人天福報，而為上求佛道下化眾生
  3. 持戒清淨：楞嚴經特別強調「四種清淨明誨」，不殺盜淫妄

6.3 圓通成就的驗證標準

初步相應（一至三個月）：

  • 妄念減少，定力增強
  • 身心輕安，煩惱減輕
  • 對修法產生信心與歡喜

深入相應（一至三年）：

  • 能於定中見本性（禪定功夫）
  • 夢中亦能覺照，不隨境轉
  • 日常生活中時時返照，不離當念

究竟成就（不可預期）：

  • 「忽然超越世出世間，十方圓明」（如觀音耳根圓通）
  • 證入無生法忍，見性成佛
  • 具體驗證標準見《楞嚴經》各圓通自述

重要提醒：修行以「破我執、證無我」為根本，非求神異境界。若著相求通，反成魔障。

結語：圓通無礙，契機為要

楞嚴經二十五圓通為我們展現了佛法的圓融無礙：無論從色聲香味、眼耳鼻舌、地水火風哪一法入手，只要一門深入，皆可證入如來藏性。文殊菩薩揀選耳根為第一，並非否定其他法門，而是指出「此方真教體，清淨在音聞」— 對娑婆世界眾生而言，耳根圓通最為契機。

然而，末法時代眾生根機不一，大勢至菩薩念佛圓通「都攝六根淨念相繼」，以其簡便直捷，成為普被三根的殊勝法門。禪宗行者可依眼根、意根見性，密宗行者可依火大、空大觀修，天台行者可依耳根圓通配合止觀雙運。

修行者當依自身根性、環境因緣、宗派傳承，選擇相應圓通，專一深入，久久功純，自然「忽然超越世出世間，十方圓明」，契入楞嚴大定。切記「歸元性無二，方便有多門」，莫執一非餘，亦莫貪多嚼不爛。一門即一切門，一切門歸一門，如是修行，方為真圓通也。
