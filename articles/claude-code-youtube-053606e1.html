Claude Code + YouTube 自動化內容生產

youtuberesearchai-automationcontent-production

核心發現：「Zen」頻道僅上傳 12 部影片、累積 13 萬訂閱，即達月入逾 6.1 萬美元，證明 AI 全自動化可大幅壓縮影片製作成本與時間。實務步驟：撰寫腳本 → 配音 → 用 Turboscribe 轉錄並提取時間戳 → 依時間戳做提示工程，由 Hickfield 生成對應圖像 → Claude Code 自動下載與重新命名素材 → 影片軟體拖放同步，全程不到 20 分鐘完成。應用場景：適合冥想、故事、知識型等「旁白＋配圖」頻道。視覺上採每 2–3 秒高頻換圖以提升完看率；配音因 11 Labs 政策禁用，改採自錄或 Fiverr 外包；並需留意 YouTube「影子封禁」演算法風險，調整發布策略以維持曝光。影片來源：https://www.youtube.com/watch?v=WODnqHPLR38 (Danny Why, 2026-05-30)
