Build a proactive agent workflow with Claude Code — Routines 功能介紹

youtuberesearchClaude Coderoutinesproactive agentauto

Anthropic Applied AI 團隊的 Maya 在 Code with Claude 大會介紹全新 Routines 功能，將 Claude Code 從「被動工具」升級為「主動隊友」。Routines 解決三大痛點：(1) 運行環境——不依賴本機，由 Claude Code 託管基礎設施處理 hosting、session state 與 connector 認證；(2) 觸發機制——支援 cron 定時、GitHub event 以及自訂 webhook 觸發，無需自建排程架構；(3) 可互動性——每個 routine 本質是一個 Claude Code session，可在 Web/CLI/Desktop 即時觀看、追蹤、引導與恢復。實際案例：Anthropic 內部用 routines 自動化文件產生。開發者只需定義 prompt、repo、connector 與 trigger，其餘由平台處理。
