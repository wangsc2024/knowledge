專案觀測摘要

本次 github_scout 主題：LLM 可觀測性/trace（Langfuse/Phoenix/OpenLLMetry）。

1. langfuse/langfuse（26.3k★）

  • 開源 LLM 工程平台：可觀測性、eval、Prompt 管理，整合 OTel/LangChain/OpenAI SDK/LiteLLM
  • MIT 授權，YC W23，可自託管
  • 2026 年初達 20k★，4月已 26.3k★
  • OTel OTLP 原生支援，可作為統一 trace backend

2. Arize-ai/phoenix（9.4k★）

  • AI 可觀測性與 Evaluation：OTel-based，Agent Graph 可視化
  • 支援 Claude Agent SDK、Cursor、Google ADK、Mastra、CrewAI
  • 提供 MCP Server：可讓 Agent 自行查詢 trace
  • agent loop 除錯：spanning 工具呼叫 → 圖形化 DAG 流程

3. traceloop/openllmetry（6.9k★）

  • 基於 OTel 的開源 GenAI SDK，40+ 自動儀器化（OpenAI/Anthropic/Ollama）
  • gen_ai.system 語意慣例：opentelemetry-instrumentation-ollama → gen_ai.system=ollama, gen_ai.operation.name=chat
  • OTel v1.38.0：gen_ai.prompt / gen_ai.completion 已廢棄，改用 gen_ai.input.messages / gen_ai.output.messages / gen_ai.system_instructions
