What Is Agentic Coding? How AI Agents Modernize Code

youtuberesearchAI agent framework

IBM 說明 agentic coding 如何支援 Java 老舊系統現代化：AI 先分析 call path/資料流/隱藏耦合，再劃安全邊界、生成測試、人工審核後迭代；以金融即時風控評分為例。

關鍵發現：60–70% 開發時間花在理解上下文；現代化痛點是「遺失的理解」而非舊碼本身。

適用情境：legacy 改構前 discovery、合規敏感系統邊界切分。
