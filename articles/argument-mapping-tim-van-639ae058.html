論證圖譜（Argument Mapping）— 批判性思考教學效果最強的視覺化推理方法論

本文為「邏輯思維」系列 foundation 階段，承接 KB 既有的論證謬誤（Logical Fallacies）與結構化分析技術（SATs），深入探討將論證結構視覺化的系統性方法。

---

一、概述

論證圖譜（Argument Mapping） 是一種將論證的邏輯結構以方框與箭頭圖表呈現的方法。它將論證的核心組件 — 結論（conclusion）、前提（premises）、反駁（objections）、回應（rebuttals） — 明確視覺化，讓推理的每一步都清晰可見。

為什麼它是「最熱門」的邏輯思維工具？

  • 效果量 0.8 標準差：高強度論證圖譜課程對批判性思考測驗（如 CCTST）的提升效果，是標準批判性思考課程的 2 倍以上，是大學三年自然增長（0.55 SD）的 1.5 倍
  • 5-6 倍增幅：相比普通大學一學期的自然增長（~0.13 SD），論證圖譜課程可達 5-6 倍效果
  • 跨領域驗證：不僅在大學教育，在情報分析、法律推理、商業決策中都有顯著效果

二、核心組件

2.1 基本元素

| 組件 | 定義 | 視覺表示 |

|------|------|----------|

| 主張（Claim） | 具有真假值的陳述句 | 方框 |

| 結論（Conclusion） | 論證要證明的主張 | 最頂層方框 |

| 前提（Premise） | 支持結論的理由 | 下層方框 |

| 共同前提（Co-premises） | 必須聯合才能支持結論的前提群 | 合併箭頭指向結論 |

| 獨立前提（Independent premises） | 各自獨立支持結論的前提 | 各有獨立箭頭 |

| 反駁（Objection） | 對前提或結論的質疑 | 紅色方框/虛線箭頭 |

| 回應（Rebuttal） | 對反駁的回應 | 綠色方框 |

2.2 共同前提 vs 獨立前提（關鍵區分）

共同前提（Linked/Co-premises）：

  • 兩個以上前提「攜手合作」支持結論
  • 移除任一前提，其餘前提對結論的支持將顯著削弱
  • 範例：「巴黎在法國（P1），法國在歐洲（P2）。所以巴黎在歐洲（C）」— P1 和 P2 缺一不可

獨立前提（Convergent/Independent）：

  • 各前提獨立提供對結論的支持
  • 移除一個前提，其餘仍可支持結論
  • 範例：「吸菸有害，因為它致癌（P1），也增加心臟病風險（P2）」— P1 或 P2 各自都能支持結論

2.3 兩大規則（隱藏前提發掘）

  • 兔子規則（Rabbit Rule）：論證的結論中不應突然出現前提中從未提及的概念（不能「變出兔子」），若有則代表缺少一個隱含前提
  • 牽手規則（Holding Hands Rule）：共同前提必須「手牽手」一起出現，不能單獨呈現其中一個而省略另一個

三、Tim van Gelder 八步建圖法（CAAM 方法論）

Tim van Gelder（墨爾本大學副教授）是論證圖譜領域的奠基人物，他的核心觀點：非正式推理是一種技能，技能透過刻意練習（deliberate practice）提升。

八步建圖流程

  1. 編號主張：為段落中的每個獨立陳述編號
  2. 確定結論：問「這段話的重點是什麼？」找到核心主張
  3. 消除冗餘：移除重複或無關的陳述，確保每句都是完整句子
  4. 合併同類：將表達相同意思的主張合併
  5. 應用抽象原則：上層主張比下層更抽象、更概括
  6. 維持層級一致：同一水平層的主張應具有相似的抽象程度
  7. 應用兔子規則與牽手規則：發掘隱含前提，使論證完整
  8. 區分共同與獨立前提：透過依賴性分析確定前提的連結方式

為何視覺化比純文字有效？

  • 降低認知負荷：圖形結構釋放工作記憶容量
  • 聚焦推理步驟：每個推理環節獨立呈現，便於逐一檢驗
  • 促進刻意練習：學生可反覆修改圖譜，獲得即時結構性反饋
  • 協作而非對抗：課堂辯論從「誰贏」轉為「論證結構是否完善」

六、與 KB 既有知識的交叉連結

  • 論證謬誤（Logical Fallacies）：論證圖譜是偵測謬誤的最佳工具 — 當你畫出圖譜後，前提跳躍（稻草人）、循環論證、滑坡謬誤等都一目了然
  • 結構化分析技術（SATs）：ACH 競爭假設分析本質上是論證圖譜的特化版 — 每個假設都是一個結論，證據是前提
  • 費米估算：費米估算的分解步驟可以用論證圖譜呈現，每個分解假設都是一個前提節點
  • 心智模型格柵：論證圖譜是「元模型」— 它不是某個特定心智模型，而是用來檢驗任何心智模型推理品質的方法
  • Scout Mindset：偵察兵心態的核心是「真相優先」，論證圖譜提供了實踐工具 — 強迫你呈現反駁與回應，避免確認偏誤

七、快速上手指南（5 分鐘版）

  1. 選一篇社論或新聞評論
  2. 找出核心主張（結論），寫在紙頂端
  3. 找出支持它的理由（前提），寫在下方
  4. 判斷各前提是共同的（必須一起）還是獨立的（各自成立）
  5. 畫箭頭：共同前提合併→一支箭頭指向結論；獨立前提各→一支箭頭
  6. 問自己：有沒有隱含前提？（兔子規則）有沒有遺漏的共同前提？（牽手規則）
  7. 加入可能的反駁和回應

八、實踐建議

待辦事項整合

  • 對重要決策用論證圖譜分析 pros/cons，而非僅列清單
  • 將論證圖譜作為會議討論的輸出格式，提升討論品質
  • 搭配 Kialo Edu（免費）練習每週一次論證圖譜

刻意練習計畫

| 週次 | 目標 | 練習 |

|------|------|------|

| 1-2 | 基礎識別 | 每天一段文字→找結論 + 前提 |

| 3-4 | 結構分析 | 區分共同/獨立前提，發掘隱含前提 |

| 5-6 | 完整建圖 | 含反駁與回應的多層論證圖 |

| 7-8 | 自主應用 | 將自己的決策用論證圖分析 |
