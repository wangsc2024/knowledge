  • - > 系列：教觀綱宗／天台判教（mechanism 深化）。與 KB 內〈化法四教之藏教〉、〈方等時說通別圓〉、〈八教交叉矩陣〉互補：本文專論化法第二門「通教」之教相與止觀入口，釐清「通前藏、通後別圓」與化儀四教（頓漸秘密不定）不可混同。
  • - 通教是天台「化法四教」中承上啟下之一環：就所詮，說即空無生之四真諦（無生四諦），使三乘同觀無生而共學；就所被，以菩薩為正機、二乘為傍機，故名「通」——通前同於藏教所證偏真，通後可接入別教但中與圓教即具。《教觀綱宗》全書以「教觀相資」為骨，通教正對應「體假即空」的觀門（與藏教「析假入空」相對），在漸教中多屬般若方等段之共般若層次。五時上屬般若時（亦與方等帶通相涉），化儀上則隨頓漸秘密不定之施設而轉，不可將「通教」誤等同於「漸教」或「不定教」。
  • - 白話：沒有教法，觀行即無所依而起；沒有觀行，教法便成徒設。通教之「教」在無生四諦，「觀」在體達即空；二者相資，方成進道。
  • - 1. 化法 vs 化儀：化法四教是「藥味」（藏通別圓）；化儀四教是「藥方」（頓漸秘密不定）。通教是藥味之一，可配入漸教多段（鹿苑後之方等、般若等），亦可在不定益中與他教並現——判教時先立化法，再論化儀與五時縱橫。 2. 五時：通教義多顯於般若時（廣明空慧，淘汰小執），與方等時「帶通別圓」共說之通相呼應；最終仍須以法華涅槃時開顯收束（此為系列他篇已述之軸線）。 3. 通前通後（化法內部）：鈍根證同偏真，與藏教邊際相接；利根即於空理達「不但空」，可入別圓。此與《天台四教儀》以般若部為通教所依之傳統讀法一致。 4. 觀門：通教用體法入空（體空觀），止門上常配體真止（天台三止中，對應藏通之真諦止息處），異於藏教析假入空之層層拆解。

  • 本研究以巴利尼柯耶、漢譯阿含、佛教學術專著與近代研究論文為基礎，分析四聖諦與八正道在佛教教義中的結構性関係。研究指出：四聖諦提供佛教對生命困境、成因、止息可能與實踐方法的總體框架，八正道則是道諦最具體、最可操作的展開。兩者並非平行條目，而是「問題診斷與解決路徑」「總綱與分項」「理解與修行」的多重互補関係。正見在其中扮演関鍵樞紐，使四諦的理解能轉化為八道的實踐，並在正念、正定中深化為對滅諦的證入。
  • 本報告採取四層方法。第一層是原典對讀，選取 SN 56.11《轉法輪經》、MN 141《分別聖諦經》、MN 117《大四十經》與 MN 9《正見經》，比對其中四聖諦與八正道的共現位置、語義展開與修行脈絡。第二層是平行文本比較，納入《雜阿含經》相関條文與 Anālayo 對《轉法輪經》漢譯平行本的研究，以檢驗「道諦即八正道」是否屬早期跨傳承共通教法。第三層是教義與概念分析，利用 Bhikkhu Bodhi、Rupert Gethin、Peter Harvey、Stephen J. Laumakis 等學者的專著與教材，說明八正道如何在戒、定、慧三學中重組，並回扣四聖諦的問題意識。第四層是爭議比較，處理 ariya-sacca 應譯為「聖諦」還是「聖者所證真實」、以及解脫是否可被拆成「純智性理解」與「純禪定體驗」兩路徑等議題。
  • 文本分析上，本研究特別関注三種連結方式。其一，是經文中直接把道諦表述為八正道的段落，這類段落提供最基本的從屬関係證據。其二，是把正見界定為理解四聖諦的支分，或把四聖諦作為正見內容的段落，這顯示四諦與八道之間存在反向滲透。其三，是把八正道視為整體修行工程，而非單列倫理條目的詮釋，這類材料可說明兩者何以在佛教教義中共同構成理論與實踐的雙重骨架。基於這套方法，本研究不把四聖諦簡化為純理論，也不把八正道簡化為修身守則，而是從結構、功能、修行與詮釋層次同時處理。
  • 四聖諦是佛教最常被視為核心的教法架構。從 SN 56.11 所呈現的初轉法輪語境看，佛陀不是先提出一套形上學宇宙論，而是以修行者最迫切的問題開始：此生何以充滿逼迫、不滿足與反覆輪轉。苦諦因此不是單純情緒上的苦，而是對一切有漏經驗之不穩定性與不可依恃性的總體判定。MN 141 進一步把苦諦拆解為生、老、病、死、怨憎會、愛別離、求不得，以及略說五取蘊苦，顯示佛教對苦的理解既有存在論面向，也有心理與倫理面向。換言之，苦諦不是悲觀命題，而是修道的現實前提：若不如實見苦，便不會啟動止苦工程。
  • 集諦在四諦中標示苦的生成條件，核心是貪愛，但不止於表面的欲求。從早期佛典來看，貪愛之所以能成為苦因，總是與無明、執取與條件性生起相連。這使集諦具有動態特徵：它不是一個單一原因，而是一整個使苦持續再生的心理與存在機制。滅諦則指出這套機制可以被止息，涅槃因此不是遙遠宗教理想，而是「苦的終止」所標示的可實現狀態。若以醫療隱喻表述，苦是病況，集是病因，滅是痊癒，道是療法。這個結構之所以重要，在於它同時避免了宿命論與虛無論。佛教既不說苦無法改變，也不說苦是幻覺而不必處理。
  • 在歷史發展上，四聖諦從早期經典開始便具備高度教學功能。Anālayo 對《轉法輪經》漢譯平行本的研究指出，不同傳承雖在敘事細節與用語上有差異，但四諦與道支的核心結構相當穩定。這意味四聖諦並非後期系統化後才被置入早期教法，而是很早就承擔總綱角色。到了後續注釋與現代詮釋中，四聖諦開始被用不同語彙重述。例如 Harvey 認為 ariya-sacca 不宜僅理解為命題性的「真理」，而更接近聖者所如實證知的真實狀態。若接受此點，四聖諦便不只是知識內容，而是待被洞見的存在實相。如此一來，道諦也不只是「第四條真理」，而成為通向證知前三諦與實現滅諦的必要通道。這種詮釋能更好說明四諦何以總是與修行而非單純思辨連在一起。
  • 漢譯阿含系統中，四聖諦也維持相似教學功能。BuddhaSpace 對《雜阿含》相関條文的導讀顯示，「道」明確指向八正道，而四諦被定位為修行者必須如實知、如實斷、如實證、如實修的四組任務。這種「知、斷、證、修」的操作性，也說明四聖諦不是靜態教條，而是一種實踐指令結構。就此而言，四聖諦的歷史穩定性，不在於所有傳統都用同一套術語，而在於不同傳承都承認：佛法不是先講世界起源，而是先處理苦與滅苦。

  • 若本變更必須在一週內回退，是否能在不影響既有 SLA 的情況下達成？
  • 該變更是否會影響 fitness function 的任一指標（success_rate / cache_hit_rate / token_budget）？
  • 是否存在更輕量的替代方案，能達成同樣的目的？
