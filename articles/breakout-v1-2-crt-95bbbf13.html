Breakout (打磚塊) v1.2 優化心得

概述

為經典 Breakout 打磚塊遊戲加入 CRT 復古濾鏡和動態星空背景，提升視覺沉浸感。

CRT 復古濾鏡實作

三層效果疊加

  1. 掃描線：每 3px 繪製一條半透明黑線（rgba(0,0,0,0.12)），模擬 CRT 水平掃描線
  2. 暈影效果：createRadialGradient 從中心到邊緣漸暗（0→0.4 opacity），聚焦玩家視線
  3. RGB 偏移微閃：用 globalCompositeOperation: 'lighter' 加上微弱紅藍偏移，模擬色差

磚塊螢光暈染

  • CRT 開啟時，磚塊繪製前設定 shadowColor 為磚塊自身顏色
  • shadowBlur = 6 * scale，產生柔和螢光外溢效果
  • 繪製後立即重置 shadowBlur = 0，避免影響後續元素

效能考量

  • 掃描線用簡單 fillRect 而非 pixel manipulation，GPU 友善
  • 暈影用單一 radialGradient，每幀只建立一次
  • RGB 偏移用 Math.sin 動態計算，極低開銷
  • 整體 CRT 效果對 60 FPS 幾乎無影響

動態星空背景

實作方式

  • 固定 60 顆星星的陣列（非動態分配）
  • 每顆星有：位置、大小（0.5-2px）、下移速度（0.1-0.3px/frame）、閃爍相位
  • 閃爍：alpha = 0.3 + 0.4 * sin(time + phase)，營造自然感
  • 星星移出底部後重置到頂部隨機位置

取代靜態格線

  • 原始版本用 20px 間距格線（rgba(100,100,200,0.06)）
  • 星空背景比格線更有太空感，且動態效果增加層次

使用者控制

  • 按 T 鍵切換 CRT 濾鏡開關
  • 狀態存入 localStorage('breakout_crt')，跨次保留
  • 切換時顯示 1 秒「CRT ON/OFF」視覺回饋
  • 選單畫面顯示快捷鍵提示

技術要點

  • globalCompositeOperation: 'lighter' 用於 RGB 偏移時的顏色疊加
  • CRT 效果在 overlay 畫面（選單/暫停）之前繪製，確保 UI 文字不受掃描線干擾
  • 所有新效果零記憶體動態分配（預分配陣列 + 物件重用）
