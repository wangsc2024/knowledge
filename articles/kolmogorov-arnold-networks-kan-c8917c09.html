Kolmogorov-Arnold Networks (KAN) — 挑戰 MLP 的下一代神經網路架構

主題概述

Kolmogorov-Arnold Networks（KAN）是由 MIT 的 Ziming Liu 等人於 2024 年提出的神經網路架構，基於 Kolmogorov-Arnold 表示定理，以可學習的邊上激活函數取代傳統 MLP 的固定激活函數 + 可學習權重模式。KAN 在 ICLR 2025 正式發表，2025-2026 年間迅速擴展至量子計算（QKAN）、視覺 Transformer（KAT）、圖神經網路（KA-GNN）、基因組學、強化學習等領域，成為深度學習架構創新的重要方向。

---

一、數學基礎：Kolmogorov-Arnold 表示定理

1.1 定理陳述

任何多變量連續函數 f(x₁,⋯,xₙ) 都可以分解為有限個單變量函數的組合與加法：

f(x₁,⋯,xₙ) = Σ_{q=1}^{2n+1} Φ_q(Σ_{p=1}^{n} ϕ_{q,p}(x_p))

這個定理表明：高維函數本質上可以由一維函數構成，這是 KAN 架構設計的理論基石。

1.2 從定理到網路

KAN 將此定理推廣為深度網路：每一層由單變量函數矩陣 Φ = {ϕ_{q,p}} 構成，透過層疊實現任意函數逼近。與 MLP 的關鍵差異在於：非線性不在節點上（固定激活函數），而在邊上（可學習激活函數）。

---

二、架構設計與 MLP 對比

2.1 核心架構差異

| 特性 | MLP | KAN |

|------|-----|-----|

| 激活函數位置 | 節點上（固定） | 邊上（可學習） |

| 激活函數類型 | ReLU/GELU 等固定函數 | B-spline 參數化的可學習函數 |

| 線性變換 | 權重矩陣 W | 無獨立權重矩陣，整合在激活函數中 |

| 參數量擴展 | O(N²L) | O(N²LG)，G 為網格間隔數 |

| 可解釋性 | 黑箱 | 可視覺化每條邊的激活函數 |

2.2 B-Spline 激活函數

每條邊上的激活函數定義為：

ϕ(x) = w_b · b(x) + w_s · spline(x)

其中 b(x) 為基函數（SiLU），spline(x) 為可學習的 B-spline。初始化策略：spline 近零，基函數用 Xavier 縮放。

2.3 網格延伸（Grid Extension）

KAN 的獨特優勢：訓練過程中可漸進式精煉 spline 網格，提升逼近精度而無需從頭重訓。這是 MLP 無法做到的。

---

三、性能基準與擴展定律

3.1 逼近理論優勢

定理 2.1：對於具有 (k+1) 次連續可微分量的函數，KAN 的逼近誤差為 C·G^{-(k+1)+m}，與輸入維度無關，從而突破維度詛咒。

  • KAN 理論預測的擴展指數 α=4，而 MLP 的 α=(k+1)/d 隨維度 d 增大而衰減
  • 實驗：2 層 width-10 的 KAN 在 PDE 求解上比 4 層 width-100 的 MLP 精確 100 倍
  • MNIST 上 TorchKAN 達到 97%+ 準確率，評估時間 0.6 秒（RTX 4090）

3.2 訓練速度權衡

  • FasterKAN 是最快的 KAN 變體，但反向傳播仍比 MLP 慢約 2 倍
  • Wav-KAN（小波基函數）在訓練速度和魯棒性上優於 B-spline KAN
  • 前向傳播平均 500 微秒（Legendre 多項式快取優化）

---

四、可解釋性與科學發現

4.1 四步簡化流程

  1. 稀疏化：L1 正則化 + 熵懲罰，識別重要激活函數
  2. 可視化：透明度按激活幅度縮放，聚焦關鍵連接
  3. 剪枝：基於進出分數的節點級移除
  4. 符號化：將懷疑的函數設定為已知符號形式

4.2 科學發現實例

  • 物理學：成功重新發現能量守恆、動量守恆、簡單擺的拉格朗日量、相對論質量關係
  • 數學（結理論）：自動發現緊湊的拓撲不變量關係
  • 凝聚態物理（Anderson 局域化）：識別相變邊界和序參量
  • 持續學習：KAN 的局部 spline 基底避免災難性遺忘 — 只重塑有數據的區域，不影響先前學習

---

五、KAN 2.0：科學發現引擎

2025 年發表的 KAN 2.0（發表於 Physical Review X）引入三大新功能：

  1. MultKAN：增加乘法節點，更清晰揭示數據中的乘法結構
  2. Kanpiler：KAN 編譯器，將符號公式編譯為 KAN 網路
  3. Tree Converter：將 KAN（或任意神經網路）轉為樹狀圖

科學應用：發現守恆量、對稱性、拉格朗日量和本構定律。框架目標是彌合現代 AI 的連結主義與科學所需的符號主義之間的鴻溝。

---

六、2025-2026 前沿應用

6.1 量子計算：QKAN（2026，Nature npj Quantum Information）

量子 Kolmogorov-Arnold 網路（QKAN）繼承 KAN 的組合結構，基於 block-encoding，使用量子奇異值變換（QSVT）遞迴構建。應用於量子機器學習和多變量態製備。

6.2 視覺 Transformer：KAT（ICLR 2025）

Kolmogorov-Arnold Transformer（KAT）以 KAN 層取代 Transformer 中的 MLP 層：

  • 使用有理函數作為 KAN 基函數
  • 群組參數共享 + 方差保持初始化
  • KAT-B 在 ImageNet-1K 達 82.3%，超越 ViT 3.1%
  • 支援影像辨識、物體偵測、語意分割

6.3 圖神經網路：KA-GNN（2025，Nature Machine Intelligence）

將 KAN 整合入 GNN 架構，用於分子性質預測，提升準確性與可解釋性。

6.4 基因組學（2025，PMC）

線性 KAN（LKAN）在基因組序列分類和生成任務上，幾乎在所有資料集上都優於基線 MLP 和卷積 KAN（CKAN）。

6.5 生物資訊學：酶委員會編號預測（2025，Nature npj AI）

KAN 整合使微平均 F1 分數提升 15.7%，巨集平均 F1 分數提升 34.2%。

6.6 強化學習：KAN-PPO

Conv-KAN-based PPO 在 HalfCheetah 和 Hopper 環境分別超越 MLP PPO 約 12.9% 和 108.7%。

6.7 材料科學：MOF-KAN（2026，J. Phys. Chem. Lett.）

用 KAN 進行金屬有機框架的數位發現。

6.8 大型語言模型：KALLM（2025 OSRE）

基於 KAN 的 Transformer 實作開源 LLM SmolLM2，探索 KAN 在語言模型中的可行性。

---

七、主要挑戰與限制

  1. 擴展性：隨著網路規模增大，可解釋性優勢遞減
  2. 訓練速度：即使是最快的 FasterKAN，反向傳播仍比 MLP 慢 2 倍
  3. 高維度問題：B-spline 在高維函數上面臨維度詛咒
  4. 生態系統成熟度：PyTorch 實作（pykan、efficient-kan、torchkan）仍在快速迭代中
  5. 大規模部署：尚未有 KAN 在數十億參數規模的成功部署案例

---

八、未來研究方向

  1. KAN + LLM：KALLM 專案探索 KAN 取代 Transformer 中 MLP 層的可行性
  2. 量子 KAN：QKAN 開闢量子機器學習新範式
  3. 持續學習：利用 KAN 的局部 spline 特性解決災難性遺忘
  4. AI for Science：KAN 2.0 的符號發現能力可能成為科學研究的標準工具
  5. 混合架構：KAN + MLP、KAN + Attention 的混合設計可能兼顧效率與可解釋性
  6. 高效實作：Wav-KAN（小波基函數）、FourierKAN（傅立葉基函數）等變體持續探索更快的實作
