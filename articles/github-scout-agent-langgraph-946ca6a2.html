專案觀測摘要

1. langchain-ai/langgraph（31,061★，REST 2026-05-03）

  • runtime_model：圖式編排（directed graph，Pregel 啟發），支援 cyclic graph 與 stateful 長跑工作流
  • state_strategy：durable checkpointing（中間態寫入持久存儲），短期 + 長期記憶，失敗自動從 checkpoint 恢復
  • human_review_model：interrupt-point 模式（圖執行遇 breakpoint 暫停等待人工 approve/edit，同步短延遲）
  • URL：https://github.com/langchain-ai/langgraph

2. openai/openai-agents-python（25,784★，REST 2026-05-02）

  • runtime_model：handoff-based delegation（agents 透過 explicit handoff 轉移控制，無顯式 graph）
  • state_strategy：session-based automatic history management，內建 tracing
  • human_review_model：interrupt-point 高層抽象（built-in human_in_the_loop approval callback）
  • URL：https://github.com/openai/openai-agents-python

3. microsoft/agent-framework（10,053★，REST 2026-05-01）

  • runtime_model：multi-agent orchestration with policy-as-code（企業級，Python + .NET）
  • state_strategy：policy-driven state，audit trail 為狀態一部分
  • human_review_model：policy-as-code gate（policy engine 決定審核需求，signal-based 風格，異步）
  • URL：https://github.com/microsoft/agent-framework
