Space Invaders v2.2 優化心得：CRT 復古濾鏡與動態背景音樂系統

優化概述

在 v2.1（道具系統）基礎上，加入兩大復古體驗增強功能：

  1. CRT 復古濾鏡：掃描線、暗角、螢幕微閃爍、偶發干擾線
  2. 動態背景音樂：8-bit 合成 BGM，節拍隨遊戲進程動態加速

CRT 濾鏡實作要點

效能優化策略

  • 預渲染離屏 Canvas：掃描線和暗角漸層只在首次呼叫時繪製到 offscreen canvas，之後每幀用 drawImage 疊加
  • 避免 getImageData/putImageData：原始方案每幀讀寫像素資料會嚴重拖累 FPS，改用純繪製疊加
  • 效能影響：從 ~15ms/幀（像素操作方案）降到 <1ms/幀（預渲染疊加方案）

四層效果組成

  1. 掃描線：每 3px 一條 1px 半透明黑線（opacity 0.08），預渲染到 offscreen canvas
  2. 暗角（Vignette）：徑向漸層，中心透明到邊緣 0.35 透明度黑色
  3. 螢幕微閃爍：淡綠色 rgba 疊加，sin 函數產生微弱脈衝（模擬 CRT 磷光閃爍）
  4. 干擾線（Glitch Band）：低機率（sin > 0.97）出現水平白色條紋

使用者控制

  • 按 C 鍵 切換 CRT 開關
  • 偏好設定持久化到 localStorage('si_crt')
  • HUD 顯示 CRT ON/OFF 狀態

動態背景音樂系統

架構設計

  • 使用 setTimeout 排程每個音符（非 setInterval，允許動態調整間隔）
  • C 小調五聲音階（pentatonic scale）確保和諧
  • 三軌混音：低音 bass（三角波）、旋律 melody（方波）、高音重音 accent（正弦波）

節拍模式

  • BASS_PATTERN：8 拍循環，每拍一個 bass note
  • MELODY_PATTERN：16 拍循環，含休止符（-1），旋律每隔一拍出現
  • 每 8 拍的第一拍加入高音重音

動態節拍（核心創意）

  • 節拍速度與存活外星人比例掛鉤
  • 公式：tempo = max(120, 120 + 180 * aliveRatio) ms/beat
  • 全員存活：300ms/beat（輕鬆）→ 少數存活：120ms/beat（緊張）
  • 完美呼應經典 Space Invaders 的逼近感

狀態機整合

  • READY→PLAYING：啟動 BGM
  • PLAYING→PAUSED：停止 BGM
  • PAUSED→PLAYING：重啟 BGM
  • GAME_OVER / LEVEL_CLEAR：停止 BGM

學到的教訓

  1. Canvas 像素操作很貴：getImageData + putImageData 每幀 ~15ms，offscreen canvas 疊加 <1ms
  2. setTimeout vs setInterval for BGM：setTimeout 允許動態改變間隔，配合 tempo 變化更靈活
  3. Web Audio 音量控制：BGM 音量要比音效低很多（0.025-0.04 vs 0.06-0.12），避免遮蓋遊戲音效
  4. localStorage 持久化 CRT 偏好：玩家不必每次重開都手動切換
