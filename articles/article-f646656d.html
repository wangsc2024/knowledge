核心經文／義理摘錄：

  • 「七處徵心」是《大佛頂首楞嚴經》卷一的核心教義，記載了佛陀與阿難尊者関於「心在何處」的七次問答。佛陀透過嚴密的邏輯詰問，逐一破斥阿難對心的七種執著，最終引導他認識到：凡夫所執著的「能思惟、能分別」的心是虛妄不實的「緣塵分別心」，而非本具的「妙明真心」。這是理解整部楞嚴經「破妄顯真」教義的基礎。
  • 阿難尊者外出乞食時，遭摩登伽女之母以「先梵天咒」幻術迷惑，幾近毀犯戒律。佛陀遙知此事，敕令文殊菩薩持楞嚴咒前往救護。阿難歸來後痛悔不已，懇請佛陀開示最根本的修證法門。
  • 佛陀指出：「一切眾生從無始以來，生死相續，皆由不知常住真心、性淨明體，用諸妄想，此想不真，故有輪轉。」為了破除阿難的妄執，佛陀七次詰問：「汝之識心，今何所在？」
  • 佛陀的破斥：如果心在身內，應該先看見身體內部的心、肝、脾、胃等臟腑，然後才能看見外面的事物。但實際上我們看不見自己的內臟，可見心不在身內。
  • 白話理解：這就像說有人住在房子裡，應該先看到房子內部，再看到窗外。但我們從未「看見」自己的內臟運作，所以心不可能單純地「住在身體裡面」。
  • 佛陀的破斥：如果心在身外，則身與心應該互不相干、各不相知。但事實上，心能感知身體的痛癢冷熱，身心是相互関聯的，可見心不在身外。
  • 白話理解：如果心真的在身體外面，那麼當身體被針刺時，心應該感覺不到痛。但實際上我們會立刻感到疼痛，說明身心密切相連，心不可能完全在身外。
  • 佛陀的破斥：如果用琉璃比喻眼睛，那麼當琉璃罩眼時，眼睛應該也能看到琉璃本身。如果眼睛能被看見，那眼睛就變成外境了，與心就是兩回事。既然如此，怎能說心潛伏在眼根裡呢？
  • 白話理解：這是邏輯上的矛盾——如果心藏在眼睛裡「透過眼睛看」，那心也應該能看到眼睛本身。但我們從未「看見」自己的眼睛在看東西。
  • 佛陀的破斥：閉眼時所見的暗，與眼睛相對嗎？如果相對，暗就在眼前，怎麼能說是「見內」？如果暗不與眼相對，又如何能被看見？這是自相矛盾的。
  • 白話理解：閉眼時看到的黑暗，如果是「內部」的景象，那應該是在眼睛後面。但暗明明就在眼前，說明閉眼見暗並非「見內」。
  • 佛陀的破斥：如果心隨和合而生，那心就沒有固定的實體。如果心有實體，那它是從內出還是從外來？是一個心還是多個心？都無法確定。而且，你用手摸身體時能感覺到，這感覺是從內出還是從外來？
  • 白話理解：如果心只是「和合」的產物，就像水和麵粉和成麵團，那麵團裡哪個部分是水、哪個是麵粉呢？分不清楚。心也不能只是「隨緣和合」的東西。
  • 佛陀的破斥：眼根有知覺，外塵無知覺，中間的心體是兼具兩者還是不兼？如果兼具，則有知無知混雜不清；如果不兼，則無所依附，沒有實體。居中之說站不住腳。
  • 白話理解：「中間」是個模糊的概念。眼睛能感知、石頭不能感知，那「中間」的心是能感知還是不能？說不清楚，所以這個定位也不成立。
  • 佛陀的破斥：如果「不著」就是心，那世間水陸飛行的一切事物，心都不著。這些事物是有還是無？如果說無，就像龜毛兔角一樣不存在，那還有什麼「不著」可言？如果說有，物在則心亦在，又怎能說無著？
  • 白話理解：「什麼都不執著」聽起來很高明，但如果連「存在」或「不存在」都說不清楚，那這種「不著」只是概念上的遊戲，不是真正認識了心。
  • 七處徵心的目的不是要找到心的「物理位置」，而是要破除阿難對「妄心」的執著。阿難所認定的心，是「能思惟、能推理、能分別」的心，這是隨著因緣生滅的「識心」或「妄心」，並非真正的「妙明真心」。
  • | 術語 | 解釋 | |------|------| | 妄心 | 隨因緣生滅、攀緣外境的分別識心，是輪迴的根本 | | 真心 | 常住不變、本自清淨的妙明覺性，即佛性、如來藏 | | 緣塵分別 | 心識攀緣外塵（六塵）而產生的分別作用 | | 周遍法界 | 真心不在內外中間，而是遍滿一切處，無在無不在 | | 達妄本空 | 了達虛妄心本來就是空的、不真實的 | | 知真本有 | 知道真如心（佛性）是本來就具有的 |
  • 1. 心無定處：真心不在內、不在外、不在中間，而是「周遍法界，無在無不在」。凡夫用有限的思維去找無限的真心，當然找不到。
  • 2. 用妄找真的矛盾：阿難用「妄心」去尋找「真心」，就像用眼睛去看眼睛一樣，是根本性的錯誤。這正是佛陀要點破的。
  • 1. 覺察分別心：在日常起心動念時，觀察自己的心是如何「攀緣」外境的。看到喜歡的東西想要，看到討厭的東西排斥——這些都是「緣塵分別」的作用。
  • 2. 不隨境轉：當情緒升起時，不急著認同或壓抑，而是觀察：這個生氣的心、這個高興的心，從哪裡來？到哪裡去？有沒有實體？
  • 3. 反問自己：仿效佛陀的詰問方式，當心被境界牽動時，問自己：「這個被牽動的心在哪裡？」透過反覆詰問，鬆動對妄心的執著。
  • 4. 認識「能知」的那個：在一切分別之中，有一個「能知道」自己在分別的覺性。這個覺性不隨分別而生滅，它更接近真心的本質。
  • - 靜坐觀心：靜坐時觀察念頭的生滅，不追不壓，只是看著。體會念頭來去，而「看著」的那個不曾移動。 - 參究話頭：可以參「念佛是誰」或「心在何處」，不求答案，只是保持疑情，讓分別心自然停歇。 - 日用覺照：在行住坐臥中保持覺知，觀察心如何因境而動，久而久之，自然能體會「心無定處」的道理。
  • 七處徵心是楞嚴經「破妄顯真」的起點。佛陀透過嚴密的邏輯，讓阿難認識到平常所執著的心是虛妄不實的。這不是否定心的存在，而是要我們超越對妄心的執著，回歸本具的妙明真心。
  • 古德說「開悟的楞嚴」，正是因為本經能直指人心，讓行者在破除妄執的過程中，逐漸體認本來面目。七處徵心的教義，至今仍是禪修與教理研究的重要指引。

核心經文／義理摘錄：

  • 「七處徵心」是《大佛頂首楞嚴經》卷一的核心對話，記載佛陀與阿難関於「心在何處」的深刻探討。阿難因被摩登伽女以咒術迷惑、險些破戒，回到佛陀座下悔責自己「多聞不務實修」，懇求開示修證法門。佛陀遂七次徵詰阿難的識心所在，每次皆予以破斥，目的是「破妄顯真」——破除眾生執著妄心為真的根本迷惑，顯現本具的妙明真心。
  • 佛陀的破斥： 若心真在身內，應當先見身中的心肝脾胃等內臟，然後才能見到外境。既然不能見到身內之物，如何能說心在身內？
  • 佛陀的破斥： 若心在身外，則身心應各不相干、互不相知。然而實際上身心能互知共感——手被針刺，心立即知痛。既然身心相知，心豈在身外？
  • 佛陀的破斥： 若心如琉璃合眼，應能同時見到眼睛本身。但當眼睛成為「被見」的對象時，眼睛與心就無法相合。又若見眼時眼即成塵境，誰為能見？
  • 佛陀的破斥： 暗境若與眼相對，則暗在眼前，豈非外境？若不相對，何以言見？閉眼所見之暗並非「內」，不能以此證明心在內外之間。
  • 佛陀的破斥： 若心無實體而隨合產生，何以能「合」？若心有體性，究竟是一體還是多體？若一體，頭與足相合則同一處；若多體，則成多心，哪個是真心？
  • 佛陀的破斥： 「中間」必有確定之處。若兼具根塵，則根塵混雜，物非體、體非物；若不兼具，離根離塵，何處為中？「中」無定位，何以安心？
  • 佛陀的破斥： 「無著」亦是一種立場。若一切物象「無」如龜毛兔角，本不存在，何須言「無著」？若物象「有」，物在則心在，何謂無著？以「無著」為心，仍是一種執著。
  • ### 原文 > 佛告阿難：「一切眾生，從無始來，生死相續，皆由不知常住真心，性淨明體，用諸妄想。此想不真，故有輪轉。」
  • ### 白話翻譯 佛陀告訴阿難：「一切眾生從無始以來，生死輪迴不斷，都是因為不認識本具的常住真心——那清淨光明的本體——反而把妄想當作真心來用。這個妄想本來不真實，所以才會輪迴流轉。」
  • | 術語 | 解釋 | |------|------| | 妄心 | 攀緣外境而生的分別心、思慮心，因緣所生、生滅變化 | | 真心 | 本具的妙明心體，常住不變、性淨明體、周遍法界 | | 徵心 | 追問心的所在，以「破妄」為手段「顯真」為目的 | | 無始無明 | 眾生本來就有、無法追溯起點的根本迷惑 | | 攀緣心 | 隨外境攀附、執取的心，是輪迴的根本原因 |
  • 4. 破執練習：當認為「我就是這樣想」、「這是我的心情」時，以七處徵心的方式自問：這個「我」在哪裡？這個「想法」住在何處？

  • - 《心經》：「無眼耳鼻舌身意」呼應真心非六根可得 - 《金剛經》：「應無所住而生其心」與第七處「無著」相應，但更進一步 - 禪宗：「即心即佛」、「見性成佛」皆以認識真心為核心
  • 「七處徵心」是入道之基石，由淺至深引導我們認識自心。佛陀並非要找出心的位置，而是要阿難（也是我們）放下對妄心的執著。當七處皆被破盡，阿難「窮其所聞終於七徵不得其心」，這才五體投地、放下自己的思想，請求佛陀開示。這正是修行的起點——知道自己「不知道」，才能開始真正的學習。
  • > 「七處徵心」重在破除妄想執著，「破妄」是為了「顯真」。真心不離不即於見聞覺知，但切勿錯認，亦不應離此而另覓真心，當下認取，當下寂滅。
