法華經：常不輕菩薩品（第20品）— 禮敬佛性、忍辱行道與末法弘法精神的圓教實踐典範

法華經佛學天台宗常不輕菩薩品一佛乘佛性戒六即佛忍辱行末法弘法圓教

法華經：常不輕菩薩品（第20品）— 禮敬佛性、忍辱行道與末法弘法精神的圓教實踐典範

一、主題概述

常不輕菩薩品（Sadāparibhūta-parivarta）為《妙法蓮華經》第二十品，位於本門流通分，夾於法師功德品（第19品）與如來神力品（第21品）之間。日蓮大聖人稱：「法華經修行之肝心在不輕品。」此品以釋迦牟尼佛的前身「常不輕菩薩」的故事，具體示範「一佛乘」理論的最直接實踐形式——禮敬一切眾生的佛性，即使面對迫害仍不改初衷。

品名「常不輕」取菩薩行持的核心：「我深敬汝等，不敢輕慢。所以者何？汝等皆行菩薩道，當得作佛。」（鳩摩羅什譯本）這二十四字被日本佛教稱為「二十四字法華」，是全經最短而最具爆炸力的宣言。

---

二、經文重點與鳩摩羅什原譯

2.1 故事背景：威音王佛時代

品章開篇設定在「威音王佛」（梵：Bhīṣmagarjitasvararāja）滅度後的「像法」時代。威音王佛在天台宗詮釋中是一切佛的「象徵性本源」，代表最初教化時代，呼應壽量品「五百塵點劫」宇宙時間觀。

2.2 常不輕的核心行持（原文段落）

原文（鳩摩羅什譯）：

「彼比丘凡有所見，若比丘、比丘尼、優婆塞、優婆夷，皆悉禮拜讚歎，而作是言：我深敬汝等，不敢輕慢。所以者何？汝等皆行菩薩道，當得作佛。」

白話：那位比丘，凡遇到任何人——不論是出家的比丘、比丘尼，或在家的男居士、女居士——都一一禮拜讚歎，並說：「我深心敬重你們，不敢有絲毫輕慢。為什麼？因為你們都在行菩薩道，將來必定成佛。」

特別值得注意：他的行持是「不專讀誦經典，但行禮拜」——不靠學問或修行形式，僅靠這二十四字宣告與禮拜行動。這揭示法華修行的根本不是知識積累，而是「見佛性、敬佛性」的存在姿態。

2.3 迫害與堅持

原文（鳩摩羅什譯）：

「如是經多年，常被罵詈，不生瞋恚，常作是言：汝當作佛。」
「以杖木瓦石而打擲之，避走遠住，猶高聲唱言：我不敢輕於汝等，汝等皆當作佛！以其常作是語故，增上慢比丘、比丘尼、優婆塞、優婆夷號之為常不輕。」

白話：對方甚至以棍棒瓦石攻擊，菩薩退走遠處，仍大聲宣告：「我不敢輕慢你們！你們都將成佛！」正因他始終說這句話，那些傲慢的四眾給他取名「常不輕（Never-Disparaging）」。

2.4 臨終業報轉化與六根清淨

原文（鳩摩羅什譯）：

「是比丘臨欲終時，於虛空中，具聞威音王佛先所說《法華經》二十千萬億偈，悉能受持，即得如上眼根清淨，耳鼻舌身意根清淨。」

白話：臨命終時，在虛空中聽到威音王佛說法華經二十千萬億偈，得六根清淨，壽命延長，廣度眾生。昔日打罵者後來皆得法緣，轉世修行，最終成佛。

2.5 身份揭示

原文（鳩摩羅什譯）：

「汝等豈謂，彼時常不輕菩薩異人乎？則我身是。若我於前世不受持、讀誦此經、為他人說者，不能疾得阿耨多羅三藐三菩提。」

白話：釋迦牟尼佛揭示：那時的常不輕菩薩就是我自己。正因受持讀誦為人說法華經，才能迅速成就無上菩提。

三、義理解析：天台宗「五時八教」脈絡

3.1 本品的「一佛乘」實踐核心

方便品（第2品）用義理宣告了「唯有一乘法」（一佛乘），常不輕菩薩品則將這個義理轉化為行動語言：

| 方便品（義理層） | 常不輕菩薩品（實踐層） |

|-----------------|-------------------|

| 開示悟入佛之知見 | 禮敬眾生的佛性（佛知見的現前） |

| 三乘是方便，一乘是實 | 不分四眾高下，一律敬為未來佛 |

| 大白牛車（一乘）是真實 | 「汝等皆當作佛」是一乘的日常宣言 |

常不輕的「二十四字」與方便品的「一大事因緣」（開示悟入佛之知見）形成完美呼應。

3.2 天台「六即佛」框架中的位次分析

天台宗的「六即佛」（智者大師《摩訶止觀》）是理解本品修行層次的關鍵：

| 六即佛 | 說明 | 本品對應 |

|-------|------|--------|

| 理即佛 | 眾生皆具如來藏之理，但不自知 | 被打罵的四眾——具佛性但不知 |

| 名字即佛 | 已知具佛性之名，但未實修 | 常不輕的啟蒙時期 |

| 觀行即佛 | 依教修行，但六根尚未清淨 | 常不輕的主要弘法期：行禮拜宣告 |

| 相似即佛 | 入圓教十信位，六根清淨 | 常不輕臨終六根清淨後升進此位 |

| 分真即佛 | 發真智，斷無明，入初住 | 佛的授記暗示將達此位 |

| 究竟即佛 | 妙覺位，成佛 | 菩薩最終成佛 |

關鍵轉折：被迫害期間是「觀行即佛」，臨終聞法後六根清淨升入「相似即佛」，完整呈現天台觀行→相似的升進路徑。

3.3 「佛性戒」義理：從戒律到實相的飛躍

天台宗詮釋此品為「佛性戒」的示範：

  • 傳統戒律（藏通別三教）：基於條文規範，禁止做某些事
  • 佛性戒（圓教）：基於「見眾生佛性」出發的倫理，「不輕慢」的根據不是戒條，而是「此人本具佛性，必將成佛」的直觀

常不輕不輕慢，不是因為「輕慢是惡行」，而是因為「每個眾生都是未來佛」——從實相認知出發的倫理，比戒律更為根本。

3.4 「難行道」與「安樂行」的對比

法華迹門收煞（第14品）教導「四安樂行」是末法弘法的主流路線（攝護、柔和）。常不輕菩薩品展示「難行道」極端形式：直接面對迫害而不退。

兩者不矛盾：安樂行防止一般修行者受不必要迫害；常不輕展示當迫害不可避免時，以「見佛性」取代「恨迫害者」。

3.5 「威音王佛」的象徵時間學

  • 梵文 Bhīṣmagarjitasvararāja，意即「大威音聲之王」
  • 代表一切教化的本源時代（最初劫）
  • 壽量品展示橫向時間超越性（五百塵點劫），常不輕菩薩品設定在威音王佛時代，呼應縱向時間超越性——一佛乘自最初劫就已傳弘

3.6 業報轉化：逆緣成法緣

打罵常不輕的人「宿殃已盡，臨命終時，得聞是常不輕菩薩廣說，隨皆化度」。與提婆達多品（第12品）呼應：

  • 提婆達多品：惡人（提婆達多）授記成佛，逆增上緣
  • 常不輕菩薩品：迫害者最終受度，「惡緣」轉「法緣」

兩品共同宣示：法華一佛乘的普遍性連最強烈的逆緣都無法阻斷。

四、與已有研究的關聯

| 已有研究 | 與常不輕菩薩品的關係 |

|---------|-------------------|

| 方便品（第2品）— 開權顯實 | 方便品宣告「開示悟入佛知見」；常不輕用行動「開示」每個眾生的佛知見；二十四字是方便品義理的日常語言化 |

| 法師功德品（第19品）— 六根清淨 | 常不輕臨終「六根清淨」直接呼應前品；受持→修行→被打罵→聞法→六根清淨，是法師功德品功德的前傳性示範 |

| 安樂行品（第14品）— 四安樂行 | 對比：安樂行是末法常規路，常不輕是難行道極端典範 |

| 提婆達多品（第12品）— 惡人授記 | 共同主題「一佛乘的普遍性在逆境中維持」 |

| 隨喜功德品（第18品）— 一念隨喜 | 常不輕是「主動隨喜」的極端：主動向每個人宣告「你將成佛」 |

| 壽量品（第16品）— 久遠實成 | 常不輕即是本生釋迦牟尼，與壽量品久遠本生宇宙觀直接呼應 |

五、修行應用

5.1 日常「敬佛性」實踐：在每次人際互動中，嘗試先看見對方的「佛種」，而非只看「煩惱相」。

5.2 忍辱行的圓教理解：常不輕的忍辱不是壓抑憤怒，而是從「見佛性」出發的自然安住。退走但不沈默，動態堅持。

5.3 弘法者的膽量：「遠走高聲唱言」是法華弘法精神的具體符號——當真理遭排斥時，退走但不沈默。

5.4 業力觀：種法緣不計報：在當下看似無效的弘法，可能在長遠因緣中成為他人的解脫種子。

5.5 自我省察：常不輕菩薩品是一面鏡子，照出自己的傲慢——日常的判斷、言語、態度是否暗含輕慢？
