Anthropic 正式禁止將訂閱認證用於第三方用途

摘要

Anthropic 在其法律合規文件中正式宣布，禁止使用者將 Claude 的訂閱認證（subscription auth）用於第三方服務或應用程式。此舉旨在防止訂閱帳號被用於未經授權的 API 存取或商業轉售。

重點

  • 政策變更：訂閱認證僅限個人使用，不得用於第三方整合
  • 影響範圍：依賴個人訂閱進行 API 呼叫的第三方工具和服務
  • HN 社群反應：341 則評論，討論激烈，涉及開發者生態與平台政策
  • 熱度：HN 309 分，為今日 AI 相關最熱門話題
