華嚴時的義理機制深化 — 七處九會、十玄門、六相圓融與天台圓兼別判教的內在邏輯

華嚴時的義理機制深化 — 七處九會、十玄門、六相圓融與天台圓兼別判教的內在邏輯

本文為「天台五時判教系列」mechanism 階段研究，承接已有的「華嚴時擬宜說法」foundation 筆記，深入華嚴經的內在義理機制，揭示天台宗「圓兼別」判教的深層邏輯。

---

一、主題概述

華嚴時是天台宗五時判教的第一時，指釋迦牟尼佛成道後最初三七日（二十一天）在菩提樹下寂滅道場宣說《大方廣佛華嚴經》的時期。前文已論述華嚴時在五時判教中的特殊地位——最初說最高法的「擬宜」策略，以及「如聾如啞」導致後續四時的必然性。然而，要真正理解天台宗為何判華嚴為「圓兼別」而非「純圓」，必須深入華嚴經本身的義理機制。

本文進入 mechanism 階段，系統闡述華嚴經的品章結構（七處九會、三十九品）、核心義理（十玄門、六相圓融、海印森羅、法界緣起），並從天台宗的判教視角，揭示華嚴經內部「圓教」與「別教」義理的並存與張力。這不僅是對華嚴經的深入理解，更是對天台判教體系精密性的驗證——同一部經典，何以既有「初發心時便成正覺」的圓頓義理，又有「十信十住十行十迴向十地」的歷別行位？天台宗如何在承認華嚴圓滿性的同時，又指出其「兼別」的判教定位？

二、華嚴經的品章結構：七處九會與三十九品

2.1 七處九會的空間與時間結構

《大方廣佛華嚴經》八十卷本（唐譯本，實叉難陀譯）共包含七處九會、三十九品、八十卷。七處九會指佛陀在七個不同地點舉行九次法會：

| 會次 | 處所 | 品章 | 核心內容 |

|------|------|------|---------|

| 第一會 | 菩提道場（地上第一處） | 世主妙嚴品、如來現相品等（卷1-5） | 毗盧遮那佛成道、華嚴世界相 |

| 第二會 | 普光明殿（地上第二處） | 淨行品、賢首品等（卷6-7） | 十信位 |

| 第三會 | 忉利天宮（天上第一處） | 昇須彌山頂品、十住品等（卷8-12） | 十住位 |

| 第四會 | 夜摩天宮（天上第二處） | 十行品（卷13-17） | 十行位 |

| 第五會 | 兜率天宮（天上第三處） | 十迴向品（卷18-25） | 十迴向位 |

| 第六會 | 他化天宮（天上第四處） | 十地品（卷26-34） | 十地位 |

| 第七會 | 普光明殿（地上第二處重會） | 十定品、十通品等（卷35-44） | 等覺位 |

| 第八會 | 普光明殿（地上第二處再會） | 離世間品（卷45-51） | 菩薩行法 |

| 第九會 | 逝多林（地上第三處） | 入法界品（卷52-80） | 善財童子五十三參 |

2.2 三十九品的內容分類

整部華嚴經可依信、解、行、證四科判攝，或以境、行、果三門統攝：

  • 境：毗盧遮那佛所居住的重重無盡華嚴世界（第一會）
  • 行：善財童子五十三參的菩薩行（第九會）
  • 果：菩薩五十二階位中十地的修證次第（第二至八會）

此結構揭示華嚴經的內在邏輯：先展現圓滿果德（境），再詳述修行階位（果），最後以善財參學示範實踐（行）。

2.3 結構中的判教線索

從天台判教視角，七處九會的結構本身已隱含「圓兼別」的判教線索：

  1. 第一會（菩提道場）：「初發心時便成正覺」——圓教義理，一念頓悟，不歷階梯
  2. 第二至八會（十信至等覺）：五十二位次的歷別說明——別教義理，逐級而升，地地分明
  3. 第九會（入法界品）：善財一生圓滿，參學即成佛——回歸圓教，行布不礙圓融

天台宗正是基於這種結構張力，判華嚴為「圓兼別」：經中既有圓教的頓悟義理，又有別教的漸次行位。

三、華嚴宗核心義理：十玄門、六相圓融、海印森羅與法界緣起

要理解天台宗為何判華嚴為「圓兼別」，必須先掌握華嚴宗自身所建立的核心義理體系。這些概念由杜順、智儼、法藏、澄觀、宗密五祖逐步發展完善。

3.1 法界緣起：華嚴宗的本體論基礎

法界緣起是華嚴教所主張的核心理論，認為「法界乃一大緣起，宇宙萬法融通，互為緣起，重重無盡」，故亦稱「無盡緣起」。與傳統十二因緣、阿賴耶緣起不同，法界緣起強調：

  • 諸法相即相入：一切法相互融攝，一即一切，一切即一
  • 大緣起陀羅尼法：整個宇宙浸沒於相即相入的緣起關係中
  • 事事無礙：不僅理事無礙（理體與現象圓融），更進一步達到事事無礙（現象與現象亦圓融）

3.2 海印三昧：佛的證悟境界

海印三昧（海印定）是佛在說《華嚴經》時所入的三昧。華嚴宗人說，當佛入海印三昧時，過去、現在、未來一切法都同時在一心中印現，猶如深淵大海，湛然映現一切景象一般，一切法都相即相入，相攝相容，圓融無礙。

此概念揭示華嚴經的說法來源：非佛陀刻意安排次第，而是從海印三昧中自然流出，一時頓說，如大海同時映現森羅萬象（海印森羅）。

3.3 十玄門：事事無礙法界的十種關係模式

十玄門由智儼提出、法藏完善，是華嚴宗描述法界緣起關係的十種模式：

| 玄門 | 義理 | 舉例 |

|------|------|------|

| 1. 同時具足相應門 | 一法具一切法，同時成就 | 一微塵中含十方世界 |

| 2. 廣狹自在無礙門 | 廣大與微小互相無礙 | 毛孔容納須彌山 |

| 3. 一多相容不同門 | 一與多相容，但不混淆 | 一佛身即一切佛身 |

| 4. 諸法相即自在門 | 諸法相即而不失本性 | 凡夫即佛，佛即凡夫 |

| 5. 祕密隱顯俱成門 | 顯現與隱藏同時成立 | 說此法時彼法隱，彼法顯時此法隱 |

| 6. 微細相容安立門 | 極微細處仍圓滿具足 | 一念心中具三千法 |

| 7. 因陀羅網境界門 | 重重無盡，如帝釋天宮的珠網 | 珠珠相映，影影相含 |

| 8. 託事顯法生解門 | 透過具體事相顯示深法 | 善財參學每一善知識的事相 |

| 9. 十世隔法異成門 | 過去現在未來三世各三，加總別為十世，互相融攝 | 一念中現三世 |

| 10. 主伴圓明具德門 | 任一法為主時，其餘法皆為伴，主伴互轉 | 說菩提心時，萬法皆助成菩提 |

十玄門展現華嚴宗「事事無礙」的極致：不僅理事圓融，連事相與事相之間都能相即相入、互不妨礙。

3.4 六相圓融：整體與部分的辯證統一

六相圓融是智儼提出、法藏和澄觀完善的理論，說明任一事物都具備六種相，彼此圓融無礙：

| 相 | 義理 | 舉例（以房屋為例） |

|-----|------|------------------|

| 總相 | 整體的統一性 | 房屋作為一個整體 |

| 別相 | 部分的差異性 | 柱、樑、瓦、門各不相同 |

| 同相 | 部分共同組成整體 | 柱樑瓦門共同成就房屋 |

| 異相 | 部分功能不同 | 柱支撐、瓦遮雨、門出入 |

| 成相 | 部分成就整體 | 諸材料成就房屋 |

| 壞相 | 部分保持自性 | 柱仍是柱，瓦仍是瓦，不失本性 |

六相圓融揭示：總與別、同與異、成與壞同時成立，無有妨礙。這是華嚴宗解決「一多關係」的獨特方式——整體不滅部分，部分不礙整體，相即相入而各住自位。

3.5 四法界觀：從事到理的四重法界

華嚴宗判一切法為四法界：

| 法界 | 層次 | 說明 |

|------|------|------|

| 事法界 | 現象世界 | 森羅萬象，各各差別 |

| 理法界 | 本體真如 | 真如平等，無有差別 |

| 理事無礙法界 | 理事圓融 | 真如與現象互不妨礙，理即事、事即理 |

| 事事無礙法界 | 最高境界 | 現象與現象亦圓融，一即一切、一切即一 |

天台圓教達到「理事無礙」，而華嚴宗主張更進一步達到「事事無礙」——這正是兩宗判教的關鍵差異點。

四、天台宗對華嚴經的判攝：圓兼別的內在邏輯

4.1 智者大師的原初判教

智者大師在《法華玄義》中判華嚴時，認為：

「如來初成正覺，在寂滅道場，現毘盧遮那身，說圓滿修多羅。約部為頓教，但若約機、約教，則又兼權，其中『初發心時便成正覺』等文，為圓機說圓教，而『處處說行布次第』，則為權機說別教，故約部為頓，約教名兼。」

此段揭示天台判華嚴為「圓兼別」的三層理由：

  1. 約部判（整體性質）：華嚴是頓教，佛陀成道後直接宣說最高法
  2. 約機判（聽法根器）：有圓機（大菩薩）與權機（權教菩薩）並存
  3. 約教判（教法內容）：經文既有圓教義理，又有別教次第

4.2 蕅益智旭的精要闡述

蕅益大師在《教觀綱宗》中更精要地判攝華嚴時：

「華嚴最初三七日。」（五時之首）
「約部為頓，約教名兼。」（頓教兼圓別）

蕅益大師在註釋中進一步說明：華嚴經中「一地具諸地」是圓教，「地地分明、位位差別」是別教。同一部經典，何以同時具備圓別二教？關鍵在於聽法者根器不同：

  • 圓機（如文殊、普賢等大菩薩）：聞「初發心時便成正覺」即悟，不歷階級
  • 別機（權教菩薩）：仍需「十信十住十行十迴向十地」的次第引導

4.3 圓兼別的具體經文證據

從華嚴經文本身可找到圓別並存的證據：

| 義理類型 | 經文依據 | 所屬教判 |

|---------|---------|---------|

| 圓教義理 | 「初發心時便成正覺」（梵行品） | 圓教：一念頓悟，不歷階梯 |

| 圓教義理 | 「一地具諸地」（十地品） | 圓教：任一地皆具足諸地功德 |

| 圓教義理 | 「一即一切，一切即一」（華嚴經總義） | 圓教：圓融無礙 |

| 別教義理 | 「十信十住十行十迴向十地」（第二至八會） | 別教：歷別次第，地地分明 |

| 別教義理 | 「菩薩初地斷見惑，二地斷思惑……」 | 別教：階位修斷 |

4.4 天台與華嚴兩宗判教的根本差異

天台與華嚴兩宗對圓教的理解有本質差異：

| 判教面向 | 天台宗 | 華嚴宗 |

|---------|-------|-------|

| 經典依據 | 《法華經》為純圓 | 《華嚴經》為純圓 |

| 圓教標準 | 會三歸一，開權顯實 | 別教一乘，法界緣起 |

| 判教定位 | 華嚴是「圓兼別」，法華才是「純圓」 | 華嚴是「別教一乘」，超越三乘 |

| 圓融方式 | 攝末歸本（三乘歸一乘） | 攝本歸末（一乘含三乘） |

| 教法特色 | 教授上極圓（開權顯實的方法最圓） | 教法上極圓（所證果德境界最圓） |

天台宗認為：華嚴雖然境界圓滿（所證果德極圓），但教授方式仍兼帶別教次第，未能如法華「會三歸一」般徹底開權顯實。法華經明確宣告「聲聞緣覺皆當成佛」，而華嚴經雖說頓悟，但小乘人「如聾如啞」，未能普被三根。

4.5 為何華嚴不能是「純圓」？

天台宗判華嚴為「圓兼別」而非「純圓」的核心理由：

  1. 聽法機緣不齊：小乘二乘「如聾如啞」，唯大菩薩得益 → 非普被三根
  2. 經文含別教行位：十信至十地的歷別次第 → 非純圓頓
  3. 未開權顯實：華嚴未明確宣告「三乘即一乘」，仍保留三乘差別 → 非會三歸一
  4. 擬宜失敗導致後四時：若華嚴已圓滿，何需後續四時？正因擬宜未成，方有鹿苑時的生滅四諦

五、圓兼別的教觀深意：從判教到修行實踐

5.1 圓別之辨的修行意義

天台宗判華嚴為「圓兼別」，不僅是學術分類，更有深刻的修行指導意義：

  • 圓教行人：可直接從「初發心時便成正覺」入手，一念相應即頓悟法界
  • 別教行人：仍需「信住行向地」的漸次修學，不可躐等

蕅益大師強調：「雖學圓教，仍從別教心」——即使學習圓教義理，若執著於階位次第、修斷分明，實質上仍是別教心。反之，別教行人若能「雖修次第，不礙圓融」，亦能轉別成圓。

5.2 華嚴時在五時判教中的定位價值

華嚴時作為五時之首的價值：

  1. 標示最高果德：先展現佛陀所證的圓滿境界（毗盧遮那法界）
  2. 檢驗眾生根器：「如聾如啞」揭示眾生根器差異，引出後四時的教化必要性
  3. 圓教種子：雖兼別教，但圓教義理已全盤托出，為後世圓機種下頓悟種子
  4. 教化次第的起點：從頓教到漸教（阿含時），再漸次提升（方等、般若），最終回歸圓教（法華涅槃時）

5.3 現代修行的啟示

對當代學人而言，理解華嚴時「圓兼別」的判教智慧，有三重實踐意義：

  1. 不可躐等：若根器未熟，直接修學圓教（如參究「初發心時便成正覺」），易墮狂慧。別教次第仍有其修學價值。
  2. 不可執著階位：若執著「我在十信位」「我在十住位」，反成法執。圓教強調「位位不住」。
  3. 圓融漸次：最佳修學是「雖修次第，心不住次第」——外修別教行門（持戒修定），內悟圓教理體（一念三千）。
