T6-T8 動態 Fetch Agent 測試詳細報告

三個 Fetch Agent 以背景並行方式執行，模擬 Phase 1 真實場景。

T6: fetch-hackernews Agent

  • 來源: HN API（hacker-news.firebaseio.com）
  • 掃描: 30 篇熱門文章
  • 匹配: 5 篇 AI 相關（使用關鍵字篩選）
  • 快取: 寫入 cache/hackernews.json（TTL 2h）
  • AI 文章清單:

1. 「奇點將在某個星期二發生」— 1305 分、705 留言 2. 「Claude Code 正在被弱化」— 301 分、214 留言 3. 「GLM-5：從氛圍編程到 Agentic 工程」— 225 分、122 留言 4. 「GLM-OCR：精準 × 快速 × 全面」— 159 分、53 留言 5. 「AI 代理透過 REST API 玩模擬城市」— 130 分、49 留言

T7: fetch-todoist Agent

  • 來源: Todoist API（api.todoist.com）
  • Filter: today（僅當日，不含過期）
  • 結果: 7 筆今日待辦
  • 快取: 寫入 cache/todoist.json（TTL 30min）
  • 發現: Todoist REST API 已從 v2 遷移到 v1，回應格式略有不同（含 user_id、checked 等 v1 欄位）但功能正常
  • 待辦清單: 拍照、研究 Claude Code、研究 AI、研究邏輯思維、研究 GitHub、daily-digest 優化、qa_system 優化

T8: fetch-news Agent

  • 來源: 屏東新聞 MCP（ptnews-mcp.pages.dev）
  • 重試: 第 1 次即成功，未觸發重試機制
  • 快取: 寫入 cache/pingtung-news.json（TTL 6h）
  • 新聞清單（2026-02-10~11）:

1. 醫護守健康、歡喜過春節 — 春節醫療服務時段 2. 屏東有春好過年 — 「神馬都好玩」遊程推廣 3. 偏遠地區採購不再高風險 — 採購聯繫平臺成果 4. 落實在地安老 — 枋山鄉日照中心啟用 5. 太夯了!「奇幻大津」— 光影藝術活動爆紅
