專案觀測摘要

1. modelcontextprotocol/servers（86,554★）

  • 官方 MCP Server 參考實作集合，覆蓋 filesystem、database、API integrations
  • REST 即時驗證：stargazers_count=86554，pushed_at=2026-05-30T18:35:21Z
  • 與本系統相關性：高；可作為 SKILL.md skills 轉 MCP Server 的參考架構

2. VoltAgent/awesome-agent-skills（23,809★）

  • 1000+ agent skills 策展庫，相容 Claude Code、Codex、Gemini CLI、Cursor、30+ agents
  • REST 即時驗證：stargazers_count=23809，pushed_at=2026-05-27T10:38:52Z

3. microsoft/skills（2,425★）

  • AGENTS.md + SKILL.md 可攜性標準；gh skill install CLI 將 repo/ref/tree SHA 寫入 frontmatter
  • REST 即時驗證：stargazers_count=2425，pushed_at=2026-05-29T17:26:58Z
  • cross_agent_compatibility：GitHub Copilot、Claude Code、Cursor、Codex、Gemini CLI
  • 與本系統相關性：高；本系統 SKILL.md frontmatter 格式與此標準相容度高

4. modelcontextprotocol/registry（stars: REST 未核）

  • 官方 MCP registry，2025-09 preview 上線，v0.1 API freeze
  • 提供 MCP clients 策展 MCP server 列表，類似「skills 的 App Store」
  • 與本系統相關性：中；長期可考慮將高品質 SKILL.md 封裝為 MCP Server 後登錄
