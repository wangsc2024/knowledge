三、技术深度分析

3.1 架构创新：Observational Memory

Mastra 提出的核心概念：

  • 被动记忆（传统）：存储 → 检索
  • 主动观察（新）：观察 → 归纳 → 预测

三层记忆架构：

  1. 短期记忆：当前对话 context
  2. 工作记忆：跨 session 的 scratch space
  3. 观察式记忆：从历史执行中提取模式

Daily Digest 适用点：

  • context/continuity/auto-task-*.json 升级为观察式记忆
  • 新增 observations 字段：「topic X 失败率」「时段 Y API 慢」等模式

3.2 成本优化：Token 经济学三阶段

第一阶段（基础）- 20-40% 降低：

  • Prompt 压缩、限制输出

第二阶段（可扩展）- 50-73% 降低：

  • Semantic Caching：相似度 > 0.92 → 直接返回缓存
  • 多层快取：精确匹配（<1ms）→ 语义快取（<10ms）→ LLM 推理（2-8s）

第三阶段（高级）- 80-95% 降低：

  • Model Routing：简单任务 → Haiku，复杂任务 → Opus
  • Cascade 降级：Haiku 失败 → 自动升级 Sonnet 重试
