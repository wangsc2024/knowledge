Agent 架構 + Skill/Plugin 系統（第二次，聚焦 Claude Code 生態）

重點專案

1. obra/superpowers（60,700 stars）

  • Claude Code 官方 Marketplace 認證的 Agentic Skills 框架
  • 強制性工作流：TDD 紅綠重構、子 Agent 並行、系統性除錯
  • 多平台支援：Claude Code / Cursor / Codex / OpenCode
  • 內建 writing-skills 元 Skill（教導如何撰寫新 Skill）
  • Git Worktree 隔離並行開發
  • 兩階段 Review：先驗規格符合度，再驗程式品質

2. Agent-Skills-for-Context-Engineering（10,100 stars）

  • Context Engineering 學科化：將上下文管理從藝術提升為工程學科
  • 5 類 Skill：基礎/架構/營運/開發/認知
  • Context Degradation 量化：U 形注意力曲線、Lost-in-the-Middle
  • BDI（Belief-Desire-Intention）心智模型
  • 三層記憶架構：短期/長期/圖結構
  • 北京大學 2026 研究引用

3. awesome-agent-skills（227 stars）

  • SKILL.md 已成跨平台標準格式（13+ 平台支援）
  • SkillsBench 基準測試（86 任務 11 領域）
  • Progressive Disclosure 被確認為最佳實踐

改進建議（按優先級）

P0 - Context Degradation 量化監控

借鑑 Context Engineering 框架，在 post_tool_logger 追蹤 context token 累積量，在 agent_guardian 加入 context_pressure 警告。

P1 - Writing-Skills 元 Skill

借鑑 Superpowers，建立 Skill 撰寫最佳實踐指引，含 frontmatter 標準、triggers 設計原則。

P1 - 兩階段 Review

借鑑 Superpowers，將 quality-gate 拆為規格符合度（硬性）+ 品質檢查（軟性）。

P2 - 三層記憶架構

借鑑 Context Engineering，引入 Session 級短期記憶和實體關係追蹤。

P2 - 多平台 Plugin 結構

借鑑 Superpowers，評估 Skills 跨平台適配可行性。

  • SKILL.md 格式已被產業廣泛採納，本專案方向正確
  • Context Engineering 正在從實務經驗走向學術研究
