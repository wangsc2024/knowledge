How to Make a Portfolio Website with Claude Code (Award Winning)

本影片示範以 Claude Code（透過 Base44 平台）打造高轉換作品集網站。

關鍵發現

  • 優質作品集需兼顧敘事、UX、技術品質、個人風格與轉換路徑
  • Base44 在 Claude 之上處理主機、響應式、部署；比裸用 Claude Code 省技術門檻
  • 建議流程：規劃定位 → Hero 價值主張 → 互動案例展示（前後對比、分類篩選）→ 聯絡表單／估價計算器／預約日曆

適用情境：設計師、創作者需將訪客轉為付費客戶
