楞嚴經研究：湛入合湛與結解邏輯 — 從根塵黏結到妙湛圓明的修證義理（卷五末至卷六初）

楞嚴經佛學湛入合湛

主題概述

「湛入合湛」是漢傳《大佛頂首楞嚴經》在「結根／顯根性」脈絡中，祖師常用的綸貫語：一方面指出覺性本然寂湛、無動無作；另一方面說明眾生因「入」於根塵交互而現起黏結，返旋時則是「合」歸於同一湛性。本文從卷五「巾結」喻與六根結元之理起講，銜接卷六「攝心為戒」之前分，顯示結與解同一因、聖凡無二路的修證骨架，並與返聞、陰盡等後文形成承啟。

---

經文重點段落（白話大意）

1. 結解同因（六根喻後之綸要）

經文大意（通行分段）：佛示「結」若生，非離巾別有結體；「解」時亦非離巾別得解因。結與解，同依一巾——推之於根塵，縛與脫不離當下一念覺性。

白話：困境的「結」並非外來實體，而是同一覺性在迷執面上的結縛；解結也不是再找一個外在工具，而是認取結元本空、還源覺海。

2. 湛性與劇傾（根塵交輳中的迷失）

經文大意：六根對六塵，若「黏湛發知」，則於妙明中析出能所；能見與所見交織，遂有輪迴路徑。此中「湛」喻覺性本澄，「入」喻緣起會遇；不取不捨則結消，隨取隨粘則結成。

白話：感官世界本是緣起相會；一黏著「这是我的／这不是我的」故事，就像水滴入墨，湛體看似混濁，實則湛性未嘗增損。

3. 旋元與無見（通於後文圓通）

經文名句型意：後文結穴常歸「知見無見，斯即涅槃無漏真智」大意——立知立見，即落在能所二元；知見不可得而般若現前，即合於湛性。

白話：修行不是再「製造」一個更好的我來觀察世界，而是鬆開能所對立，讓本有的照了自明。

義理解析（術語簡釋）

| 術語 | 解釋 |

|------|------|

| 湛 | 澄寂、澄湛，形容覺性本無動搖、不與生滅波濤交涉的體相。 |

| 入 | 根塵相接時的「涉入／會遇」，neutral 本無過患；隨妄分别成立能所時即成結。 |

| 合湛 | 不於能所起實見實取，旋歸覺源，見與見緣並消，同一寂湛。 |

| 結元 | 結之「所依」，喻中同在一巾；義中指虛妄分別未起之本際，非實有法可得。 |

| 旋元 | 於念境界不隨流散，反觀覺性本際，與後文觀音「入流亡所」同一型態之操作語。 |

| 知見無見 | 泯絕「能知／所知」之對立後，般若自顯；非撥無知見，而是知見不著。 |

與「六解一亡」喻的接榫：一巾打六結，結數不同而体唯一；解時先解末結或先解本結，喻從麤至細皆可歸元。湛入合湛即說明：每解一結，即是還歸全巾，非六結外別有「第七神秘結」。

---

與已有知識庫研究的關聯（承上啟下）

  1. 六解一亡、華巾六結：本次把「解結」從操作喻延伸到湛性／涉入／還源的語言，補足「為何結無體而解亦無功」的體驗層說明。
  2. 耳根圓通／返聞：「合湛」在實修上對應不逐聲塵、旋流無寄；與「入流亡所」同一歸趣。
  3. 卷十五陰盡相、十二類生：陰盡是細結層層脱落；本則從根塵粗結先講清「結元本空」，避免修到後頭把境界當實法而著魔。
  4. 四種清淨明誨：戒以攝心，心若不黏即結不成；湛入合湛提供戒體的心理學基礎（非僅倫理禁令）。

---

修行應用（日常觀行）

  1. 觸境三問（簡便檢核）：此是緣起會遇否？能所故事起未？若起，還看誰在黏？
  2. 聽聞一行：聲來不拒、聲去不追；知其如空中畫，聞性不隨聲生滅。可從日常對話、通知聲、環境音練習。
  3. 見色一行：眼見色時，辨「見」與「好色／惡色分別」二層；分別鬆開，見精自朗。
  4. 與心念合作：不壓念、不隨念；念如客塵，湛如虛空——知已足，不必再加一層「我要變成很空的人」的自我戲。
