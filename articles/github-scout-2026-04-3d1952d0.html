GitHub Scout 2026-04-22：自癒 Agent 系統（self-healing / auto-remediation）

專案觀測摘要

本輪蒐集 3 個與「自癒 Agent 系統」相關的 GitHub 熱門專案：

1. matebenyovszky/healing-agent（★540）

核心模式：Python decorator 封裝函式，例外時擷取 stack trace/變數/原始碼，呼叫 LLM 診斷並自動套用修復

關鍵設計：

  • Decorator Pattern（@healing_agent 透明攔截例外）
  • Provider Abstraction（OpenAI/Azure/Anthropic/Ollama/LiteLLM 可插拔）
  • Iterative Fix Loop（Detect→Analyze→Generate→Test→Apply→Retry）
  • Fail-Safe Operations（自動備份 + JSON 記錄所有修復嘗試）

2. AdminTurnedDevOps/Agentic-AI-The-Hard-Way（★1100）

核心模式：K8s Agentic AI 工作流 Labs，含 AI Gateway 安全層

關鍵設計：

  • Human-In-The-Loop Autonomy（平台工程助手，人工監督）
  • Observability-First（Observer 場景）
  • Progressive Autonomy（從輔助到全自主）
  • AI Gateway Safety Layer（Prompt Guard/Rate Limit/MCP Security）

3. EvoAgentX/Awesome-Self-Evolving-Agents（★1400）

核心模式：自我演化 Agent 全面調查 survey（TOSEM 2026）

關鍵設計：self-healing、自我修正、自主恢復的設計模式庫
