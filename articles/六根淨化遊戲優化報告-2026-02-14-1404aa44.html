六根淨化遊戲優化報告

專案資訊

  • 遊戲名稱：六根淨化 - 禪修打字遊戲
  • 檔案位置：D:/source/game/six-roots-zen/index.html
  • 優化日期：2026-02-14

優化前分析

遊戲類型

單人禪修打字遊戲，玩家輸入中文禪語以「淨化」六根（眼耳鼻舌身意）。

原有優點

  • 禪修主題獨特，禪語內容豐富（每根 5-8 條）
  • 六根輪盤視覺化設計精美
  • 基礎回饋系統（正確/錯誤顏色變化、定力條）
  • 粒子背景增添禪意氛圍
  • 支援中文輸入法（compositionend 事件處理）

改進空間

  1. 即時回饋不足：正確輸入缺乏視覺獎勵
  2. 成就感設計缺失：無連擊獎勵、無里程碑系統
  3. 心流保持困難：無 Combo 系統增強沉浸
  4. 獎勵倍率缺失：無連擊倍率激勵

實作的優化（3 大系統）

1. Combo 連擊系統

  • 觸發條件：連續正確輸入 3 字以上
  • 視覺呈現：右上角顯示連擊數與倍率
  • 倍率計算：每 5 連擊 +0.1 倍率（5 連擊 ×1.1、10 連擊 ×1.2）
  • 脈衝動畫：每次連擊數字放大 1.3 倍後回彈
  • 技術實作：updateCombo() 函數控制顯示/隱藏與動畫

2. 成就系統（8 項成就）

成就圖示解鎖條件
初心🌱完成首次淨化
專注🎯達成 10 連擊
入定🧘達成 25 連擊
禪定✨達成 50 連擊
定力圓滿🔥定力達到 100%
六根清淨☯️完成一輪淨化
修行精進🏆完成三輪淨化
無瑕💎準確率保持 100%（至少 20 字）
  • 彈窗動畫：成就解鎖時螢幕中央彈出金色徽章
  • 技術實作：checkAchievements() + showAchievement() 函數

3. 粒子爆發特效

  • 觸發時機：每次正確輸入字符
  • 粒子數量：6 顆
  • 顏色池：金色、橙色、綠色、米色（契合禪意配色）
  • 動畫效果：從字符位置向外輻射，逐漸縮小消失
  • 技術實作：spawnCorrectParticles(x, y) 動態建立 DOM 元素

4. 定力條強化

  • 滿定力特效：100% 定力時定力條發出綠色脈衝光暈
  • 連擊加成：連擊越高，定力增加越快（基礎 5 + 連擊數/10）

設計理論依據

根據心流理論（Mihály Csíkszentmihályi）與核心循環設計原則：

  1. 即時回饋強化：粒子爆發讓每個動作都有視覺獎勵
  2. 挑戰與技能平衡：成就系統設置階梯式目標
  3. 清楚目標：Combo 數字與成就進度可視化
  4. 動作與意識合併：連續正確輸入進入心流狀態

技術細節

新增 CSS 類別

  • .combo-display / .combo-count / .combo-multiplier
  • .achievement-popup / @keyframes achievementPop
  • .correct-particle / @keyframes particleBurst
  • .focus-bar-fill.maxed / @keyframes focusGlow

新增 JavaScript 函數

  • updateCombo() - Combo 顯示更新
  • checkAchievements() - 成就條件檢查
  • showAchievement(ach) - 成就彈窗顯示
  • spawnCorrectParticles(x, y) - 粒子爆發生成

修改的函數

  • handleInput(e) - 整合 Combo、粒子、成就檢查
  • compositionend 事件處理 - 同步新功能至中文輸入
  • updateFocusBar() - 加入滿定力特效
  • purifyRoot() / completeRound() - 加入成就檢查
