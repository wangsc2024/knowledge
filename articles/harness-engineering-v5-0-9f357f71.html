一、風險評估與緩解策略

風險矩陣

| 風險 | 機率 | 影響 | 緩解策略 | 負責人 |

|------|------|------|---------|--------|

| R1：Rubric 品質不足 | 中 | 高 | 參考 Harvey 範例，逐步迭代，建立 review 機制 | P0 |

| R2：Autoresearch 過度優化 | 低 | 中 | 設定收斂條件（最多 40 輪），人工審查高影響變更 | P1 |

| R3：ML 排名模型過擬合 | 中 | 中 | 保留 20% hold-out set，監控 A/B 測試指標 | P1 |

| R4：Context 排名降低品質 | 低 | 高 | 保留 fallback 到原始 RAG，監控用戶反饋 | P0 |

| R5：實施工作量超出預期 | 高 | 中 | 分階段部署，優先 P0 快速勝利項目 | P0 |

| R6：團隊技能缺口 | 中 | 中 | 提供訓練材料，從簡單 Rubric 開始 | P1 |

關鍵決策點（Go/No-Go）

Phase A 結束時（2 週）：

  • 至少 1 個高品質 Rubric 完成並驗證
  • Embedded Rubrics 在 3 個模板中運作
  • 用戶行為追蹤收集到至少 100 筆資料
  • Go 條件：3/3 達成 → 繼續 Phase B；否則暫停並調整

Phase B 結束時（6 週）：

  • Autoresearch Engine 自動執行無錯誤
  • ML 排名模型 AUC ≥ 0.75
  • Diagnostic Filesystem 保留至少 30 天歷史
  • Go 條件：2/3 達成 → 繼續 Phase C；否則延長 Phase B

二、Rubric 範例模板

Tech Research Rubric（技術研究任務評分標準）

完整 YAML 定義包含：

  • 5 個維度：來源品質（25%）、三角佐證（30%）、實用性（20%）、完整性（15%）、結構品質（10%）
  • 評分標準：每個維度 1-5 分，明確條件
  • 品質門檻：8.0（優秀）、7.0（良好）、6.0（及格）
  • 嵌入式反饋模板：包含分數、建議、預期改進

使用範例：


# tools/rubric_scorer.py
def score_tech_research(note_id: str) -> dict:
    # 自動評分（基於規則 + LLM 輔助）
    final_score = (
        source_quality * 0.25 +
        triangulation * 0.30 +
        actionability * 0.20 +
        completeness * 0.15 +
        structure * 0.10
    )
    return {"final_score": final_score, ...}

三、實施檢查清單

Phase A：快速勝利（2 週）

#### Week 1：Rubric 建立

  • Day 1-2：閱讀 Harvey 論文 Rubric 範例

驗收標準：Rubric 平均分與人工評分相關係數 ≥ 0.75

#### Week 2：Embedded Rubrics + 用戶追蹤

六、Harvey 實施時間軸參考

  • Week 1-2: 建立 12 個任務的 Rubrics
  • Week 3-4: Autoresearch Engine
  • Week 5-8: 執行 8 輪 Autoresearch（每輪 3-5 天）
  • Week 9-10: 驗證與穩定化
  • 結果: 40.8% → 87.7%（10 週）

關鍵成功因素

  1. 高品質 Rubrics（5-10 維度，可量化）
  2. 自動化優先（人工審查僅限 P0）
  3. 快速迭代（每輪 ≤5 天）
  4. 持續監控（每輪追蹤指標）

七、FAQ

Q1：Rubric 評分會增加執行時間嗎？

A：單次 2-5 秒（LLM 輔助），非同步執行不影響主流程。

Q2：ML 排名需要多少訓練資料？

A：最少 500 筆，建議 1000+ 筆（Phase A/B 期間收集）。

Q3：Meta-Harness 會產生不安全的程式碼嗎？

A：所有候選經人工審查 + 自動掃描（Bandit/Semgrep）。

Q4：Rubric 與人工評分不一致怎麼辦？

A：每月校準一次權重，目標相關係數 ≥ 0.80。

Q5：Context 排名會遺漏重要資訊嗎？

A：保留 fallback，若 top-5 分數 < threshold，自動擴展到 top-10。
