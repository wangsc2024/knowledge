關鍵發現

Claude Skills 是可重複使用的 prompt 檔案，透過 /skill名稱 觸發或由 Claude 自動識別上下文載入。建立流程：Customize > Create with Claude > 利用 Skill Creator 元技能引導式問答 > 測試真實範例 > 安裝。Skill 由 frontmatter（描述與觸發方式）+ body（實際 prompt）組成，複雜版可內嵌 Python 程式碼。重要觀點：好的 Skill 需具備三要素 — 具體、基於真實範例、經時間考驗。Skill prompt 不會預載全部對話，僅觸發時才載入，避免 Claude 混淆。可在 Chat、Cowork、Claude Code 三種模式跨平台使用。

適用情境

設計 Claude Skills 工作流、教學團隊成員建立自訂 Skill、優化現有 Skill prompt 結構。
