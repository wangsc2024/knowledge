- 2026-02-18 23:30 Claude API Internal Server Error - 根因：Anthropic API 端暫時性錯誤，無法本地修正

🟡 改善建議

  1. Gmail Agent OOM Crash

- 2026-02-18 08:00 Gmail agent 遇到 V8 Fatal process out of memory - 影響：該次每日摘要 Gmail 區塊失敗 - 建議：監控 Gmail agent 記憶體使用

- 根因：timeout 中斷的 Job 產生空 stderr 檔案，清理邏輯在 Job 內部無法執行 - 修正：check-health.ps1 新增空 stderr 清理
