GitHub Scout 研究報告：Agent 架構（2026-06-24）

專案觀測摘要

1. openai/openai-agents-python（27,383★）

  • 網址：https://github.com/openai/openai-agents-python
  • runtime_model：handoff-based 多代理委派；事件驅動；每個 Agent 是獨立的指令+工具+防護欄配置單元
  • state_strategy：Sessions 自動對話記錄 + 追蹤機制；支援 Redis 可選後端；支援長期 workflow 狀態維護
  • human_review_model：handoff-based — explicit handoff 轉移控制權，HITL 透過 approval callback 實現；適合多代理委派審查
  • 最新 push：2026-06-24（REST 即時觀測）

2. microsoft/agent-framework（11,603★）

  • 網址：https://github.com/microsoft/agent-framework
  • runtime_model：圖式工作流（sequential/concurrent/handoff/collaboration 四模式）；YAML 宣告式 Agent 定義
  • state_strategy：內建 checkpointing + time-travel；工作流可重啟；DevUI 互動開發
  • human_review_model：interrupt-point 模式 — 圖執行暫停等人工 approve/edit；OpenTelemetry 可觀測性整合
  • 最新 push：2026-06-24（REST 即時觀測）

3. langchain-ai/langgraph（35,590★）

  • 網址：https://github.com/langchain-ai/langgraph
  • runtime_model：圖式編排（StateGraph）；stateful multi-actor；支援事件驅動循環與中斷點
  • state_strategy：checkpoint 持久化（SQLite/Redis/Postgres）；支援 time-travel 重播；thread/session 隔離
  • human_review_model：interrupt-point 模式 — interrupt() API，遇 breakpoint 暫停等待 approve/edit；同步短延遲審核
  • 最新 push：2026-06-24（REST 即時觀測）
