已實作的留存機制（最佳實踐）

連續天數追蹤 (Streak)


function updateStreakOnComplete() {
    const today = getDateString();
    const lastPlay = progress.lastPlayDate;
    if (lastPlay === today) return; // 今天已玩過
    const daysDiff = getDaysDiff(lastPlay, today);
    if (daysDiff === 1) {
        progress.streak++; // 連續 +1
    } else {
        progress.streak = 1; // 中斷重新計算
    }
    progress.lastPlayDate = today;
}

原理：根據《原子習慣》，連續性是習慣形成的關鍵。視覺化的連續天數讓玩家「不想中斷」。

建議優化（提升留存率）

1. 難度分級

目前 mindful-memory 只有 4x4 固定難度。建議增加：

  • 簡單：3x4（6 對）- 新手友好
  • 標準：4x4（8 對）- 現有難度
  • 挑戰：4x5（10 對）- 進階玩家

2. 每日挑戰


function getDailyChallenge() {
    const seed = new Date().toDateString();
    const challenges = [
        { goal: 'moves', target: 20, desc: '用 20 次以內翻牌完成' },
        { goal: 'time', target: 45, desc: '45 秒內完成' },
        { goal: 'streak', target: 3, desc: '連續完成 3 局' }
    ];
    return challenges[hashString(seed) % challenges.length];
}

效果：提供每日回訪理由，強化習慣迴路。

3. 連續天數視覺化

用火焰 🔥 數量表示連續強度：


.streak-fire::before {
    content: '🔥';
    animation: flicker 1s infinite;
}
  • 1-3 天：1 個 🔥
  • 4-7 天：2 個 🔥🔥
  • 8-14 天：3 個 🔥🔥🔥
  • 15+ 天：4 個 🔥🔥🔥🔥 + 特效

4. 音效回饋

使用 Web Audio API 產生簡單音效：


const audioCtx = new AudioContext();
function playTone(freq, duration) {
    const osc = audioCtx.createOscillator();
    osc.frequency.value = freq;
    osc.connect(audioCtx.destination);
    osc.start();
    osc.stop(audioCtx.currentTime + duration);
}

結論

正念遊戲的核心是持續練習，因此留存機制比一般遊戲更重要。結合《原子習慣》的四大法則：

  1. 讓它顯而易見：連續天數、最佳紀錄顯示在首頁
  2. 讓它有吸引力：成就解鎖、視覺回饋
  3. 讓它簡單容易：難度分級、快速開始
  4. 讓它令人滿足：即時音效、完成動畫、統計成長
