楞嚴經研究：七趣因果 — 卷八至卷九六道升沉與仙趣因果的完整圖譜

楞嚴經佛學七趣因果十習因六交報情想升沉地獄餓鬼畜生仙趣天道阿修羅輪迴因果

楞嚴經研究：七趣因果 — 卷八至卷九六道升沉與仙趣因果的完整圖譜

主題概述

《大佛頂首楞嚴經》卷八至卷九的「七趣因果」，是佛陀應阿難之請，開示眾生輪迴六道的根本因果機制。楞嚴經的獨特貢獻在於：在傳統佛教「六道」（地獄、餓鬼、畜生、人、天、阿修羅）之外，另立「仙趣」為獨立一趣，形成「七趣」的完整分類體系。此段經文以「情」與「想」的比例為核心判準，層層展開十習因、六交報、十種鬼、十種畜生、十種人、十種仙、欲界六天、色界十八天、無色界四天、四種阿修羅等，構成佛教中最系統、最詳盡的輪迴因果圖譜之一。

這段教法的修行意義極為深遠：不僅讓修行者認清輪迴的全貌，更揭示一切趣的「虛妄性」——「此七趣者，皆是昏沉諸有為相，妄想受生，妄想隨業」——從而堅定出離心與菩提心。

---

一、情想升沉 — 七趣的核心判準

1.1 經文重點

楞嚴經卷八開示七趣因果之前，先以「情」與「想」的比例，總括一切眾生升沉的根本機制：

「純想即飛，必生天上；若飛心中兼福兼慧及與淨願，自然心開，見十方佛，一切淨土，隨願往生。情少想多，輕舉非遠，即為飛仙……純情即沈，入阿鼻獄。」

1.2 情想比例與七趣對應

| 情想比例 | 所生趣道 | 特徵 |

|----------|----------|------|

| 純想（十分想） | 天道（乃至淨土） | 純粹的清淨意願，心開見佛 |

| 九想一情 | 天道上品 | 微有情執，仍在天界 |

| 八想二情 | 天道 | 偏多清淨想 |

| 七想三情 | 飛仙 | 輕舉非遠，為飛行仙等 |

| 六想四情 | 鬼道上品或仙趣下品 | 情想交雜 |

| 五情五想 | 人道 | 情想均等，為人中正報 |

| 六情四想 | 畜生道 | 情稍偏重，墮入旁生 |

| 七情三想 | 餓鬼道 | 情重想輕，常被飢渴 |

| 八情二想 | 地獄（有間） | 情極重而想微弱 |

| 九情一想 | 無間地獄 | 幾近純情 |

| 純情（十分情） | 阿鼻地獄 | 完全被愛染覆蓋 |

1.3 義理解析

  • 「情」：由諸愛染發起的妄情。情積不休，能生愛水，故偏重者往下沉墮。「情」屬陰氣，性質沉重。
  • 「想」：由渴仰修行發明的虛想。想積不休，能生勝氣，故偏多者往上飛升。「想」屬陽氣，性質輕舉。
  • 此處的「想」非普通的妄想分別，而是指向善的、清淨的、出離的思維；「情」則是指向貪愛執著的、染污的心理活動。
  • 大安法師強調：這個「情想」判準揭示了念佛法門的深層原理——念佛即是以清淨之「想」（憶佛念佛）取代染污之「情」（五欲六塵），從而「純想即飛，必生天上……兼福兼慧及與淨願，自然心開，見十方佛」。

二、十習因與六交報 — 地獄趣的因果體系

2.1 十習因

卷八詳述招感地獄果報的十種惡習（習因），每一種習因都對應特定的地獄境界：

| 序號 | 習因 | 核心經文 | 對應地獄果報 |

|------|------|----------|-------------|

| 1 | 淫習 | 「淫習交接，發於相磨」 | 鐵床銅柱地獄（二習相然，故有猛火） |

| 2 | 貪習 | 「貪習交計，發於相吸」 | 寒冰地獄（吒吒、波波、羅羅、青赤白蓮） |

| 3 | 慢習 | 「慢習交陵，發於相恃」 | 血河灰河、熱砂毒海、融銅灌吞 |

| 4 | 瞋習 | 「瞋習交衝，發於相忤」 | 刀山劍樹、鐵鑊湯煎 |

| 5 | 詐習 | 「詐習交誘，發於相調」 | 繩木絞校、柙壓槌擊 |

| 6 | 誑習 | 「誑習交欺，發於相罔」 | 塵土屎尿、飛血刺骨、不淨等事 |

| 7 | 怨習 | 「怨習交嫌，發於銜恨」 | 投擲擒捉、擊射拋撮 |

| 8 | 見習 | 「見習交明，如薩迦耶見、邊見等」 | 王吏證執、勘問審明 |

| 9 | 枉習 | 「枉習交加，發於誣謗」 | 枷鎖鞭杖、拘禁鞭撻 |

| 10 | 訟習 | 「訟習交諠，發於藏覆」 | 鑑見照燭、業鏡火珠 |

2.2 六交報

十習因成就後，六根各各受報，稱為「六交報」。地獄眾生因六根（眼、耳、鼻、舌、身、意）造作的不同業行，在地獄中受不同的感官折磨。經文描述極為細緻：見報（看到地獄慘景）、聞報（聽到地獄苦聲）、嗅報（聞到地獄腥穢）、味報（嘗到地獄苦味）、觸報（感受地獄苦觸）、思報（心受地獄煎熬）。

2.3 關鍵義理

  • 十習因不是外在的「神」在懲罰，而是「自業自受」——「惡業如是，從其心生……其同業地獄，不是上帝造，不是閻王造，乃是自己的身口意三業所感」。
  • 宣化上人強調：「阿賴耶識把你所做的業，都給你存好了，等到因緣成熟，業報就現前。」地獄不是別處，正是自心妄想所變現。

三、七趣各論

3.1 地獄趣

地獄為七趣之首（最苦），經文先述十習因、六交報，再描述地獄的多種形態：

  • 阿鼻地獄（無間地獄）：純情無想者所入，受苦無間斷
  • 有間地獄：八情二想、九情一想者所入
  • 十方地獄：由十習因各各所感

3.2 餓鬼趣

地獄刑期圓滿後，罪業未盡者轉生餓鬼道。經文描述十種鬼，對應十習因：

  • 淫習 → 怪鬼（魅鬼）
  • 貪習 → 餓鬼
  • 慢習 → 魃鬼（大力鬼）
  • 瞋習 → 厲鬼（疫鬼）
  • 詐習 → 役使鬼
  • 誑習 → 不淨鬼
  • 怨習 → 魘鬼
  • 見習 → 疫癘鬼
  • 枉習 → 傳送鬼
  • 訟習 → 王魅鬼

3.3 畜生趣

餓鬼報盡，轉生畜生還債。十種畜生同樣對應十習因，形象地說明業力如何牽引投生：

  • 淫習 → 梟類（不孝之鳥）
  • 貪習 → 毒蟲（蝎蛇蜂蠆）
  • 慢習 → 蛔蟲（寄生蟲）
  • 瞋習 → 猛獸（虎狼等）
  • 詐習 → 狐類
  • 其餘類推

3.4 人趣

畜生報盡還生為人，但帶有前世餘業的特質。十種人對應十習因的殘餘：

  • 從梟類（淫習）來者 → 為人諂曲不正
  • 從毒蟲（貪習）來者 → 為人剛強難化
  • 以此類推，每種人都帶有前生習氣的痕跡

人道的特殊地位在於「五情五想」——情與想均等，既有向上之力也有向下之因，是修行的最佳起點。

3.5 仙趣（楞嚴經獨特貢獻）

十種仙是楞嚴經最獨特的分類，傳統佛教不列仙為獨立一趣。經文描述十種仙人，皆由人道中「堅固妄想」修煉而成：

| 序號 | 仙名 | 修煉方法 | 特徵 |

|------|------|----------|------|

| 1 | 地行仙 | 堅固服餌（藥物） | 行步輕捷，壽千萬歲 |

| 2 | 飛行仙 | 堅固草木（草藥） | 身輕如煙，駕雲騰霧 |

| 3 | 遊行仙 | 堅固金石（煉丹） | 煉金石為丹，遊行自在 |

| 4 | 空行仙 | 堅固動止（吐納導引） | 精化為氣，氣化為神，遊行虛空 |

| 5 | 天行仙 | 堅固津液（咽津服氣） | 容貌返童，上行天際 |

| 6 | 通行仙 | 吸收日月精華 | 採日月精華，神通初現 |

| 7 | 道行仙 | 堅固咒禁（持咒守戒） | 持咒修法，得道法力 |

| 8 | 照行仙 | 堅固思念（存想觀照） | 觀想金光，身發光明 |

| 9 | 精行仙 | 堅固交遘（內丹陰陽） | 體內精氣交會，內丹成就 |

| 10 | 絕行仙 | 堅固變化（窮盡造化） | 移山倒海，變化不測 |

仙趣的關鍵限制：

「是等皆於人中煉心，不修正覺，別得生理，壽千萬歲，休止深山或大海島，絕於人境。斯亦輪迴妄想流轉，不修三昧，報盡還來，散入諸趣。」

仙人雖壽命極長，但仍在輪迴之中，因為其修煉基於「妄想」而非「正覺」。他們追求的是身體的長生而非心性的解脫，待報盡之後仍要散入六趣輪迴。這是楞嚴經對道教仙術的嚴肅批評——長生不等於解脫。

3.6 天趣

天趣分三界：

欲界六天：

  1. 四天王天 — 未全斷淫，但能遵守戒律
  2. 忉利天（三十三天） — 淫愛薄少
  3. 須焰摩天 — 逢時暫交，去無思憶
  4. 兜率陀天 — 一切時靜，有應觸來未能違戾
  5. 化樂天 — 無世間心，同世行事，於行事交了然超越
  6. 他化自在天 — 超越色觸，以無為心應有為事

色界十八天（四禪各含數天）和無色界四天（空無邊處、識無邊處、無所有處、非想非非想處），各以禪定深淺為升進次第。

3.7 阿修羅趣

卷九最後述阿修羅（非天），分四種：

  1. 卵生阿修羅：鬼趣所攝，以護法力成為鬼王
  2. 胎生阿修羅：人趣所攝，嗔心猛烈而有福德
  3. 化生阿修羅：天趣所攝，好鬥爭勝的天人
  4. 濕生阿修羅：畜生趣所攝

修羅的共同特徵是「有天福而無天德」——具福報但缺德行，好勝嗔恚。

四、與已有研究的關聯

承上

  1. 四種清淨明誨（卷六~七）→ 本研究：四明誨是修行的根本戒律（不淫、不殺、不盜、不妄），七趣因果正是說明違犯這四戒的果報。十習因中的淫、貪、瞋、詐、誑等正對應四根本戒的違犯。
  2. 十二類生（卷七）→ 本研究：十二類生從「眾生分類」角度說明，七趣因果從「因果業報」角度說明，二者互為表裡。
  3. 三種相續（卷四）→ 本研究：世界相續、眾生相續、業果相續的宇宙觀，在七趣因果中得到具體展開——業果相續正是七趣輪轉的驅動力。

啟下

  1. 本研究 → 五十陰魔（卷九~十）：七趣因果說明墮入惡趣的業因，五十陰魔說明修行過程中可能遭遇的魔境。修行者須先認清七趣的虛妄性，才能在面對陰魔時不被迷惑。
  2. 本研究 → 五十五位真菩提路（卷八）：七趣是迷界的全景，五十五位是悟界的地圖。二者形成鮮明對比——迷則七趣流轉，悟則菩提路上。

五、修行應用

5.1 以因果為鑑，堅固出離心

七趣因果的首要修行意義在於「怖畏輪迴」。了知十習因的可怕果報，自然對惡業生起防護之心。如淨界法師所言：「菩薩畏因，眾生畏果。」修行者應在「因」上斷除，而非等到受報時才後悔。

5.2 情想判準的日常覺察

在日常生活中觀察自己的心念：

  • 當心念傾向貪愛執著（情）→ 警覺這是向下沉墮的力量
  • 當心念傾向清淨善法（想）→ 鼓勵這是向上提升的力量
  • 修行的本質就是不斷增加「想」的比例，減少「情」的比例

5.3 念佛與情想轉化

大安法師特別指出：念佛法門正是最直接的「轉情為想」之法。以佛號取代妄念，以淨土願取代世間情執，「若飛心中兼福兼慧及與淨願，自然心開」——這正是楞嚴經為淨土法門提供的義理支撐。

5.4 仙趣的警示

修行者應當辨別「長生」與「解脫」的本質區別。追求肉身不死是「不修正覺，別得生理」，仍在輪迴之中。唯有以不生不滅的真心為本修因，才能真正超越七趣。

5.5 七趣皆妄的空觀

經文最後總結：

「是七趣者，皆是昏沉諸有為相，妄想受生，妄想隨業。於妙圓明無作本心，皆如空華，元無所著。」

七趣雖有苦樂差別，但從真心本性來看，皆是「空華」——如虛空中的花朵，看似存在實無自性。這是楞嚴經「破妄顯真」思想在因果論上的極致展現：既要敬畏因果（事相上不可不慎），又要了知因果本空（理體上不可執著）。

六、佛學術語解釋

| 術語 | 解釋 |

|------|------|

| 七趣 | 楞嚴經特有分類：地獄、餓鬼、畜生、人、仙、天、阿修羅。傳統佛教為六道（無仙趣） |

| 情 | 愛染妄情，由貪愛執著生起，性質沉重，使眾生下墮 |

| 想 | 清淨善念，由渴仰修行生起，性質輕舉，使眾生上升 |

| 十習因 | 招感地獄果報的十種惡習：淫、貪、慢、瞋、詐、誑、怨、見、枉、訟 |

| 六交報 | 六根各自受報，眼見苦境、耳聞苦聲、鼻嗅穢氣等 |

| 阿鼻地獄 | 又名無間地獄，五無間：趣果無間、受苦無間、時無間、命無間、形無間 |

| 仙趣 | 人道中堅固妄想修煉者，壽雖千萬歲，報盡仍入輪迴 |

| 阿修羅 | 有天之福無天之德，好勝嗔恚，分胎卵濕化四生 |

| 空華 | 眼病所見虛空中之花朵，喻一切有為法無實自性 |
