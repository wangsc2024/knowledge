Space Invaders v2.4 成就解鎖系統實作心得

概述

為 HTML5 Canvas 遊戲 Space Invaders 加入 12 個成就解鎖系統，涵蓋射擊精準、連擊、UFO 擊殺、道具收集、護盾觸發、關卡進度與累計分數等多面向目標。

架構設計

成就定義結構

每個成就包含 id、name、desc、icon、check 函式。兩種觸發模式：統計型（定期檢查 stats 物件）和事件型（在程式碼特定位置直接呼叫 tryUnlock）。

通知佇列

使用 FIFO 佇列管理多個同時解鎖，每次只顯示一個，3 秒動畫（滑入、停留、淡出）。解鎖時噴射金色粒子（複用既有粒子池，零額外記憶體）。

持久化

localStorage 存 { [id]: { date } } 格式，每次解鎖立即寫入，跨 session 保留。

12 個成就

| 成就 | 類型 | 條件 |

|------|------|------|

| FIRST BLOOD | 統計型 | totalKills >= 1 |

| SHARPSHOOTER | 事件型 | 連續命中 10 次 |

| COMBO MASTER | 事件型 | x4 連擊倍率 |

| COMBO KING | 事件型 | x8 連擊倍率 |

| UFO HUNTER | 統計型 | ufoKills >= 1 |

| UFO ACE | 統計型 | 單場 3 架 UFO |

| SHIELD SAVE | 事件型 | 護盾擋子彈 |

| COLLECTOR | 統計型 | 收集 10 道具 |

| SURVIVOR | 事件型 | 3 命過第 1 關 |

| VETERAN | 統計型 | 到第 5 關 |

| LEGEND | 統計型 | 到第 10 關 |

| 10K CLUB | 統計型 | 分數 >= 10000 |

設計模式

  • 物件池複用：粒子效果零額外開銷
  • 觀察者模式變體：事件型在業務邏輯直接觸發
  • 單一責任：tryUnlock 負責去重 + 持久化 + 通知佇列
