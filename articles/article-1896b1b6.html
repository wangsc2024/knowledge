法華經藥王菩薩本事品研究：燃身供養與法華十喻的極致信願

法華經佛學天台宗藥王菩薩本事品燃身供養一切眾生喜見菩薩法華十喻現一切色身三昧真法供養一佛乘

法華經藥王菩薩本事品研究：燃身供養與法華十喻的極致信願

主題概述

藥王菩薩本事品為《妙法蓮華經》第二十三品，位於卷六，是法華經流通分中最具震撼力的一品。此品以「宿王華菩薩」的提問為緣起，釋迦牟尼佛述說藥王菩薩前世為「一切眾生喜見菩薩」（梵：Sarvasattvapriyadarśana）時，於日月淨明德佛座下聞法華經、證「現一切色身三昧」後，以燃身燃臂供養佛法的壯烈行持。此品有三大主題：一是以菩薩的極致苦行證明法華經的無上殊勝；二是以「法華十喻」建立法華經在一切經典中的至尊地位；三是以持經功德開示勉勵末法眾生受持弘傳。天台宗判此品為本門流通分的重要環節，承接如來神力品的「結要付囑」與囑累品的「摩頂總付」，進入具體的弘經行持示範。

---

一、一切眾生喜見菩薩的修行因緣

1.1 日月淨明德佛的國土

經文（鳩摩羅什譯）：

「過去無量恒河沙劫，有佛號日月淨明德如來……其佛國土，地平如掌，琉璃所成，寶樹莊嚴，寶帳覆上，垂寶華旛。」

白話翻譯：在過去無量恒河沙數的漫長劫數之前，有一尊佛名為日月淨明德如來。他的佛國大地平坦如手掌，由琉璃構成，珍寶樹木整齊排列，寶帳覆蓋上方，垂掛著華麗的旗旛。

此國土的莊嚴描述，象徵聞法者心地平坦、清淨無染時所感召的依報環境。日月淨明德佛的名號本身即含深意——「日月」表智慧照破無明，「淨明德」表清淨光明的功德圓滿。

1.2 證得「現一切色身三昧」

一切眾生喜見菩薩在日月淨明德佛座下精進修行，歷經一萬二千年，證得「現一切色身三昧」——能變現一切身相教化眾生的甚深禪定。此三昧是法華經修行的具體果報，與普門品觀世音菩薩的三十三應身遙相呼應。

1.3 第一次燃身供養（1200 年）

經文：

「即服諸香，栴檀、薰陸、兜樓婆……滿千二百歲已……以神通力願而自燃身。光明遍照八十億恒河沙世界。」

白話翻譯：菩薩服食各種香料——栴檀、薰陸、兜樓婆等——滿一千二百年後，以神通力發願自燃其身，光明遍照八十億恒河沙數的世界。

諸佛讚歎：

「善男子，是真精進，是名真法供養如來……假使國城妻子布施，亦所不及。」

十方諸佛同聲讚歎此為「真精進」、「真法供養」，其功德超越以七寶、國城、妻子等一切財物的布施。此處的核心教義在於：法供養（以法修行、以身證法）遠勝於財供養。

1.4 第二次燃臂供養（72,000 年）

日月淨明德佛入滅後，一切眾生喜見菩薩在八萬四千塔前「燃其兩臂，供養七萬二千歲」，以此引導無量眾生發阿耨多羅三藐三菩提心，令其安住於「現一切色身三昧」。

燃臂後，其臂自然還復如故。此「捨身復得身」的敘事，在義理上揭示了大乘佛教「空性不礙妙有」的核心思想——真正的捨離不是失去，而是在空性中獲得更圓滿的存在。

二、法華十喻 — 經中之王的譬喻論證

此品以十個譬喻建立法華經在一切經典中的至尊地位：

經文：

「譬如一切川流江河諸水之中，海為第一。此法華經亦復如是，於諸如來所說經中，最為深大。」

| 序號 | 譬喻 | 法華經的對應 |

|------|------|-------------|

| 1 | 一切川流江河中，海為第一 | 於一切經中最為深大 |

| 2 | 一切山中，須彌山為第一 | 於一切經中最為其上 |

| 3 | 一切眾星之中，月天子最為第一 | 於千萬億種諸法中最為照明 |

| 4 | 日天子能除諸闇 | 此經亦復如是，能破一切不善之闇 |

| 5 | 諸小王中，轉輪聖王最為第一 | 於眾經中最為其尊 |

| 6 | 帝釋為三十三天之主 | 此經亦為眾經之王 |

| 7 | 大梵天王為一切眾生之父 | 此經亦為一切賢聖之父 |

| 8 | 一切凡夫中，須陀洹等四道果為第一 | 於一切凡夫所說善法中最為第一 |

| 9 | 聲聞、辟支佛中，菩薩為第一 | 於一切諸經法中最為第一 |

| 10 | 佛為法中之王 | 此經亦復如是，於眾經中最為第一 |

此十喻從水到山、從星辰到王者、從凡夫到佛陀，層層遞進，構成一個完整的「經中之王」論證體系。天台宗以此為法華經判教至尊的重要經證。

三、義理解析 — 天台「五時八教」脈絡下的藥王菩薩品

3.1 本品在天台判教中的位置

天台宗將法華經判為第五時「法華涅槃時」的圓教至經。藥王菩薩本事品屬於「流通分」，在天台架構中的功能是：

  • 迹門流通（前十四品為迹門正宗，法師品起為迹門流通）
  • 本門流通（壽量品起為本門正宗，如來神力品起為本門流通）

藥王菩薩品位於本門流通分中，其角色是以身教示範具體展示如何以生命護持弘傳法華經。

3.2 「真法供養」與華嚴「普賢行願品」的呼應

《華嚴經・普賢行願品》十大願王中，第三願「廣修供養」明言「諸供養中，法供養最」。藥王菩薩品以一則完整的本生故事，將此教義具象化——菩薩先以最極端的身體供養（燃身燃臂）作為鋪墊，佛陀再明確指出「聞是經一偈一句，乃至一念隨喜者」的功德猶勝於此。

此結構揭示法華經的核心轉化邏輯：先破「法供養低於身供養」的凡夫執著，再立「受持法華即是最上供養」的究竟真理。

3.3 「現一切色身三昧」與普門品的關係

一切眾生喜見菩薩所證的「現一切色身三昧」，與觀世音菩薩在普門品中示現三十三應身的神力，在修證層次上屬於同一境界。兩品形成法華經流通分的雙重示範：

  • 藥王菩薩品：以自利行（燃身供法）展示極致信願
  • 普門品：以利他行（隨類應現）展示普度慈悲

自利與利他圓融無礙，正是天台圓教「一心三觀」的修行實踐。

四、與已有研究的關聯

4.1 研究脈絡

本研究是法華經系列研究的第十八篇，與前十七篇共同構成完整的法華經義理體系：

概論 → 方便品（開權顯實 + 十如是） → 譬喻品（三車火宅） → 信解品 → 藥草喻品 → 化城喻品 → 授記品 → 法師品（三軌弘經） → 提婆達多品（女身成佛） → 見寶塔品 → 壽量品（開近顯遠） → 安樂行品 → 常不輕菩薩品 → 如來神力品 → 普門品 → 一佛乘系統總結 → 方便品十如是 → 藥王菩薩本事品（本篇）

4.2 與楞嚴經的交叉

  • 楞嚴經卷五有「藥王藥上菩薩」從味塵入道的圓通法門，與本品的藥王菩薩為同一菩薩的不同面向——楞嚴經重認識論（從六根入手），法華經重信願行（從極致供養入手）
  • 楞嚴經的「五十陰魔」警示不可執著禪定境界，藥王菩薩品的燃身供養則示範了在正知見引導下的極致行持不屬魔事——關鍵在於「以神通力願而自燃」，是正定正慧中的自覺行為

4.3 與教觀綱宗的對應

教觀綱宗以「圓教」判法華，藥王菩薩本事品的「法華十喻」正是圓教判教的核心經證之一。蕅益智旭在《教觀綱宗》中引用「此經為眾經之王」作為法華圓教至尊的論據基礎。

五、修行應用

5.1 從「身捨」到「心捨」的轉化

藥王菩薩燃身供養的故事，在現代修行中不宜照字面模仿。其深層教義是「捨身為法」的精神——願意為追求真理放下一切執著，包括對身體的貪愛、對舒適的依賴、對名利的追逐。日常生活中，每一次放下自我中心的思維，就是一次微小的「燃身供養」。

5.2 法供養的實踐

佛陀明確指出，受持法華經一偈一句的功德勝過燃身供養。這啟示修行者：

  • 每日讀誦法華經，即便只讀一品一偈，即是最上供養
  • 向他人分享法華經的義理，即是法布施
  • 在生活中實踐「一切眾生皆能成佛」的平等觀，即是活出法華精神

5.3 「良藥」精神的日常化

經文以「良藥」比喻法華經：「此經則為閻浮提人病之良藥。」在現代生活中，當遇到煩惱、困境、人際衝突時，回想法華經的核心教導——一切眾生皆具佛性、一切困境皆是修行的道場——即是服用這帖良藥。
