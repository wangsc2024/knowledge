AI 原生文件平台：ReadMe 提供 Agent、Linter、Docs Audit、MCP Server；Mintlify 則提供 Agent Job、搜尋、頁面內容擷取與 llms.txt，可把 docs 變成可程式化維護的系統。 6. LLM 友善規格層：LAPIS 實測對 5 組 production API 規格，相較 OpenAPI YAML 平均減少 85.5% token、相較 JSON 減少 88.6%，適合作為 agent runtime 的壓縮表示。 7. 生成品質守門：DocAgent 採 Reader / Searcher / Writer / Verifier / Orchestrator 五角分工；ReadMe Linter 則把風格規範、錯字、broken links 檢查搬進文件工作流。
