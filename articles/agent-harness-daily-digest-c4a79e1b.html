Agent Harness 自主管理機制

核心問題

agent harness 要如何把研究、優化、審查與實作串成可自主管理的控制迴路。

關鍵結論

  • 將控制面從 PowerShell 巨型腳本內聯條件抽離成單一 supervisor。
  • 以 run-fsm.json、scheduler-state.json、failure-stats.json、failed-auto-tasks.json、api-health.json 作為統一決策輸入。
  • 以 autonomous-harness-plan.json 和 autonomous-recovery-queue.json 承接 go/no-go gate 與恢復派發。
  • 對 stale run、連續失敗、自動任務失敗、open circuit 建立明確 action：restart_agent 或 queue_self_heal。
