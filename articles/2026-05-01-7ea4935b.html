系統優化落實報告 2026-05-01

任務：KB 洞察驅動系統優化（kb_system_optimize）

KB 洞察摘要

  • bb5a3b7b：Windows Task Scheduler FSM 最佳實踐（score 0.5，daily-digest tagged）
  • Backlog 130 pending 項目，最高優先為 emergency 等級的 daily_success_rate / cache_hit_ratio

落實方案 A：KB sync preflight guard（WinError 10061 降級）

問題根因

tools/kb-sync-incremental.ps1 在 KB API（localhost:3000）未啟動時直接呼叫 Invoke-RestMethod，拋出 WinError 10061（連線被拒絕），exit 1 觸發上游 CRITICAL 告警。這是 false positive：服務未啟動不等於同步失敗。

修復方式

在 Step 2（GET KB API）前加入 Test-NetConnection preflight：


$kbPort = 3000
$kbReachable = Test-NetConnection -ComputerName "localhost" -Port $kbPort `
    -InformationLevel Quiet -WarningAction SilentlyContinue 2>$null
if (-not $kbReachable) {
    Write-Host "[KB-Sync] KB 服務未啟動（localhost:$kbPort 連線拒絕），跳過同步（skipped）"
    exit 0
}
  • 服務未啟動 → exit 0（skipped），不觸發告警
  • 服務在線但 API 失敗 → 保留原有 exit 1 路徑
