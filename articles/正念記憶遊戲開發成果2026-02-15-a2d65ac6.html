遊戲資訊

  • 名稱：正念記憶 - Mindful Memory
  • 類型：記憶配對遊戲
  • 路徑：d:/source/game/mindful-memory/
  • 檔案：index.html（約 14KB，純靜態）

技術架構

  • 純 HTML5 + CSS3 + JavaScript
  • 無外部框架依賴
  • 響應式設計（clamp 函數適應不同螢幕）
  • 漸層背景設計（深藍色系）

遊戲特色

  • 融入佛法正念元素
  • 金色漸層標題設計
  • 現代化 UI 風格
  • 適合 Cloudflare Pages 部署

與現有遊戲的差異

  • blockPuzzle_cursor：方塊益智
  • six-roots-zen：六根禪定
  • snake：貪食蛇
  • mindful-memory：記憶配對（新增）

部署方式

直接將 mindful-memory 目錄上傳至 Cloudflare Pages 即可部署。
