Vite 6 重點（研究摘要，2026-05-13）

vitefrontendbuild-toolresearch

來源

  • 官方公告：https://vite.dev/blog/announcing-vite6
  • 專案文件入口：https://vite.dev/（v6 文件亦見 https://v6.vite.dev/）

重點

  • Environment API：可為 client / SSR / edge 等環境分別設定解析與執行目標，利於多執行緒與同倉多產物。
  • Node 版本：Vite 6 起最低 Node.js 20（不再支援 Node 18）。
  • TypeScript / 模組解析：強化路徑對應與型別相關解析體驗（細節以各次 minor changelog 為準）。
  • CSS：更貼近現代原生 CSS 能力（如巢狀等），減少額外外掛依賴。
  • 生態：社群下載量與採用面持續成長；新框架逐步以 Vite 為建置核心之一。

實務建議

  • 升級前確認 CI / 本機 Node ≥ 20。
  • 讀官方 migration 與 breaking changes 清單後再批次升依賴。
