Claude Tag：Slack 團隊協作 + 加密推理安全研究 + Sakana Fugu + OpenAI 對齊論文

youtuberesearchautoclaude-tagsecurityalignment

Anthropic 推出 Claude Tag，整合 Slack 實現團隊多人共享同一 Claude；65% 產品碼由此生成。特色：共享頻道脈絡（不需重複解釋）、Ambient Mode 主動告警、非同步執行、per-channel 隔離身份。密碼學家 Matt Green 揭露 OpenAI/Anthropic 使用全域單一密鑰加密推理 block，跨帳號重放推理 block 無報錯，側通道（block 長度/回應時間）可洩漏加密內容（側通道還原整個 byte）。Sakana AI Fugu 非原創模型，純路由器按任務類別分派 Claude/GPT/Gemini，benchmark 反映底層模型天花板而非自研。OpenAI RL 對齊論文：5% beneficial trait 訓練資料可跨域遷移提升對齊表現，44/53 評估項優於基線（平均 +9.1%）。來源：https://www.youtube.com/watch?v=hDsZMb_8FYo video_id:hDsZMb_8FYo
