【YouTube 摘要】Claude Skills 完整介紹（Tristen O'Brien, 2026-05-24）

來源：https://www.youtube.com/watch?v=O6tQ6V_P8a0

關鍵發現：

  1. Claude Skills 是可重複使用的指令集（.md 檔案 + YAML front matter），安裝後 Claude 永久記住，且僅在需要時載入不影響速度
  2. 使用 Skill Creator 內建工具可免手寫格式，告訴 Claude 需求後自動建立完整 Skill
  3. Anthropic 提供官方 Skill 目錄 + Plugins（行銷、品牌等整套 workflow bundle）
  4. 安全警告：逾 1/3 公開第三方 Skills 含安全漏洞，部分惡意竊取資料；建議僅用官方或自建 Skill
  5. 所有方案（Free/Pro/Max/Enterprise）均支援，需開啟 Code Execution

適用情境：想為重複性任務（客戶報價、工單生成、食譜計算）建立自動化流程的個人或企業用戶
