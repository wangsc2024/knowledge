正念記憶配對遊戲 (Mindful Memory)

概述

一款融合佛法八正道元素的 HTML5 記憶配對遊戲。玩家需翻開卡片找出配對的八正道符號，在遊戲中培養專注與覺察。

技術實作

技術棧

  • 純 HTML5 + CSS3 + JavaScript
  • 無外部框架依賴
  • 響應式設計（clamp() 函數）
  • CSS 3D Transform 翻牌動畫

核心功能

  1. 八正道卡片系統：正見、正思惟、正語、正業、正命、正精進、正念、正定
  2. Fisher-Yates 洗牌演算法：確保卡片隨機排列
  3. 配對邏輯：每次最多翻兩張，配對成功保持翻開
  4. 計時與計分：記錄翻牌次數與遊戲時間
  5. 視覺回饋：配對成功綠色光暈、結束鼓勵語句

關鍵程式碼


// Fisher-Yates 洗牌
function shuffle(array) {
    const shuffled = [...array];
    for (let i = shuffled.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
    }
    return shuffled;
}

  • 👁️ 正見：正確的見解
  • 💭 正思惟：正確的思維
  • 💬 正語：正確的言語
  • 🙏 正業：正確的行為
  • ⚖️ 正命：正確的生活方式
  • 💪 正精進：正確的努力
  • 🧘 正念：正確的覺察
  • 🌸 正定：正確的禪定
