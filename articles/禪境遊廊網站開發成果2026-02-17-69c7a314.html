禪境遊廊網站開發成果

專案概述

將 d:\source\game 目錄中的遊戲整合至 d:\source\game_web，建立一個部署於 Cloudflare Pages 的專業遊戲門戶網站，並推送至 GitHub。

技術架構

1. 響應式設計

  • CSS Grid + Flexbox：遊戲卡片網格自適應（1/2/3 欄）
  • CSS 變數系統：統一管理色彩、間距、字體、圓角、陰影
  • 媒體查詢：768px、1024px 斷點優化
  • 載入動畫：漸入效果 + 延遲動畫（0.1s 遞增）

2. 無障礙友善（WCAG 2.1）

  • ARIA 標籤：role="banner/main/contentinfo/list/listitem"
  • 鍵盤導航：所有互動元素支援 Tab + Enter/Space
  • 語義化 HTML：header、main、footer、article、section
  • 跳轉連結：「跳到主內容」無障礙快捷鍵
  • 聚焦指示器：2px 藍色 outline + 2px offset

3. SEO 優化

  • Meta Tags：description、keywords、author、robots
  • Open Graph：og:type/url/title/description/image/locale
  • Twitter Card：summary_large_image
  • Structured Data：JSON-LD（WebSite schema）
  • Sitemap：8 個遊戲頁面 + 主頁
  • Robots.txt：允許所有爬蟲 + Sitemap 位置

4. 效能優化

  • CSS 變數：減少重複定義，提升可維護性
  • 字體預連接：preconnect to Google Fonts
  • 靜態資源：純 HTML/CSS/JS，無 Node.js 依賴
  • 快取策略（_headers）：

- HTML：1800s（30 分鐘） - CSS/JS：31536000s（1 年）+ immutable - 遊戲資源：86400s（1 天）

5. 安全性

Cloudflare Pages 安全標頭（_headers）：

X-Frame-Options: DENY
X-Content-Type-Options: nosniff
X-XSS-Protection: 1; mode=block
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: geolocation=(), microphone=(), camera=()
Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline' ...

遊戲元資料系統

`js/gameMetadata.js` 集中管理 8 款遊戲：

const games = [
  {
    id: 'six-roots-zen',
    title: '六根淨化',
    subtitle: 'Six Roots Zen',
    icon: '🧘',
    description: '...',
    tags: ['禪修', '打字', '佛學'],
    path: 'games/six-roots-zen/',
    featured: true,
    difficulty: '簡單',
    playtime: '5-10分鐘'
  },
  // ...
];

功能：

  • getAllGames() - 取得所有遊戲
  • getFeaturedGames() - 篩選精選遊戲
  • getGameById(id) - 根據 ID 查詢
  • getGamesByTag(tag) - 根據標籤篩選
  • getAllTags() - 取得所有標籤

主要檔案

1. index.html（主頁）

  • 完整 SEO Meta Tags（Open Graph、Twitter Card、JSON-LD）
  • 語義化 HTML 結構
  • 遊戲網格由 JavaScript 動態生成

2. css/variables.css（CSS 變數）

  • 色彩系統（主色、強調色、禪意色系）
  • 間距系統（xs/sm/md/lg/xl/2xl）
  • 字體系統（sans/serif/mono + 字級）
  • 圓角、陰影、過渡、Z-index
  • 暗色主題支援（prefers-color-scheme: dark）

3. css/main.css（主樣式）

  • 重置樣式 + 無障礙（sr-only、focus-visible）
  • 容器、頁首、遊戲網格、遊戲卡片
  • 響應式設計（768px、1024px 斷點）
  • 載入動畫（fadeIn + nth-child 延遲）
  • 列印樣式

4. js/main.js（主腳本）

  • 動態生成遊戲卡片（createGameCard）
  • 無障礙增強（鍵盤導航、跳轉連結）
  • 分析追蹤（Google Analytics 4 支援）
  • Service Worker 註冊（PWA 支援）
  • 效能監控（Page Load Time）

5. _headers（Cloudflare Pages）

  • 安全標頭（XSS、CSP、Frame Options）
  • 快取策略（HTML/CSS/JS/遊戲資源不同 TTL）

6. README.md

  • 專案簡介、遊戲列表、技術特色
  • 本地開發指南
  • Cloudflare Pages 部署步驟
  • 授權條款（MIT License）

部署流程

  1. 建立目錄結構：css/、js/、games/、assets/
  2. 同步遊戲檔案：8 款遊戲（six-roots-zen、mala-meditation、mindful-memory、snake、zen-rhythm、zen-ink-flow、blockPuzzle_cursor、space_invaders）
  3. 建立網站核心檔案：index.html、CSS、JS、遊戲元資料
  4. 配置 Cloudflare Pages：_headers、robots.txt、sitemap.xml
  5. Git 推送：https://github.com/wangsc2024/game_web
