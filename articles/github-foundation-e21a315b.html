  • - 《大佛頂首楞嚴經》不僅是一部理論經典，更是一部系統的修行指南。經中以「妙奢摩他、三摩、禪那」三種修行方法為核心，構建了從初發心到究竟成佛的完整修證體系。佛陀在經首即宣示：「十方如來，於妙奢摩他、三摩、禪那，最初方便」，明確指出這三法是一切諸佛成就菩提的初始方便。本研究系統探討楞嚴經中圓頓止觀的義理架構，從三種修行方法的定義與関係，到三種漸次的具體實踐，再到首楞嚴大定的究竟圓滿，呈現楞嚴經獨特的修證體系。
  • - 奢摩他（Śamatha，止）：意為「寂靜」「止息」，對應修行中的「放下」。指通過止息妄念、安住真心，令心不隨境轉。在楞嚴經的脈絡中，奢摩他不僅是一般意義的「止」，而是「觀一切法空」的勝義止——了知一切諸法皆如來藏妙真如性，無有實體可得。
  • - 三摩（Samāpatti/Samādhi，觀/等至）：又稱「三摩缽提」，意為「等至」「正受」，對應「看破」。指在止的基礎上生起觀照智慧，如實觀察十二因緣、四聖諦等法，了知一切法如幻如化。三摩的重點在於「觀諸法如幻」——不是否定現象的存在，而是了知現象的幻化本質。
  • - 禪那（Dhyāna，止觀等持）：意為「靜慮」「思維修」，是止與觀的圓融統一。在楞嚴經中，禪那被詮釋為「中道第一義諦」的實踐——既不偏止（空），也不偏觀（有），在止觀平等中任運自在。

  • - 失衡的過患： - 止多觀少：心雖安定但智慧不明，容易落入昏沉、枯定、無記空，如「土木金石」 - 觀多止少：智慧雖明但心不安定，妄念紛飛、掉舉不已，如「風中之燈」
  • - 1. 以如來藏為修行之體：不是從生滅心起修（如一般禪觀），而是直接從「不生不滅」的如來藏性起觀。經中七處徵心、十番顯見即是確立此修行之基。
  • - 2. 以耳根圓通為入手方便：經過二十五圓通的揀擇，文殊菩薩選出觀世音菩薩的耳根圓通法門為此方眾生最契機的修行方法——「反聞聞自性，性成無上道」。
  • - 3. 圓頓不廢漸次：雖然理體上是圓頓的（一悟即悟全體），但事修上仍需三種漸次的漸進對治，體現「理則頓悟，事非頓除」的原則。

  • 若本變更必須在一週內回退，是否能在不影響既有 SLA 的情況下達成？
  • 該變更是否會影響 fitness function 的任一指標（success_rate / cache_hit_rate / token_budget）？
  • 是否存在更輕量的替代方案，能達成同樣的目的？
