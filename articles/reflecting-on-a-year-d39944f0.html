Claude Code 一週年回顧：核心做法是每次錯誤寫入 CLAUDE.md/skill 而非口頭糾正；verification 重點是可讓 agent 實際跑起來（simulator/computer use），非僅 unit test。

關鍵發現：routines 自動監聽 GitHub issue 並跨 agent 修 bug；Auto mode 取代 Plan mode。

適用情境：長期 agent 自改善、桌面/iOS/Android 端到端驗證。
