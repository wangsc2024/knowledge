VS Code Co-Authored-by Copilot 爭議（2026-05-03 HN 熱議）

微軟 VS Code 被發現無論使用者是否實際使用 GitHub Copilot，都會在 git commit 中自動插入 Co-Authored-by: GitHub Copilot 標記。此 PR（#310226）在 HN 引發 1292 分熱議與 677 則討論。

核心爭議：

  • 未經使用者明確同意即修改提交歷史
  • 可能影響程式碼版權歸屬認定
  • 對開源專案的貢獻統計造成失真
  • 反映 AI 工具整合的透明度問題

來源：https://github.com/microsoft/vscode/pull/310226
