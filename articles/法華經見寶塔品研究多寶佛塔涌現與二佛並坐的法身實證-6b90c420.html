法華經見寶塔品研究：多寶佛塔涌現與二佛並坐的法身實證

主題概述

見寶塔品為《妙法蓮華經》第十一品，位於卷四，是天台宗智者大師所判「迹門流通分」的重要一品。此品記載佛陀說法華經時，大地忽然湧出七寶莊嚴的巨大寶塔，塔中傳出多寶如來的讚歎之聲，證明法華經「皆是真實」。繼而釋迦牟尼佛召集十方分身諸佛、三次變淨國土（三變土田），最後開塔與多寶如來並坐，構成全經最壯觀的法會場景。此品在義理上具有三重意涵：一是以十方諸佛的共同證明，確立法華經為一代時教之最高教法；二是以「二佛並坐」象徵過去佛與現在佛的法身一如，印證法華經所說的「一佛乘」為十方三世諸佛共同承認的究竟真理；三是以「三變土田」淨化娑婆世界，預示法華經能轉穢土為淨土的不可思議功德。

一、經文重點段落

1.1 七寶塔從地涌出

經文（鳩摩羅什譯）：

「爾時佛前有七寶塔，高五百由旬，縱廣二百五十由旬，從地涌出，住在空中。種種寶物而莊校之，五千欄楯，龕室千萬，無數幢幡以為嚴飾，垂寶瓔珞，寶鈴萬億而懸其上。四面皆出多摩羅跋栴檀之香，充遍世界。」

白話翻譯：這時佛陀面前出現一座七寶所成的巨塔，高五百由旬、寬二百五十由旬，從地面湧出懸停在虛空之中。塔身以各種珍寶莊嚴，有五千層欄杆、千萬間龕室，無數幢幡旗幟裝飾其上，垂掛著寶石瓔珞，億萬寶鈴懸於其間。四面散發多摩羅跋栴檀的香氣，遍滿整個世界。

1.2 塔中大音聲證明法華

經文：

「爾時寶塔中出大音聲，歎言：善哉善哉！釋迦牟尼世尊，能以平等大慧，教菩薩法，佛所護念，妙法蓮華經，為大眾說。如是如是！釋迦牟尼世尊，如所說者，皆是真實。」

白話翻譯：這時寶塔中傳出洪亮的聲音讚歎道：「太好了太好了！釋迦牟尼世尊，能以平等大智慧，為眾生說教菩薩的佛法、諸佛所護念的妙法蓮華經。正是如此！釋迦牟尼世尊，您所宣說的一切，全都是真實不虛的。」

1.3 多寶如來的因地本願

經文：

「是名多寶如來。其佛行菩薩道時，作大誓願：若我成佛，滅度之後，於十方國土，有說法華經處，我之塔廟，為聽是經故，涌現其前，為作證明，讚言善哉。」

白話翻譯：這位佛名為多寶如來。他在修菩薩道時立下大誓願：「如果我成佛，入滅之後，在十方一切國土中，凡是有人宣說法華經的地方，我的寶塔就會為了聽聞此經而湧現在其前，為之作證並讚歎『善哉』。」

1.4 三變土田與十方分身佛雲集

經文：

「爾時釋迦牟尼佛，以神通力，接諸大眾，皆在虛空。……即時釋迦牟尼佛，以右指開七寶塔戶。出大音聲，如卻關鑰，開大城門。」

為了迎接十方分身諸佛，釋迦牟尼佛三次變淨國土（三變土田）：第一次將娑婆世界變為清淨莊嚴；第二次再變二百萬億那由他國土；第三次又變二百萬億那由他國土——使所有分身佛各有安住之處，象徵法華會之廣大無邊。

1.5 二佛並坐

經文：

「即時多寶佛，於寶塔中，分半座與釋迦牟尼佛，而作是言：釋迦牟尼佛，可就此座。即時釋迦牟尼佛入其塔中，坐其半座，結跏趺坐。」

白話翻譯：多寶如來在寶塔中分出半個座位給釋迦牟尼佛，說：「釋迦牟尼佛，請坐此座。」釋迦牟尼佛隨即進入塔中，坐在半座上，結跏趺坐。

二、義理解析

2.1 天台宗「五時八教」脈絡下的見寶塔品

天台智者大師將法華經判為五時中的第五時——法華涅槃時，屬於圓教最高位。見寶塔品在迹門流通分中具有特殊地位：

  • 結構功能：承接法師品（五種法師行），開啟提婆達多品與勸持品，是迹門「令他弘經」段的核心
  • 教判意義：多寶如來從東方寶淨世界來證明法華，表示此經不僅是娑婆世界的最高教法，更是十方三世一切諸佛共同護持的普遍真理
  • 迹本交涉：見寶塔品雖屬迹門，但「二佛並坐」已暗示本門——多寶佛代表久遠過去的法身常住，與釋迦佛的現在應身並坐，預示壽量品「久遠實成」的本門義理

2.2 多寶如來本願的深層義涵

多寶如來的本願有三層深意：

  1. 跨時空的護法精神：身雖滅度（「全身舍利」安住塔中），願力不滅，凡有說法華經處即現身證明。這說明佛的願力超越生死、跨越時空，法身常住不滅
  2. 法華經的殊勝地位：十方無量世界中，多寶佛唯為法華經而涌現。不為其他經典，不為其他法會——這是諸佛以實際行動確認法華經為「經中之王」
  3. 全身舍利與法身不壞：多寶佛以「全身舍利」安住寶塔，不同於一般佛入滅後分散的碎身舍利，象徵法身真如不可分割、完整圓滿

2.3 二佛並坐的圓教象徵

「二佛並坐」是見寶塔品最核心的意象，天台宗從多個層面解讀：

  1. 過去佛與現在佛的法身一如：多寶佛（過去）與釋迦佛（現在）並坐，表示十方三世諸佛雖名號不同、國土各異，但其所證的法身實相完全一致——「佛佛道同」
  2. 法華真理的普遍性與永恆性：過去佛為現在佛作證，意味法華經所說的一佛乘真理非釋迦佛的獨創，而是法爾如是、本來如此的宇宙法則
  3. 理事無礙的圓融：多寶佛代表「理體」（法性常住）、釋迦佛代表「事用」（應機說法）。二佛並坐即理事圓融、體用一如，正是圓教「一念三千、三諦圓融」的具象化表現
  4. 星雲大師的詮釋：「過去、現在二佛並座，為了令世人相信，只要信仰受持法華經，必當作佛。」強調此一場景的核心功能是堅定眾生的成佛信心

2.4 三變土田的淨土觀

釋迦佛為迎接十方分身佛而三次變淨國土，其義理包含：

  1. 穢土即淨土：娑婆世界經佛力變淨，不需另到他方世界，此處即是道場——契合天台「煩惱即菩提、生死即涅槃」的圓教精神
  2. 心淨則國土淨：宣化上人開示：「三變土田象徵聞法者心識的逐步淨化與提升。」外在國土的變淨，實質是內心的轉化
  3. 法華會的宇宙規模：三變之後，十方無量分身佛齊聚一處，使法華法會擴展為遍及法界的宇宙性大會，彰顯一佛乘真理的無限廣大

三、與已有研究的關聯

3.1 與前後品的承續

見寶塔品承接法師品（弘經的五種實踐），轉為以諸佛實際行動證明弘經的價值；接續提婆達多品，則從「諸佛證明」進入「具體授記」，展示法華經的平等精神。全經研究脈絡：

概論 → 方便品 → 譬喻品 → 信解品 → 藥草喻品 → 化城喻品 → 壽量品 → 普門品 → 一佛乘總結 → 提婆達多品 → 安樂行品 → 授記品 → 法師品 → 見寶塔品（本篇）

3.2 與楞嚴經的交叉

  • 楞嚴經重「如來藏常住真心」，法華經見寶塔品以多寶佛全身舍利常住寶塔，具象化地呈現「法身常住不滅」的相同義理
  • 楞嚴經的「七處征心」破除妄執，見寶塔品的「三變土田」則正面展示心淨土淨的圓教實相
  • 兩經互補：楞嚴從認識論入手（破妄），法華從教判論入手（顯實），最終歸於同一「法身圓滿」的證悟

3.3 與教觀綱宗的對應

教觀綱宗的「五時判教」以法華經為最高圓教。見寶塔品中十方諸佛齊聚證明法華，正是「五時」判教中法華獨尊地位的最有力證據。四悉檀中，見寶塔品以「第一義悉檀」為主——直顯諸法實相，不經方便，令大眾直入圓教見地。

四、修行應用

4.1 建立不退轉的信心

十方諸佛為法華經共同作證，修行者應由此生起堅定信心：法華經所說「一切眾生皆能成佛」是宇宙的究竟真理，不是權巧方便，不是鼓勵之辭。如同多寶佛跨越時空前來證明，修行者也應以不退轉的信心持受法華。

4.2 發長遠心護持正法

多寶如來因地發願——入滅後仍為法華經涌現作證。這啟示修行者：護持正法不應以一生為限，而應發跨越生死的長遠心。在日常中，以讀誦、受持、解說、書寫、供養法華經為修行，即是回應多寶佛的本願。

4.3 心淨土淨的日常實踐

「三變土田」教導我們：娑婆世界不必捨棄，只需轉化——以清淨心面對現實生活中的困境，即是將穢土變為淨土。每一次以正念面對煩惱、以慈悲回應逆境，都是個人的「變土田」。

4.4 布施與利他的多寶精神

多寶如來名號中的「多寶」，象徵法財與世財的圓滿。修行者可在日常生活中實踐布施——不僅是物質的布施，更包括法布施（分享佛法智慧）、無畏施（給予他人勇氣和安慰），以此與多寶如來的願力相應。
