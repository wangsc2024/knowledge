專案觀測摘要（2026-06-25）

主題：Agent 架構（runtime orchestration / state management / human-in-the-loop）

搜尋方向：LangGraph/AutoGen/Temporal，焦點 state management 與 HITL 設計（Haiku 建議）

LangGraph（langchain-ai/langgraph）35,681★

  • runtime_model：狀態圖式編排（帶循環），事件驅動節點架構
  • state_strategy：Checkpoint-based 持久化（MemorySaver/自訂 CheckpointSaver）；thread-based 狀態快照支援 time-travel 與容錯
  • human_review_model：interrupt-point 模式 — LangGraph interrupt() API，graph 遇 breakpoint 暫停等待人工 approve/edit，適合同步短延遲審核
  • evidence：REST 觀測 stargazers_count=35681，pushed_at=2026-06-25T03:32:44Z

CrewAI（crewAIInc/crewAI）54,326★

  • runtime_model：雙層架構 Crews + Flows（事件驅動 @router DAG 條件分支）
  • state_strategy：短期記憶 + Pydantic 狀態 + 外部持久存儲
  • human_review_model：interrupt-point 模式 — wait_for_input() 方法，Flow 節點暫停等待人工輸入
  • evidence：REST 觀測 stargazers_count=54326，pushed_at=2026-06-25T04:02:07Z

Microsoft Agent Framework（microsoft/agent-framework）11,641★

  • runtime_model：圖式工作流（順序/並行/handoff/群組）；CodeAct 模式 hyperlight 容器隔離
  • state_strategy：Checkpoint-based 持久執行，自動狀態持久化，從最後成功步驟恢復
  • human_review_model：signal-based 模式 — RequestInfoEvent emission，workflow 暫停等外部信號注入，適合非同步高延遲審核
  • evidence：REST 觀測 stargazers_count=11641，pushed_at=2026-06-25T05:54:56Z
