主題概述

Claude Code 是 Anthropic 推出的終端機 AI 編程工具，2026 年迎來重大更新。主要亮點包括：Agent Teams 多代理協作、Checkpoints 自動備份回滾、Automatic Memory 自動記憶、Claude Opus 4.6 整合（1M token 上下文）、以及 VS Code 原生擴充套件。這些功能大幅提升了開發效率，讓 Claude 能處理更複雜、更長時間的程式開發任務。

核心新功能

1. Agent Teams（多代理協作）

  • 狀態：研究預覽版（Research Preview）
  • 啟用方式：設定環境變數 CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
  • 功能：多個代理並行工作，自主協調，適合可拆分的獨立任務
  • 用例：一個代理建構後端 API，另一個同時建構前端

2. Checkpoints（檢查點系統）

  • 自動保存：每次變更前自動保存程式碼狀態
  • 快速回滾：按 Esc 兩次或使用 /rewind 命令回到先前版本
  • 好處：可放心進行大膽的重構，隨時可還原

3. Automatic Memory（自動記憶）

  • Claude 自動記錄並回想工作過程中的記憶
  • 提升跨會話的上下文理解能力

4. Claude Opus 4.6 整合

  • 1M token 上下文：首次 Opus 級模型支援百萬 token（Beta）
  • 改進能力：

- 更細緻的規劃 - 更長時間的代理任務持續力 - 更可靠地處理大型程式碼庫 - 更好的程式碼審查和除錯能力

  • Fast Mode：最高 2.5 倍速度（需額外付費）
  • 擴展輸出：支援最高 128k token 輸出

5. Background Tasks（背景任務）

  • 長時間運行的開發伺服器可在背景持續運作
  • 不阻擋 Claude 處理其他工作

6. Hooks（鉤子）

  • 在開發流程特定時點觸發預定義動作
  • 如：程式碼修改後自動執行測試套件、提交前執行 linting

CLI 更新（Claude Code 2.1.x）

v2.1.0（2026 年 1 月）

  • Shift+Enter 換行（免設定）
  • Skills 熱重載、分叉上下文、自訂代理支援
  • /teleport 傳送會話到 claude.ai/code
  • 工具權限萬用字元支援（如 Bash(*-h*)）
  • 多語言回應設定（日文、西班牙文等）

其他 CLI 功能

  • --from-pr 旗標：從 GitHub PR 恢復會話
  • /debug 命令：協助排查當前會話問題
  • 可點擊檔案路徑超連結（OSC 8 終端機）
  • WinGet 安裝支援
  • 會話自動連結 PR（透過 gh pr create）

PDF 處理改進

  • Read 工具新增 pages 參數，可指定頁面範圍（如 pages: "1-5"）
  • 大型 PDF（>10 頁）以輕量參考形式返回，避免填滿上下文

MCP Server 增強

  • 預設 OAuth 用戶端憑證
  • 支援 --client-id 和 --client-secret 參數

VS Code 擴充套件（Beta）

  • 原生擴充套件，提供側邊欄 inline diff 即時檢視
  • 遠端會話支援
  • 可瀏覽並恢復 claude.ai 上的會話
  • 會話選擇器顯示 git 分支和訊息數量

Claude in Chrome（Beta）

  • 與 Chrome 擴充套件整合
  • 直接從 Claude Code 控制瀏覽器

安裝方式變更

  • npm 安裝已棄用
  • 推薦方式：Homebrew、WinGet 或原生安裝
  • 原生安裝會自動背景更新

使用範例

啟用 Agent Teams

export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
claude

回滾程式碼變更

# 方法 1：快捷鍵
按 Esc 兩次

# 方法 2：命令
/rewind

從 PR 恢復會話

claude --from-pr 123
# 或
claude --from-pr https://github.com/owner/repo/pull/123
