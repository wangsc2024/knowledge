How to build an AI Agent and MCP Server (step-by-step)

youtuberesearchMCP server

Google Cloud 教學：用 MCP 讓 ADK agent 經 stdio 發現/呼叫外部工具。示範 server.py 暴露 Google Trends，blog agent 寫稿前先取即時趨勢。重點：工具隔離、跨語言、runtime schema 自描述、name+args→JSON 結果。

關鍵發現：MCP 讓 agent 無需改 core 即可接外部 data tool。

適用情境：ADK/Gemini agent 接第三方 API 或內部微服務。
