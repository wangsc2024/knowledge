專案觀測摘要

mem0ai/mem0（~53.5k★）

架構：混合存儲三層（vector+KV+graph）+ 多範圍隔離（user/session/agent scope）。Mem0g 圖增強變體做衝突偵測。LongMemEval: 49.0%（GPT-4o）。整合 21 個框架。

架構：時態知識圖（Temporal KG）— 每個 fact 含 valid_from/valid_to 窗口。舊 fact 失效而非刪除；支援時點查詢。LongMemEval: 63.8%（比 mem0 高 15%）。

letta-ai/letta（est. 13k★，MemGPT 繼承者）

架構：OS 式三層分頁 — core（永在 context，RAM-like）+ archival（外部向量庫）+ recall（對話歷史）。Agent 主動呼叫記憶管理函式（非被動注入）。
